## 1. 概述

Lance Andersen 是 Oracle 的 **JDBC (Java Database Connectivity)** 技术负责人，专注于 Java 数据库连接和 SQL 相关功能。他在 JDK 26 中以 6 次提交贡献于 JDBC 模块。

## 3. 核心技术贡献

### 1. JDBC

Lance Andersen 负责 Java 数据库连接：
- **JDBC API**: java.sql 包
- **RowSet**: 行集支持
- **DataSource**: 数据源实现

### 2. SQL 支持

- **SQL Syntax**: SQL 语法支持
