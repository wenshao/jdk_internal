## 3. 主要贡献

### 1. Core Libraries

Ivan Gerasimov 贡献于 Java 核心库：
- **java.util.regex**: 正则表达式引擎维护和 bug 修复
- **java.nio**: NIO 文件和通道相关改进
- **ProcessBuilder**: 进程管理 (包括 Windows 平台 inheritIO 修复)
- **java.lang**: 核心语言类优化
