## 1. 概述

Mikael Vidstedt 是 Oracle Java Platform Group 的 **Senior Director of the Java Virtual Machine (JVM)**，同时担任 OpenJDK JDK Project 的 **VM Area Lead**。他拥有 20 多年的 Java 和 JVM 技术经验，涵盖编译器优化、操作系统和服务器虚拟化等多个领域，曾担任 JVM tech lead 和 Architect。他是 Project Portola（Alpine Linux JDK 移植）的发起人和负责人，该项目在 JDK 18 中交付。他持有瑞典皇家理工学院 (KTH) 计算机科学硕士学位。

## 3. 核心技术贡献

### 1. HotSpot Runtime

Mikael Vidstedt 贡献于 HotSpot 运行时：
- **Runtime**: 运行时系统
- **Threads**: 线程管理
- **Architecture**: 架构相关代码
- **Build System**: 构建系统 (libsleef, AArch64 支持)

### 2. Project Portola

作为 Project Portola 的发起人和负责人，Mikael 主导了 JDK 向 Alpine Linux 和 musl C 库的移植工作。该项目在 JDK 18 中交付，项目于 2024 年 5 月解散。

### 3. 技术演讲
