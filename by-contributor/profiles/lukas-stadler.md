## 1. 概述

Lukas Stadler 是 Oracle Labs Austria 的 **Director, GraalVM Dynamic Languages**，管理 GraalVM 多语言生态系统中的 Python (GraalPy)、LLVM (Sulong) 和 R (FastR) 运行时。他是 Graal 项目的联合创始人之一（2010 年在 JKU Linz 启动），拥有 JKU Linz 博士学位（论文主题: Partial Escape Analysis and Scalar Replacement for Java），在 JVM JIT 编译优化、动态语言支持和编译器基础设施方面做出了重要贡献。
