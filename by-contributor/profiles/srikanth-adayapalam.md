## 1. 概述

Srikanth Adayapalam 是 Oracle 的 **javac 编译器开发者**，专注于 Java 编译器实现和语言特性支持。加入 Oracle 之前，他曾是 IBM 的 **Eclipse Compiler (ECJ) 技术主管**。2015 年被提名为 JDK 9 Committer。他在 JDK 26 中以 4 次提交贡献于 javac 相关功能。
