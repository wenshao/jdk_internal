## 1. 概述

Markus Grönlund 是 Oracle 的 **Java Flight Recorder (JFR)** 核心开发者，与 Erik Gahlin 同为 JFR 团队核心成员，也是 JEP 349 (JFR Event Streaming) 的共同作者。他自 1995 年开始接触 Java，2005 年开始参与 Java 开发，2007 年签署 Sun Contribution Agreement。他在 JDK 26 中以 17 次提交贡献于 JFR 功能的改进。他是 OpenJDK Reviewer (280+ reviews) 和 Committer (220+ commits)，同时是 Hotspot、Build、Porters 和 Vulnerability 组的成员。
