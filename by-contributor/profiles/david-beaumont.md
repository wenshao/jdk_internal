## 3. 代表性工作

### 1. Google Flogger (Fluent Logging API)

David Beaumont 设计并实现了 **Google Flogger**，这是一个为 Java 设计的 Fluent Logging API：

- **GitHub**: [google/flogger](https://github.com/google/flogger)
- **特点**: 流式 API、类型安全、高性能
- **影响**: 被 Google 内部广泛使用，影响了 Java 日志库的设计

### 2. "Principles of Fluent API Design" 演讲

- **会议**: Google
- **时长**: 20 分钟
- **链接**: [YouTube](https://www.youtube.com/watch?v=VPu-ytfYTeU)
- **内容**: 分享了在 Google Core Java Libraries 团队的 API 设计经验
