## 3. 主要贡献

### 1. 核心类库性能优化

Sergey Tsypanov 贡献于 Java 核心类库性能改进：
- **JDK-8263561**: 重新审视 LinkedList 的使用，替换为更高效的 ArrayList 或 ArrayDeque
- **JDK-8264896**: 移除 int-to-byte 转换中冗余的 `& 0xFF` 操作
- **JDK-8298380**: 清理 JDK 代码库中冗余的数组长度检查
