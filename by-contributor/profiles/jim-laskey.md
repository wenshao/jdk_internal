## 1. 概述

Jim Laskey 是 Oracle 的 Software Development Director，拥有 40+ 年编译器/运行时开发经验，曾在 Symantec、Apple、Azul、Sun 和 Oracle 工作。他是多个重要 JEP 的作者，包括 Text Blocks、String Templates 和 Enhanced Pseudo-Random Number Generators。他还领导了 Nashorn JavaScript 引擎项目 (JDK 8-14)。

> **数据来源**: [Inside.java](https://inside.java/u/JimLaskey/), [GitHub](https://github.com/JimLaskey), [JEP 430](https://openjdk.org/jeps/430), [Oracle Blog](https://blogs.oracle.com/authors/jim-laskey)

## 5. String Templates 撤回

Jim Laskey 领导的 String Templates 项目经历三次预览 (JEP 430/459/465) 后被撤回：

**撤回原因**:
- 设计需要进一步讨论
- 社区反馈需要更多考虑
- 没有确定何时会重新引入

详见: [JDK-8343684](https://bugs.openjdk.org/browse/JDK-8343684)
