## 概述

Jim Laskey 是 Oracle 的 Java 语言设计师，专注于字符串处理、模板和语言简化。他是多个重要 JEP 的作者。

## String Templates 撤回

Jim Laskey 领导的 String Templates (JEP 430) 项目在 JDK 21-22 经历三次预览后，于 JDK 23 前被撤回：

**撤回原因**:
- 设计需要进一步讨论
- 社区反馈需要更多考虑
- 没有确定何时会重新引入

详见: [JDK-8343684](https://bugs.openjdk.org/browse/JDK-8343684)
