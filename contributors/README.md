## 如何阅读贡献者页面

每个贡献者页面包含:

1. **基本信息**: 姓名、组织、专注领域
2. **贡献统计**: Commits 数量、JEP 参与
3. **PR 列表**: 按类别组织的详细 PR 列表（含 JBS 链接）
4. **关键贡献**: 最重要的几个贡献详解
5. **代码示例**: 相关代码变更示例
6. **相关链接**: OpenJDK 页面、邮件列表等
