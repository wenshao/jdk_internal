## 1. 概览

Azul Systems 是 [Zulu JDK](https://www.azul.com/downloads/) 的维护者，也是 OpenJDK 6 的维护领导者 (2017 年从 Red Hat 接手)。Azul 在 OpenJDK 上游有持续的贡献，同时推动了 [CRaC (Coordinated Restore at Checkpoint)](https://wiki.openjdk.org/display/crac) 项目。

| 指标 | 值 |
|------|-----|
| **源码版权** | 67 个文件含 Azul 版权 |
| **上游贡献者** | 1+ |
| **活跃时间** | 2013 - 至今 |
| **主要领域** | JDK 维护, CRaC, 性能优化 |
| **发行版** | [Zulu JDK](https://www.azul.com/downloads/) |
| **OpenJDK 6** | 2017 年接手维护领导 |

> **统计说明**: Azul 的贡献部分通过上游 PR，部分通过 backport 和维护工作。上游 PR 统计可能不完全反映其贡献。

## 3. Zulu JDK

| 特性 | 说明 |
|------|------|
| **基于** | OpenJDK |
| **特点** | 免费, TCK 认证, 多平台 (x86, ARM, PPC, SPARC) |
| **支持版本** | 6 / 7 / 8 / 11 / 17 / 21 / 25 |
| **历史** | 2013 年与 Microsoft 合作创建，首个第三方 OpenJDK 认证发行版之一 |
| **许可** | GPLv2 |
