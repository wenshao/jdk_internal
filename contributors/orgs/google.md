## 1. 概览

Google 在 OpenJDK 的直接代码贡献相对有限，主要通过 Android 生态和 Chromium/V8 的 Java 互操作间接影响 Java 平台。JDK 26 开发周期中 Google 有约 14 个提交、3 位贡献者，主要集中在构建系统、javac 和并发领域。

| 指标 | 值 |
|------|-----|
| **JDK 26 提交数** | ~14 |
| **JDK 26 贡献者数** | ~3 |
| **主要领域** | 构建系统、javac、并发 |
