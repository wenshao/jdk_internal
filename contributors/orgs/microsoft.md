## 1. 概览

Microsoft 主要通过维护 [Microsoft Build of OpenJDK](https://learn.microsoft.com/java/openjdk/) 发行版参与 OpenJDK 生态。上游直接代码贡献极少，贡献模式以下游发行版维护和 Azure 平台集成为主。

| 指标 | 值 |
|------|-----|
| **上游 Integrated PRs** | 极少 |
| **活跃时间** | 2020+ - 至今 |
| **主要角色** | 下游发行版维护 |
| **发行版** | [Microsoft Build of OpenJDK](https://learn.microsoft.com/java/openjdk/) |

> **统计说明**: Microsoft 的 OpenJDK 参与主要通过下游发行版维护，上游 openjdk/jdk 仓库直接贡献极少。

## 2. Microsoft Build of OpenJDK

Microsoft 维护自己的 OpenJDK 发行版，面向 Azure 和企业用户：

| 特性 | 说明 |
|------|------|
| **基于** | OpenJDK |
| **支持版本** | JDK 11 / 17 / 21 |
| **平台** | Windows, Linux, macOS (含 ARM64) |
| **许可** | GPLv2 |
| **特点** | 免费使用, Azure 集成, 企业级 LTS 支持 |
| **下载** | [learn.microsoft.com/java/openjdk](https://learn.microsoft.com/java/openjdk/) |
