## 8. 关键里程碑

### 2014: JDK 8 GA
- Lambda 表达式
- Stream API
- Date/Time API
- Nashorn JavaScript 引擎

### 2017: JDK 9 GA
- 模块系统 (Jigsaw)
- JShell
- jlink
- G1 成为默认 GC

### 2018: JDK 11 LTS
- HTTP Client API
- ZGC 实验性
- Epsilon GC
- Flight Recorder 开源

### 2021: JDK 17 LTS
- Sealed Classes
- Pattern Matching
- Records
- 强封装 JDK 内部 API

### 2023: JDK 21 LTS
- **虚拟线程 GA**
- 分代 ZGC 预览
- Sequenced Collections
- Record Patterns

### 2025: JDK 25
- String Templates GA
- 分代 Shenandoah
- 性能优化 (启动 +40%)
