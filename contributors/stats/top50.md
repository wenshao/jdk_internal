## 2. 统计说明

- **排名依据**: GitHub Integrated PRs 数量
- **排除账号**: 自动化账号 (如 J. Duke, Lana Steuck)
- **同一人合并**: 已合并同一贡献者的多个 GitHub 账号
- **数据时间**: 2021-2026 (GitHub PR 集成后)
- **统计来源**: [GitHub PR Search](https://github.com/openjdk/jdk/pulls?q=is%3Apr+label%3Aintegrated+is%3Aclosed)

> ⚠️ **注意**: 不使用 Git commits 统计，因为 OpenJDK Committer 使用 `@openjdk.org` 邮箱提交，无法准确反映实际组织归属。详见 [AGENTS.md](/AGENTS.md#contribution-statistics-principles)
