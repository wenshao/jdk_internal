## 2. 概述

JSR 201 是一个"伞式 JSR"（Umbrella JSR），将五个小型但高频使用的语言增强打包为一个规范：

| 特性 | 说明 |
|------|------|
| **枚举** | 取代 `int` 常量模式，类型安全、可扩展 |
| **自动装箱/拆箱** | 基本类型与包装类型自动转换 |
| **增强 for** | 简化集合和数组的遍历 |
| **静态导入** | 直接使用静态成员，无需类名前缀 |
| **可变参数** | 方法接受任意数量的参数 |

这五个特性合在一起，显著改善了 Java 代码的简洁性和可读性。

## 3. 枚举类型

### 基本枚举

```java
// JDK 1.4: int 枚举模式 (Typesafe Enum 之前)
public static final int SEASON_SPRING = 0;
public static final int SEASON_SUMMER = 1;  // 不类型安全!

// JDK 5: 类型安全枚举
public enum Season {
    SPRING, SUMMER, AUTUMN, WINTER
}
```

### 带字段和方法的枚举

```java
public enum Planet {
    MERCURY(3.303e+23, 2.4397e6),
    VENUS(4.869e+24, 6.0518e6),
    EARTH(5.976e+24, 6.37814e6);

    private final double mass;    // kg
    private final double radius;  // meters

    Planet(double mass, double radius) {
        this.mass = mass;
        this.radius = radius;
    }

    // 表面重力 (Surface Gravity)
    public double surfaceGravity() {
        final double G = 6.67300E-11;
        return G * mass / (radius * radius);
    }

    // 表面重量 (Surface Weight)
    public double surfaceWeight(double otherMass) {
        return otherMass * surfaceGravity();
    }
}
```

### 抽象方法枚举 — 策略模式

```java
public enum Operation {
    PLUS("+") {
        @Override
        public double apply(double x, double y) { return x + y; }
    },
    MINUS("-") {
        @Override
        public double apply(double x, double y) { return x - y; }
    },
    TIMES("*") {
        @Override
        public double apply(double x, double y) { return x * y; }
    },
    DIVIDE("/") {
        @Override
        public double apply(double x, double y) { return x / y; }
    };

    private final String symbol;
    Operation(String symbol) { this.symbol = symbol; }

    public abstract double apply(double x, double y);

    @Override
    public String toString() { return symbol; }
}

// 使用
double result = Operation.PLUS.apply(3, 4);  // 7.0
```

### EnumSet 与 EnumMap

```java
// EnumSet — 基于位向量 (Bit Vector)，极高性能
EnumSet<Season> warmSeasons = EnumSet.of(Season.SPRING, Season.SUMMER);
EnumSet<Season> allSeasons = EnumSet.allOf(Season.class);
EnumSet<Season> noSeason = EnumSet.noneOf(Season.class);

// 替代位标志 (Bit Flags)
// 旧: int flags = BOLD | ITALIC;
// 新:
EnumSet<Style> styles = EnumSet.of(Style.BOLD, Style.ITALIC);

// EnumMap — 基于数组，比 HashMap 快
EnumMap<Season, String> seasonNames = new EnumMap<>(Season.class);
seasonNames.put(Season.SPRING, "春天");
seasonNames.put(Season.SUMMER, "夏天");
```

## 4. 自动装箱与拆箱

### 基本用法

```java
// 自动装箱 (Autoboxing): 基本类型 → 包装类型
Integer n = 42;          // 编译器: Integer.valueOf(42)
Double d = 3.14;         // 编译器: Double.valueOf(3.14)
Boolean b = true;        // 编译器: Boolean.valueOf(true)

// 自动拆箱 (Unboxing): 包装类型 → 基本类型
int x = n;               // 编译器: n.intValue()
double y = d;             // 编译器: d.doubleValue()

// 混合运算
Integer sum = 0;
for (int i = 0; i < 100; i++) {
    sum += i;  // 拆箱 → 加法 → 装箱 (性能陷阱!)
}
```

### 缓存机制 (Integer Cache)

```java
// Integer 缓存 -128 ~ 127 (JLS 5.1.7)
Integer a = 127;
Integer b = 127;
System.out.println(a == b);    // true  (同一缓存对象)

Integer c = 128;
Integer d = 128;
System.out.println(c == d);    // false (不同对象!)
System.out.println(c.equals(d)); // true  (值相等)

// 可通过 JVM 参数扩大缓存范围
// -XX:AutoBoxCacheMax=1000
```

### 拆箱陷阱 — NullPointerException

```java
// ⚠ 拆箱 null 引发 NPE
Integer nullable = null;
int value = nullable;     // NullPointerException!

// 安全写法
int value = (nullable != null) ? nullable : 0;

// JDK 9+ Optional 替代
OptionalInt opt = OptionalInt.of(42);
int value = opt.orElse(0);
```

### == 与 equals 的陷阱

```java
// ❌ 包装类型不要用 == 比较
Integer x = 200;
Integer y = 200;
if (x == y) { }           // false! 超出缓存范围

// ✓ 用 equals 或拆箱后比较
if (x.equals(y)) { }      // true
if (x.intValue() == y.intValue()) { }  // true
```

## 5. 增强 for 循环

### 遍历数组与集合

```java
// JDK 1.4: Iterator 模式
for (Iterator<String> it = list.iterator(); it.hasNext(); ) {
    String s = it.next();
    System.out.println(s);
}

// JDK 5: 增强 for (Enhanced For / For-Each)
for (String s : list) {
    System.out.println(s);
}

// 数组遍历
int[] numbers = {1, 2, 3, 4, 5};
for (int n : numbers) {
    System.out.println(n);
}

// 嵌套遍历
for (List<String> row : matrix) {
    for (String cell : row) {
        System.out.print(cell + " ");
    }
}
```

### Iterable 接口

```java
// 任何实现 Iterable<T> 的类都可以使用 for-each
public class Range implements Iterable<Integer> {
    private final int start, end;
    public Range(int start, int end) { this.start = start; this.end = end; }

    @Override
    public Iterator<Integer> iterator() {
        return new Iterator<>() {
            int current = start;
            public boolean hasNext() { return current < end; }
            public Integer next() { return current++; }
        };
    }
}

// 使用
for (int i : new Range(0, 10)) {
    System.out.println(i);
}
```

### 限制

```java
// ❌ 不能在 for-each 中删除元素
for (String s : list) {
    list.remove(s);  // ConcurrentModificationException!
}

// ✓ 使用 Iterator.remove()
Iterator<String> it = list.iterator();
while (it.hasNext()) {
    if (it.next().isEmpty()) it.remove();
}

// ✓ JDK 8+ removeIf
list.removeIf(String::isEmpty);
```

## 6. 静态导入

```java
// JDK 1.4
double r = Math.sqrt(Math.pow(x, 2) + Math.pow(y, 2));
Assert.assertEquals(expected, actual);

// JDK 5: 静态导入
import static java.lang.Math.sqrt;
import static java.lang.Math.pow;
import static org.junit.Assert.*;

double r = sqrt(pow(x, 2) + pow(y, 2));
assertEquals(expected, actual);

// 通配符导入 (谨慎使用)
import static java.lang.Math.*;
```

## 7. 可变参数

```java
// JDK 1.4: 数组参数
public static String format(String fmt, Object[] args) { }
String s = String.format("%s is %d", new Object[]{"Java", 5});

// JDK 5: 可变参数 (Varargs)
public static String format(String fmt, Object... args) { }
String s = String.format("%s is %d", "Java", 5);

// 编译器将 varargs 转为数组
// format("Java", 5) → format(new Object[]{"Java", Integer.valueOf(5)})

// 自定义可变参数方法
public static <T> List<T> listOf(T... elements) {
    return Arrays.asList(elements);
}
List<String> names = listOf("Alice", "Bob", "Charlie");
```

### 泛型可变参数的安全问题

```java
// ⚠ 泛型可变参数存在堆污染风险 (Heap Pollution)
@SafeVarargs  // JDK 7: 抑制警告
public static <T> List<T> asList(T... a) {
    return new ArrayList<>(Arrays.asList(a));
}

// 堆污染示例
@SafeVarargs
static <T> T[] toArray(T... args) {
    return args;  // 实际类型是 Object[]，不安全!
}
```

## 8. 实现原理

### 枚举的字节码

```java
// 源码
public enum Color { RED, GREEN, BLUE }

// 编译器生成等价类 (反编译)
public final class Color extends Enum<Color> {
    public static final Color RED = new Color("RED", 0);
    public static final Color GREEN = new Color("GREEN", 1);
    public static final Color BLUE = new Color("BLUE", 2);
    private static final Color[] $VALUES = {RED, GREEN, BLUE};

    private Color(String name, int ordinal) { super(name, ordinal); }
    public static Color[] values() { return $VALUES.clone(); }
    public static Color valueOf(String name) { return Enum.valueOf(Color.class, name); }
}
```

### 增强 for 的脱糖

```java
// 源码
for (String s : list) { System.out.println(s); }

// 编译器脱糖 (集合)
for (Iterator<String> it = list.iterator(); it.hasNext(); ) {
    String s = it.next();
    System.out.println(s);
}

// 编译器脱糖 (数组)
// for (int n : arr) → for (int i = 0; i < arr.length; i++) { int n = arr[i]; }
```

## 10. 最佳实践

### 1. 枚举优于 int 常量

```java
// 不推荐
public static final int STATUS_ACTIVE = 1;
public static final int STATUS_INACTIVE = 0;

// 推荐
public enum Status { ACTIVE, INACTIVE }
```

### 2. 避免不必要的装箱

```java
// 不推荐 (每次循环都装箱/拆箱)
Long sum = 0L;
for (long i = 0; i < Integer.MAX_VALUE; i++) {
    sum += i;  // 巨大的性能开销
}

// 推荐
long sum = 0L;
for (long i = 0; i < Integer.MAX_VALUE; i++) {
    sum += i;
}
```

### 3. 用 EnumSet 替代位标志

```java
// 不推荐 (C 风格位标志)
public static final int BOLD = 1;
public static final int ITALIC = 2;
public static final int UNDERLINE = 4;
int style = BOLD | ITALIC;

// 推荐
enum Style { BOLD, ITALIC, UNDERLINE }
Set<Style> style = EnumSet.of(Style.BOLD, Style.ITALIC);
```
