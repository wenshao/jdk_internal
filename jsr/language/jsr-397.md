# JSR 397: Sealed Classes

> **状态**: ✅ Final | **JDK**: 17 | **链接**: https://jcp.org/en/jsr/detail?id=397

---

## 概述

JSR 397 定义了 Sealed Classes (密封类) 的语言规范，允许类/接口限制其子类型。

---

## 核心语法

```java
// 密封抽象类
public sealed class Shape 
    permits Circle, Rectangle, Triangle {
    
    public abstract double area();
}

// 允许的子类
public final class Circle extends Shape {
    private final double radius;
    
    public Circle(double radius) { this.radius = radius; }
    
    @Override
    public double area() { return Math.PI * radius * radius; }
}

public final class Rectangle extends Shape {
    private final double width, height;
    
    public Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }
    
    @Override
    public double area() { return width * height; }
}

// non-sealed 允许进一步继承
public non-sealed class Triangle extends Shape {
    // 可以被其他类继承
}
```

---

## 密封接口

```java
// 密封接口
public sealed interface Expr 
    permits ConstantExpr, BinaryExpr, UnaryExpr {
    
    int evaluate();
}

public record ConstantExpr(int value) implements Expr {
    @Override
    public int evaluate() { return value; }
}

public record BinaryExpr(Expr left, String op, Expr right) implements Expr {
    @Override
    public int evaluate() {
        return switch (op) {
            case "+" -> left.evaluate() + right.evaluate();
            case "-" -> left.evaluate() - right.evaluate();
            case "*" -> left.evaluate() * right.evaluate();
            default -> throw new IllegalArgumentException(op);
        };
    }
}
```

---

## 子类修饰符

| 修饰符 | 说明 |
|--------|------|
| `final` | 不可继承 |
| `sealed` | 继续密封 |
| `non-sealed` | 开放继承 |

```java
sealed class A permits B, C, D { }
final class B extends A { }      // 不可继承
sealed class C extends A permits E { }  // 继续密封
non-sealed class D extends A { }  // 开放继承
```

## 模式匹配结合

```java
// 密封类 + switch 模式匹配
double calculateArea(Shape shape) {
    return switch (shape) {
        case Circle c -> Math.PI * c.radius() * c.radius();
        case Rectangle r -> r.width() * r.height();
        // 编译器知道所有子类，无需 default
    };
}

// instanceof 模式匹配
if (shape instanceof Circle c) {
    System.out.println("Circle with radius: " + c.radius());
}
```

## 穷尽性检查

密封类允许编译器进行穷尽性检查：

```java
// 编译器确保处理了所有情况
String describe(Shape shape) {
    return switch (shape) {
        case Circle c -> "Circle";
        case Rectangle r -> "Rectangle";
        case Triangle t -> "Triangle";
        // 如果 Shape 新增子类，编译错误
    };
}
```

## 使用场景

### 1. 代数数据类型 (ADT)

```java
public sealed interface Option<T> permits Some, None { }
public record Some<T>(T value) implements Option<T> { }
public record None<T>() implements Option<T> { }
```

### 2. 访问者模式

```java
public sealed interface AST permits Node, Leaf { }

public interface Visitor<T> {
    T visit(Node node);
    T visit(Leaf leaf);
}
```

### 3. 状态机

```java
public sealed interface State permits Idle, Running, Stopped { }
```
