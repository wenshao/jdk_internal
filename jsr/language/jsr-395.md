# JSR 395: Records

> **状态**: ✅ Final | **JDK**: 16 (2021-03) | **链接**: https://jcp.org/en/jsr/detail?id=395

---

## 概述

JSR 395 定义了 Record (记录类) 的语言规范，用于简洁地声明不可变数据载体。这是 Java 数据建模的重大改进，与 Sealed Classes 和 Pattern Matching 紧密配合。

## 源码分析

### java.lang.Record

```java
// 来源: src/java.base/share/classes/java/lang/Record.java
package java.lang;

/**
 * 所有 Java 语言 record 类的公共基类。
 *
 * record 类是一种浅不可变、透明的值载体。
 * 它有以下强制成员：
 * - 规范构造器
 * - 每个组件对应的 private final 字段
 * - 每个组件对应的公共访问器方法
 * - equals, hashCode, toString 方法
 */
public abstract class Record {
    /**
     * 子类必须重写此方法
     */
    @Override
    public abstract boolean equals(Object obj);

    @Override
    public abstract int hashCode();

    @Override
    public abstract String toString();
}
```

## 核心语法

```java
// 基本语法
public record Point(int x, int y) { }

// 使用
Point p = new Point(10, 20);
int x = p.x();
int y = p.y();
String str = p.toString();  // "Point[x=10, y=20]"
```

## 自定义方法

```java
public record Point(int x, int y) {
    // 静态方法
    public static Point origin() {
        return new Point(0, 0);
    }
    
    // 实例方法
    public double distance() {
        return Math.sqrt(x * x + y * y);
    }
    
    // 紧凑构造器 (验证)
    public Point {
        if (x < 0 || y < 0) {
            throw new IllegalArgumentException("Coordinates must be non-negative");
        }
    }
}
```

## 实现接口

```java
public record Point(int x, int y) implements Comparable<Point> {
    @Override
    public int compareTo(Point other) {
        return Double.compare(this.distance(), other.distance());
    }
}
```

## 与其他特性结合

### 记录与模式匹配

```java
// instanceof 模式匹配
if (obj instanceof Point(int x, int y)) {
    System.out.println("x=" + x + ", y=" + y);
}

// switch 模式匹配
String describe = switch (obj) {
    case Point(int x, int y) -> "Point at (" + x + ", " + y + ")";
    default -> "Unknown";
};
```

### 记录与密封类

```java
public sealed interface Shape 
    permits Circle, Rectangle { }

public record Circle(double radius) implements Shape { }
public record Rectangle(double width, double height) implements Shape { }
```
