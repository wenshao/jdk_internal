## 概述

JSR 310 引入了全新的日期时间 API，解决了旧的 `Date` 和 `Calendar` 类的所有问题：

| 旧 API 问题 | JSR 310 解决方案 |
|-------------|------------------|
| 月份从 0 开始 | 月份从 1 开始 |
| 线程不安全 | 所有类不可变 |
| API 设计混乱 | 清晰的类型分离 |
| 时区处理困难 | 完善的时区支持 |

## 核心类型

### 时间线

```
┌─────────────────────────────────────────────────────────────────────┐
│                        JSR 310 类型关系                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  Instant (机器时间)                                                  │
│     │                                                               │
│     ▼                                                               │
│  LocalDateTime ←── ZonedDateTime                                    │
│     │                   │                                           │
│     │                   └── ZoneId                                  │
│     │                                                               │
│  LocalDate + LocalTime                                              │
│                                                                     │
│  Duration (时间量)     Period (日期量)                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 本地日期时间

```java
// 日期
LocalDate date = LocalDate.of(2024, 3, 20);
LocalDate today = LocalDate.now();
LocalDate parsed = LocalDate.parse("2024-03-20");

// 时间
LocalTime time = LocalTime.of(10, 30, 45);
LocalTime now = LocalTime.now();

// 日期时间
LocalDateTime dateTime = LocalDateTime.of(2024, 3, 20, 10, 30);
LocalDateTime dt = date.atTime(time);
```

### 带时区的日期时间

```java
// 时区
ZoneId zone = ZoneId.of("Asia/Shanghai");
ZoneId utc = ZoneId.of("UTC");
ZoneId system = ZoneId.systemDefault();

// 带时区的日期时间
ZonedDateTime zdt = ZonedDateTime.of(2024, 3, 20, 10, 30, 0, 0, zone);
ZonedDateTime now = ZonedDateTime.now();

// 转换
ZonedDateTime shanghai = LocalDateTime.now().atZone(zone);
ZonedDateTime newYork = shanghai.withZoneSameInstant(ZoneId.of("America/New_York"));
```

### 时间戳

```java
// Instant - 机器时间
Instant now = Instant.now();
Instant epoch = Instant.EPOCH;
Instant parsed = Instant.parse("2024-03-20T10:30:45Z");

// 时间戳转换
long epochMilli = now.toEpochMilli();
long epochSecond = now.getEpochSecond();
Instant fromMillis = Instant.ofEpochMilli(epochMilli);

// 与 LocalDateTime 转换
LocalDateTime ldt = LocalDateTime.ofInstant(instant, zone);
Instant instant = ldt.atZone(zone).toInstant();
```

### 时间段

```java
// Duration - 时间量 (小时、分钟、秒)
Duration d1 = Duration.ofHours(2);
Duration d2 = Duration.ofMinutes(30);
Duration d3 = Duration.between(time1, time2);

// Period - 日期量 (年、月、日)
Period p1 = Period.ofYears(1);
Period p2 = Period.ofMonths(3);
Period p3 = Period.between(date1, date2);

// 计算
LocalDateTime later = now.plus(d1);
LocalDate future = today.plus(p2);
```

## 格式化与解析

### DateTimeFormatter

```java
// 预定义格式
DateTimeFormatter isoDate = DateTimeFormatter.ISO_LOCAL_DATE;
DateTimeFormatter isoDateTime = DateTimeFormatter.ISO_LOCAL_DATE_TIME;

// 自定义格式
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
DateTimeFormatter formatterWithLocale = DateTimeFormatter
    .ofPattern("yyyy年MM月dd日")
    .withLocale(Locale.CHINA);

// 格式化
String formatted = dateTime.format(formatter);
String formatted2 = formatter.format(dateTime);

// 解析
LocalDateTime parsed = LocalDateTime.parse("2024-03-20 10:30:45", formatter);
```

### 格式符号

| 符号 | 含义 | 示例 |
|------|------|------|
| yyyy | 年 | 2024 |
| MM | 月 | 03 |
| dd | 日 | 20 |
| HH | 时 (24) | 14 |
| hh | 时 (12) | 02 |
| mm | 分 | 30 |
| ss | 秒 | 45 |
| SSS | 毫秒 | 123 |
| EEEE | 星期全称 | Wednesday |
| EEE | 星期简称 | Wed |
| a | AM/PM | AM |
| VV | 时区ID | Asia/Shanghai |
| Z | 时区偏移 | +0800 |

## 日期计算

### plus/minus

```java
LocalDate date = LocalDate.of(2024, 3, 20);

// 加
LocalDate nextWeek = date.plusWeeks(1);
LocalDate nextMonth = date.plusMonths(1);
LocalDate nextYear = date.plusYears(1);

// 减
LocalDate lastWeek = date.minusWeeks(1);

// 使用 ChronoUnit
LocalDate in10Days = date.plus(10, ChronoUnit.DAYS);
LocalDate in3Months = date.plus(3, ChronoUnit.MONTHS);
```

### with (调整)

```java
LocalDate date = LocalDate.of(2024, 3, 20);

// 调整到特定值
LocalDate firstDay = date.withDayOfMonth(1);
LocalDate lastDay = date.withDayOfMonth(date.lengthOfMonth());
LocalDate inDecember = date.withMonth(12);

// 使用 TemporalAdjusters
import java.time.temporal.TemporalAdjusters;

LocalDate nextMonday = date.with(TemporalAdjusters.next(DayOfWeek.MONDAY));
LocalDate lastFriday = date.with(TemporalAdjusters.previous(DayOfWeek.FRIDAY));
LocalDate firstDayOfMonth = date.with(TemporalAdjusters.firstDayOfMonth());
LocalDate lastDayOfMonth = date.with(TemporalAdjusters.lastDayOfMonth());
```

## 时区处理

### ZoneId 和 ZoneOffset

```java
// ZoneId - 时区 ID
ZoneId shanghai = ZoneId.of("Asia/Shanghai");
ZoneId newYork = ZoneId.of("America/New_York");
ZoneId utc = ZoneId.of("UTC");

// ZoneOffset - 时区偏移
ZoneOffset offset8 = ZoneOffset.ofHours(8);
ZoneOffset offset = ZoneOffset.of("+08:00");

// 获取所有时区
Set<String> zones = ZoneId.getAvailableZoneIds();
```

### 时区转换

```java
ZonedDateTime shanghai = ZonedDateTime.now(ZoneId.of("Asia/Shanghai"));

// 保持同一时刻，不同时区显示
ZonedDateTime london = shanghai.withZoneSameInstant(ZoneId.of("Europe/London"));
ZonedDateTime tokyo = shanghai.withZoneSameInstant(ZoneId.of("Asia/Tokyo"));

// 示例
// Shanghai:  2024-03-20T10:30+08:00
// London:    2024-03-20T02:30Z
// Tokyo:     2024-03-20T11:30+09:00
```

## 与旧 API 的互操作

### Date ↔ Instant

```java
// Date → Instant
Date date = new Date();
Instant instant = date.toInstant();

// Instant → Date
Date newDate = Date.from(instant);
```

### Calendar ↔ ZonedDateTime

```java
// Calendar → ZonedDateTime
Calendar cal = Calendar.getInstance();
ZonedDateTime zdt = cal.toInstant().atZone(cal.getTimeZone().toZoneId());

// ZonedDateTime → Calendar
Calendar newCal = GregorianCalendar.from(zdt);
```

### SimpleDateFormat ↔ DateTimeFormatter

```java
// 旧方式
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");
String old = sdf.format(date);

// 新方式
DateTimeFormatter dtf = DateTimeFormatter.ofPattern("yyyy-MM-dd");
String new = localDate.format(dtf);
```

## 最佳实践

### 1. 选择正确的类型

| 场景 | 使用类型 |
|------|----------|
| 生日、纪念日 | LocalDate |
| 闹钟时间 | LocalTime |
| 日程安排（本地） | LocalDateTime |
| 国际会议时间 | ZonedDateTime |
| 日志时间戳 | Instant |
| 计时器、超时 | Duration |
| 年龄、合同期限 | Period |

### 2. 避免错误

```java
// 错误: 直接比较不同时区的时间
if (shanghaiTime.isAfter(londonTime)) { ... }  // 语义不清

// 正确: 转换为同一时区或 Instant
if (shanghaiTime.toInstant().isAfter(londonTime.toInstant())) { ... }

// 错误: 用 Duration 计算日期差
Duration.between(date1, date2);  // 可能不正确 (夏令时)

// 正确: 用 Period 计算日期差
Period.between(date1, date2);
```

### 3. 解析异常处理

```java
// 使用 try-catch
try {
    LocalDate date = LocalDate.parse(input, formatter);
} catch (DateTimeParseException e) {
    // 处理解析失败
}
```

## 性能考量

### 对象创建

```java
// 不可变对象，频繁创建
// 但 JVM 优化良好，通常不是瓶颈
LocalDateTime now = LocalDateTime.now();  // 快速
```

### 格式化

```java
// 重用 formatter (线程安全)
private static final DateTimeFormatter FORMATTER = 
    DateTimeFormatter.ofPattern("yyyy-MM-dd");

// 避免
for (LocalDate date : dates) {
    DateTimeFormatter f = DateTimeFormatter.ofPattern("yyyy-MM-dd");  // 每次创建
}
```
