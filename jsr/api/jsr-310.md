# JSR 310: Date and Time API

> **状态**: ✅ Final | **JDK**: 8 (2014-03) | **链接**: https://jcp.org/en/jsr/detail?id=310

---

## 概述

JSR 310 定义了新的日期时间 API，取代了旧的 `java.util.Date` 和 `java.util.Calendar`。这个 API 基于 **ThreeTen** 项目，由 Stephen Colebourne 主导开发。

## 设计原则

- **不可变**: 所有类都是不可变的
- **线程安全**: 无需同步
- **清晰**: 方法名明确
- **可扩展**: 支持多种日历系统

## 核心类

### 本地日期时间

| 类 | 说明 | 示例 |
|----|------|------|
| `LocalDate` | 日期 (无时区) | 2024-03-20 |
| `LocalTime` | 时间 (无时区) | 10:30:45 |
| `LocalDateTime` | 日期+时间 (无时区) | 2024-03-20T10:30:45 |

```java
LocalDate date = LocalDate.of(2024, 3, 20);
LocalTime time = LocalTime.of(10, 30, 45);
LocalDateTime dateTime = LocalDateTime.of(date, time);

// 当前时间
LocalDate today = LocalDate.now();
LocalDateTime now = LocalDateTime.now();
```

### 时区日期时间

| 类 | 说明 |
|----|------|
| `ZonedDateTime` | 带时区的日期时间 |
| `OffsetDateTime` | 带 UTC 偏移的日期时间 |
| `Instant` | 时间戳 (UTC) |

```java
ZonedDateTime zoned = ZonedDateTime.of(dateTime, ZoneId.of("Asia/Shanghai"));
Instant instant = Instant.now();

// 时区转换
ZonedDateTime tokyo = zoned.withZoneSameInstant(ZoneId.of("Asia/Tokyo"));
```

### 时间段

| 类 | 说明 |
|----|------|
| `Duration` | 精确时间量 (小时/分/秒) |
| `Period` | 日历时间量 (年/月/日) |

```java
Duration duration = Duration.between(start, end);
Period period = Period.between(date1, date2);

// 计算
LocalDate nextWeek = date.plus(Period.ofWeeks(1));
LocalDateTime later = dateTime.plus(Duration.ofHours(2));
```

## 格式化与解析

```java
// 格式化
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
String formatted = dateTime.format(formatter);

// 解析
LocalDate parsed = LocalDate.parse("2024-03-20");
LocalDateTime parsedDateTime = LocalDateTime.parse("2024-03-20 10:30:45", formatter);
```

## 与旧 API 互操作

```java
// Date -> Instant
Instant instant = oldDate.toInstant();

// Instant -> LocalDateTime
LocalDateTime dateTime = LocalDateTime.ofInstant(instant, ZoneId.systemDefault());

// Calendar -> ZonedDateTime
ZonedDateTime zoned = calendar.toInstant().atZone(ZoneId.systemDefault());
```

---

## 最佳实践

```java
// ✅ 使用新的 API
LocalDate date = LocalDate.now();

// ❌ 避免旧 API
Date oldDate = new Date();

// ✅ 使用 Duration 计算时间差
Duration diff = Duration.between(start, end);

// ❌ 避免手动计算毫秒
long diffMs = end.getTime() - start.getTime();
```
