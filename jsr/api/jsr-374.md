## 2. 概述

JSR 374 是 JSON-P 1.1，在 JSR 353 (1.0) 基础上新增 Pointer/Patch/Stream 支持，属于 Java EE 8。JSON-P 是底层处理 API (Processing)，JSON-B (JSR 367) 是对象绑定 API (Binding)，两者互补。

---

## 3. JSON Pointer (RFC 6901)

```java
JsonPointer ptr = Json.createPointer("/store/books/0/title");
JsonString title = (JsonString) ptr.getValue(doc);   // "Java 编程思想"
boolean exists   = ptr.containsValue(doc);           // true
JsonObject updated = ptr.replace(doc, Json.createValue("Effective Java"));
```

**语法**: `"/foo"` 顶层键、`"/foo/0"` 数组元素、`"/a~1b"` 含 `/` 转义、`"/m~0n"` 含 `~` 转义

---

## 4. JSON Patch (RFC 6902)

```java
JsonPatch patch = Json.createPatchBuilder()
    .add("/store/phone", "010-12345678")        // 添加
    .replace("/store/name", "新华书城")           // 替换
    .remove("/store/books/1")                    // 删除
    .move("/store/name", "/store/title")         // 移动
    .copy("/store/phone", "/store/fax")          // 复制
    .test("/store/title", "新华书城")             // 测试值匹配
    .build();

JsonObject patched = patch.apply(doc);
```

**操作类型**: `add`、`remove`、`replace`、`move`、`copy`、`test`（不匹配抛异常）

---

## 5. JSON Merge Patch (RFC 7396)

```java
JsonMergePatch patch = Json.createMergePatch(Json.createObjectBuilder()
    .add("store", Json.createObjectBuilder()
        .add("name", "新华书城")            // 替换值
        .add("phone", "010-12345678")      // 添加新字段
        .addNull("obsoleteField"))         // null = 删除
    .build());
JsonValue result = patch.apply(doc);
```

**Patch vs Merge Patch**: Patch 用操作数组精确控制（支持 move/copy/test）；Merge Patch 用目标结构简化覆盖（null 表示删除，数组只能整体替换）。

---

## 6. JsonCollectors 与 Stream

```java
// Stream → JsonArray
JsonArray names = books.stream()
    .map(b -> Json.createValue(b.getString("title")))
    .collect(JsonCollectors.toJsonArray());

// Stream → JsonObject
JsonObject priceMap = books.stream().collect(JsonCollectors.toJsonObject(
    b -> b.getString("title"), b -> b.getJsonNumber("price")));
```

## 7. 与 JSR 353 的关系

```
JSR 353 (JSON-P 1.0, 2013, EE 7): JsonParser, JsonGenerator, JsonObject, JsonArray
JSR 374 (JSON-P 1.1, 2017, EE 8): + JsonPointer + JsonPatch + JsonMergePatch + JsonCollectors
Jakarta (JSON-P 2.0+, 2020, EE 9): jakarta.json 包名迁移
```
