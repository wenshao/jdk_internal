## 2. 概述

JSR 367 定义了 Java 对象与 JSON 的标准绑定层 (Standard Binding Layer)，Java EE 8 组成部分：

| 特点 | 说明 |
|------|------|
| 约定优于配置 | 零注解即可完成基本序列化 (Convention over Configuration) |
| 与 JSON-P 互补 | JSON-B 处理绑定，JSON-P (JSR 374) 处理底层解析 |
| 标准化 | Java EE/Jakarta EE 平台标准，类似 JAXB 之于 XML |

---

## 3. 核心 API

```java
import javax.json.bind.Jsonb;
import javax.json.bind.JsonbBuilder;

Jsonb jsonb = JsonbBuilder.create();

// 序列化 (Serialization): Object → JSON
String json = jsonb.toJson(new Person("张三", 30));  // {"name":"张三","age":30}

// 反序列化 (Deserialization): JSON → Object
Person person = jsonb.fromJson(json, Person.class);

jsonb.close();  // 实现 AutoCloseable
```

## 4. 注解详解

```java
public class Employee {
    @JsonbProperty("full_name")           // 属性名映射 (Property Mapping)
    private String name;
    @JsonbTransient                        // 跳过字段 (Skip Field)
    private String password;
    @JsonbDateFormat("yyyy-MM-dd HH:mm")  // 日期格式 (Date Format)
    private LocalDateTime joinDate;
    @JsonbNumberFormat("#,##0.00")        // 数字格式 (Number Format)
    private BigDecimal salary;
}

@JsonbPropertyOrder({"id", "name", "email"})  // 属性顺序
public class Contact { ... }

// @JsonbCreator — 不可变对象构造 (Immutable Object Construction)
public Money(@JsonbProperty("amount") BigDecimal amount,
             @JsonbProperty("currency") String currency) { ... }
```

## 5. 自定义配置

```java
JsonbConfig config = new JsonbConfig()
    .withFormatting(true)                       // 美化输出 (Pretty Print)
    .withNullValues(true)                       // 包含 null 值
    .withPropertyNamingStrategy(PropertyNamingStrategy.LOWER_CASE_WITH_UNDERSCORES)
    .withDateFormat("yyyy-MM-dd", Locale.CHINA);
Jsonb jsonb = JsonbBuilder.create(config);
```

**命名策略**: `IDENTITY`(默认)、`LOWER_CASE_WITH_UNDERSCORES`、`LOWER_CASE_WITH_DASHES`、`UPPER_CAMEL_CASE`

## 6. 与 Jackson/Gson 对比

| 特性 | JSON-B (JSR 367) | Jackson | Gson |
|------|------------------|---------|------|
| 标准化 | Java EE/Jakarta 标准 | 第三方 (de facto) | 第三方 |
| 注解 | `@JsonbProperty` | `@JsonProperty` | `@SerializedName` |
| 忽略字段 | `@JsonbTransient` | `@JsonIgnore` | `@Expose` |
| 性能 | 良好 | 优秀 (最快) | 一般 |
| 生态 | Jakarta EE 集成 | Spring Boot 默认 | Android 常用 |

**选择建议**: Jakarta EE 应用优先 JSON-B；Spring Boot 通常用 Jackson。

---

## 9. 最佳实践

1. **优先使用约定** — 仅在需要时添加注解 (Minimize Annotations)
2. **复用 Jsonb 实例** — 线程安全，避免反复创建
3. **@JsonbTransient 保护敏感数据** — 密码、令牌绝不序列化
4. **@JsonbCreator 支持不可变对象** — 配合 final 字段与构造函数
