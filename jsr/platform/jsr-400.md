## 2. 概述

JSR 400 是 Java SE 25 的平台 JSR，定义了 JDK 25 的完整平台规范。JDK 25 是继 JDK 21 之后的下一个长期支持版本 (Long-Term Support, LTS)，计划于 2025 年 9 月发布。

| 属性 | 值 |
|------|-----|
| 平台版本 | Java SE 25 |
| 发布日期 | 2025-09 (计划) |
| 支持类型 | LTS (长期支持) |
| 上一个 LTS | JDK 21 (2023-09) |
| 涵盖 JDK 版本 | JDK 22、23、24、25 四个版本的累积特性 |

**JDK 25 是企业升级的重要目标版本**，汇聚了 JDK 22-25 四个版本中从预览走向正式的所有特性。

## 3. 正式特性

### 语言特性 (Language Features)

| JEP | 特性 | 说明 |
|-----|------|------|
| JEP 492 | Flexible Constructor Bodies | 构造函数中 super() 前可执行语句 |
| JEP 494 | Module Import Declarations | `import module java.base` 导入整个模块 |
| JEP 495 | Compact Source Files | 简化单文件程序，隐式类声明 |

```java
// Flexible Constructor Bodies — 灵活构造函数体
public class LoggedWidget extends Widget {
    public LoggedWidget(String name) {
        // super() 之前可以执行验证和准备逻辑
        Objects.requireNonNull(name);
        var processed = name.strip().toLowerCase();
        super(processed);
    }
}

// Module Import — 模块导入
import module java.base;  // 导入 java.base 所有公开包

// Compact Source Files — 紧凑源文件
// 文件: hello.java (无需 class 声明)
void main() {
    println("Hello, Java 25!");
}
```

### 并发与运行时 (Concurrency & Runtime)

| JEP | 特性 | 说明 |
|-----|------|------|
| JEP 487 | Scoped Values | 线程内/跨虚拟线程的作用域值传递 |
| JEP 491 | Structured Concurrency | 结构化并发，简化多线程编程 |
| JEP 485 | Stream Gatherers | 自定义中间流操作 |
| JEP 484 | Class-File API | 标准类文件读写 API |

```java
// Scoped Values — 作用域值
static final ScopedValue<User> CURRENT_USER = ScopedValue.newInstance();

ScopedValue.runWhere(CURRENT_USER, user, () -> {
    handleRequest();  // 内部可通过 CURRENT_USER.get() 获取
});

// Structured Concurrency — 结构化并发
try (var scope = StructuredTaskScope.open()) {
    Subtask<String> name = scope.fork(() -> fetchName());
    Subtask<Integer> age = scope.fork(() -> fetchAge());
    scope.join();
    return new Person(name.get(), age.get());
}
```

### 性能优化 (Performance)

| JEP | 特性 | 说明 |
|-----|------|------|
| JEP 483 | Ahead-of-Time Class Loading | AOT 类加载/编译，加速启动 |
| JEP 450 | Compact Object Headers | 对象头从 128 位压缩到 64 位 |

## 4. 预览与孵化特性

JDK 25 中仍在预览或孵化的特性（未计入平台标准）：

| JEP | 特性 | 状态 |
|-----|------|------|
| JEP 499 | Structured Concurrency (Preview) | 第四轮预览 |
| JEP 498 | Warn upon Use of Memory-Access Methods in sun.misc.Unsafe | 弃用警告 |
| JEP 501 | Deprecate 32-bit x86 Port | 弃用 32 位 x86 |

## 8. LTS 升级指南

### 从 JDK 21 升级到 JDK 25 的注意事项

| 类别 | 事项 |
|------|------|
| sun.misc.Unsafe | 内存访问方法将显示弃用警告，迁移至 VarHandle/FFM |
| 32 位 x86 | 已弃用，计划未来版本移除 |
| SecurityManager | 继续弃用状态 |
| 模块系统 | Module Import 简化迁移 |
| 构造函数 | Flexible Constructor Bodies 可简化子类代码 |
