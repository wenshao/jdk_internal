# JSR 376: Java Platform Module System

> **状态**: ✅ Final | **JDK**: 9 | **链接**: https://jcp.org/en/jsr/detail?id=376

---

## 概述

JSR 376 定义了 Java 平台模块系统 (JPMS)，也称为 Project Jigsaw，是 Java 9 最重要的特性。

---

## 核心概念

### 模块 (Module)

模块是包和资源的集合，具有明确的依赖声明。

```java
// module-info.java
module com.example.myapp {
    // 依赖
    requires java.sql;
    requires transitive java.logging;
    
    // 导出
    exports com.example.api;
    exports com.example.internal to com.example.tests;
    
    // 服务
    uses com.example.spi.Service;
    provides com.example.spi.Service with com.example.impl.ServiceImpl;
}
```

### 关键字

| 关键字 | 说明 |
|--------|------|
| `module` | 定义模块 |
| `requires` | 声明依赖 |
| `requires transitive` | 传递依赖 |
| `requires static` | 可选依赖 |
| `exports` | 导出包 |
| `opens` | 开放反射访问 |
| `uses` | 使用服务 |
| `provides...with` | 提供服务实现 |

## 模块类型

### 具名模块 (Named Module)

```java
// module-info.java
module my.module {
    requires other.module;
}
```

### 自动模块 (Automatic Module)

```java
// 无 module-info.java，放在 modulepath 上
// 模块名来自 JAR 文件名
```

### 未命名模块 (Unnamed Module)

```java

// 可访问所有模块导出的包
```

## JDK 模块化

### Java SE 模块

```bash
# 查看所有 JDK 模块
java --list-modules

# 关键模块
java.base          # 核心类库
java.sql           # JDBC
java.logging       # java.util.logging
java.desktop       # AWT, Swing
java.xml           # XML 处理
```

### 紧凑配置

```bash
# jlink 创建自定义运行时
jlink --add-modules java.base,java.sql \
      --output /path/to/custom-jre
```

## 兼容性影响

### 强封装

```java
// JDK 内部 API 不再可访问
// --add-opens 绕过 (不推荐)
java --add-opens java.base/java.lang=ALL-UNNAMED
```

### 迁移策略

1. **无 module-info.java**: 放在 classpath
2. **自动模块**: 放在 modulepath
3. **具名模块**: 添加 module-info.java
