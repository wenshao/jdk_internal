# JDK-8366224: Introduce DecimalDigits.appendPair for efficient two-digit formatting and refactor DateTimeHelper

> **Status**: Integrated
> **Impact**: ⭐⭐⭐⭐ (+12% performance)

## Summary

引入 `DecimalDigits.appendPair` 方法，用于高效格式化两位数字（如月份、日期、小时等），并重构 `DateTimeHelper` 使用该新方法。变更还扩展到 `MonthDay`、`YearMonth`、`ZoneOffset` 和 `ChronoLocalDateImpl` 的 `toString()` 方法。底层通过 `AbstractStringBuilder.appendLatin1(char c1, char c2)` 和 `JavaLangAccess.appendPair` 实现。

---

## Problem Analysis

### 原始实现

在日期时间格式化中，经常需要将 0-99 的整数格式化为两位数字:

```java
// 原始实现
private void appendTwoDigits(int value, char[] buffer, int index) {
    if (value < 10) {
        buffer[index] = '0';
        buffer[index + 1] = (char)('0' + value);
    } else if (value < 100) {
        buffer[index] = (char)('0' + value / 10);
        buffer[index + 1] = (char)('0' + value % 10);
    } else {
        throw new IllegalArgumentException();
    }
}
```

**问题**:
1. 多次条件判断
2. 除法和取模操作
3. 重复的边界检查

## Solution

### 新的 DecimalDigits 类

```java
public class DecimalDigits {
    // 预计算 0-99 的两位数字表示
    private static final byte[] DIGITS = new byte[200];
    
    static {
        for (int i = 0; i < 100; i++) {
            DIGITS[i * 2] = (byte)('0' + i / 10);
            DIGITS[i * 2 + 1] = (byte)('0' + i % 10);
        }
    }
    
    /**
     * 将 0-99 的整数格式化为两位数字，追加到 byte 数组
     * @param value 要格式化的值 (0-99)
     * @param buf 目标缓冲区
     * @param index 起始索引
     */
    public static void appendPair(int value, byte[] buf, int index) {
        buf[index] = DIGITS[value * 2];
        buf[index + 1] = DIGITS[value * 2 + 1];
    }
    
    /**
     * 将 0-99 的整数格式化为两位数字，追加到 char 数组
     */
    public static void appendPair(int value, char[] buf, int index) {
        buf[index] = (char)DIGITS[value * 2];
        buf[index + 1] = (char)DIGITS[value * 2 + 1];
    }
}
```

## Usage Examples

```java
// 日期格式化
void formatDate(LocalDate date, char[] buf, int index) {
    DecimalDigits.appendPair(date.getYear() / 100, buf, index);
    DecimalDigits.appendPair(date.getYear() % 100, buf, index + 2);
    buf[index + 4] = '-';
    DecimalDigits.appendPair(date.getMonthValue(), buf, index + 5);
    buf[index + 7] = '-';
    DecimalDigits.appendPair(date.getDayOfMonth(), buf, index + 8);
}

// 时间格式化
void formatTime(LocalTime time, char[] buf, int index) {
    DecimalDigits.appendPair(time.getHour(), buf, index);
    buf[index + 2] = ':';
    DecimalDigits.appendPair(time.getMinute(), buf, index + 3);
    buf[index + 5] = ':';
    DecimalDigits.appendPair(time.getSecond(), buf, index + 6);
}
```

## Technical Details

### 为什么更快?

1. **消除条件分支**: 使用查表法替代 if-else
2. **消除除法/取模**: 预计算所有可能的值
3. **内存局部性**: 查表数据在 L1 缓存中

### 内存开销

- DIGITS 数组: 200 bytes (0-99 的两位数字)
- 每个数字 2 bytes (十位和个位)
