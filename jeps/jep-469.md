# JEP 469: Implicit Classes and Instance Main Methods

> **状态**: 🔍 预览版 (JDK 25, 第4次预览) | **JEP 链接**: https://openjdk.org/jeps/469

---

## 概述

隐式类简化单文件程序的编写，无需显式类声明。

---

## 核心特性

### 无类声明

```java
// 单文件程序无需类声明
void main() {
    System.out.println("Hello, World!");
}
```

### 实例 main 方法

```java
// 支持无 static 修饰的 main 方法
void main(String[] args) {
    for (String arg : args) {
        System.out.println(arg);
    }
}
```
