# JEP 382: Metal Rendering Pipeline

> **Status**: Permanent | **Type**: Feature | **Component**: Client Libraries
>
> **Lead**: Philip Race (Oracle)
>
> **JDK Version**: JDK 14 (2020-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

为 macOS 引入 Metal 渲染管道，替代废弃的 OpenGL。

---

## 目标

- 实现 Metal 渲染
- 替代 OpenGL
- 提高 macOS 图形性能
- 支持现代 macOS 版本

---

## 技术细节

### Metal vs OpenGL

| 特性 | OpenGL | Metal |
|------|--------|-------|
| **API 类型** | 跨平台 | Apple 专有 |
| **性能** | 中等 | 高 |
| **支持** | macOS 10.14 废弃 | macOS 10.11+ |
| **功耗** | 高 | 低 |

### Java2D Metal

```java
// 启用 Metal 渲染
System.setProperty("sun.java2d.metal", "true");

// 或使用 VM 参数
java -Dsun.java2d.metal=true MyApp
```
