# JEP 441: Pattern Matching for switch

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 21 (2023-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

将 switch 的模式匹配从预览提升为标准功能。允许 switch 表达式使用类型模式、Record 模式和 guard 条件，简化复杂条件分支。

---

## 目标

- 简化类型分支代码
- 支持 Record 解构
- 支持 guard 条件
- 成为标准功能

---

## 语法

### JDK 20 及之前

```java
// 传统 switch
String format(Object obj) {
    if (obj instanceof Integer i) {
        return "int: " + i;
    } else if (obj instanceof Long l) {
        return "long: " + l;
    } else if (obj instanceof String s) {
        return "string: " + s;
    } else {
        return "unknown";
    }
}
```

### JDK 21+

```java
// 模式匹配 switch
String format(Object obj) {
    return switch (obj) {
        case Integer i -> "int: " + i;
        case Long l -> "long: " + l;
        case String s -> "string: " + s;
        default -> "unknown";
    };
}
```

## 关键特性

### 1. 类型模式

```java
double getValue(Number n) {
    return switch (n) {
        case Integer i -> i.doubleValue();
        case Long l -> l.doubleValue();
        case Float f -> f.doubleValue();
        case Double d -> d;
    };
}
```

### 2. Record 模式

```java
record Point(int x, int y) {}
record Rectangle(Point ul, Point lr) {}

String describe(Shape shape) {
    return switch (shape) {
        case Point(int x, int y) -> 
            "Point at (" + x + ", " + y + ")";
        case Rectangle(Point ul, Point lr) -> 
            "Rectangle";
        default -> "Unknown";
    };
}
```

### 3. Guard 条件

```java
String classify(Number n) {
    return switch (n) {
        case Integer i when i > 0 -> "Positive int";
        case Integer i when i < 0 -> "Negative int";
        case Integer i -> "Zero";
        default -> "Non-integer";
    };
}
```

### 4. 嵌套模式

```java
String getAuthorName(Book book) {
    return switch (book) {
        case Book(String title, Author(String name, _), _) -> 
            name;
        default -> "Unknown";
    };
}
```

### 5. null 处理

```java
String check(Object obj) {
    return switch (obj) {
        case null -> "null value";
        case String s -> "String: " + s;
        default -> "Other";
    };
}
```

## 使用场景

### 推荐

```java
// AST 处理
sealed interface Expr {}
record Lit(int val) implements Expr {}
record Add(Expr l, Expr r) implements Expr {}
record Mul(Expr l, Expr r) implements Expr {}

int eval(Expr e) {
    return switch (e) {
        case Lit(int v) -> v;
        case Add(Expr l, Expr r) -> eval(l) + eval(r);
        case Mul(Expr l, Expr r) -> eval(l) * eval(r);
    };
}

// API 响应
record Result<T>(int code, T data, String msg) {}

<R> R handleResult(Result<R> result) {
    return switch (result) {
        case Result(200, R data, _) -> data;
        case Result(_, _, String msg) -> 
            throw new ApiException(msg);
    };
}

// 事件处理
String handleEvent(Event event) {
    return switch (event) {
        case ClickEvent(int x, int y) -> 
            "Click at (" + x + ", " + y + ")";
        case KeyEvent(String key) when "Enter".equals(key) -> 
            "Enter pressed";
        case KeyEvent(String key) -> 
            "Key: " + key;
    };
}
```

### 不推荐

```java
// ❌ 过度复杂
return switch (obj) {
    case A(B(C(D(int x)))) when x > 0 && x < 100 -> ...;
    // 太复杂，难以维护
};

// 推荐：分层处理
```

## 编译后

```java
// 源代码
return switch (obj) {
    case Integer i -> i * 2;
    case String s -> s.length();
    default -> 0;
};

// 编译后（简化）
if (obj instanceof Integer i) {
    return i * 2;
} else if (obj instanceof String s) {
    return s.length();
} else {
    return 0;
}
```
