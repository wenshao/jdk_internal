# JEP 361: Switch Expressions (Standard)

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 14 (2020-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

将 Switch Expressions 从预览功能提升为标准功能。这是 [JEP 325](jep-325.md) 的后续，基于社区反馈进行了完善。

---

## 目标

- 将 Switch Expressions 标准化
- 移除预览标志要求
- 推荐用于生产环境

---

## 变化总结

### JDK 13 (预览) vs JDK 14 (标准)

| 特性 | JDK 13 Preview | JDK 14 Standard |
|------|----------------|-----------------|
| **启用标志** | `--enable-preview` | 无需 |
| **语法** | 相同 | 相同 |
| **yield 关键字** | 支持 | 支持 |
| **生产就绪** | ❌ | ✅ |

## 完整语法

```java
// 1. 简单表达式
int value = switch (day) {
    case MONDAY, FRIDAY -> 6;
    case TUESDAY -> 7;
    default -> 8;
};

// 2. 带代码块
String result = switch (code) {
    case 1 -> {
        log.info("Code 1");
        yield "One";
    }
    case 2 -> {
        log.info("Code 2");
        yield "Two";
    }
    default -> "Unknown";
};

// 3. 作为方法参数
processResult(switch (status) {
    case SUCCESS -> Result.OK;
    case ERROR -> Result.FAIL;
});

// 4. 抛出异常
int value = switch (input) {
    case 1, 2, 3 -> input * 10;
    default -> throw new IllegalArgumentException();
};
```

## 迁移指南

### 从传统 switch 迁移

```java
// 传统写法
public String getDayType(Day day) {
    String type;
    switch (day) {
        case MONDAY:
        case TUESDAY:
        case WEDNESDAY:
        case THURSDAY:
        case FRIDAY:
            type = "Weekday";
            break;
        case SATURDAY:
        case SUNDAY:
            type = "Weekend";
            break;
        default:
            throw new IllegalArgumentException();
    }
    return type;
}

// 新写法
public String getDayType(Day day) {
    return switch (day) {
        case MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY -> "Weekday";
        case SATURDAY, SUNDAY -> "Weekend";
        default -> throw new IllegalArgumentException();
    };
}
```

## 最佳实践

### 推荐

```java
// ✅ 用于枚举映射
String name = switch (color) {
    case RED -> "Red";
    case GREEN -> "Green";
    case BLUE -> "Blue";
};

// ✅ 用于状态转换
State next = switch (current) {
    case IDLE -> RUNNING;
    case RUNNING -> STOPPED;
    case STOPPED -> IDLE;
};

// ✅ 用于验证
int level = switch (priority) {
    case HIGH, CRITICAL -> 1;
    case MEDIUM -> 2;
    case LOW -> 3;
    default -> throw new IllegalStateException();
};
```

### 不推荐

```java
// ❌ 过度嵌套
var result = switch (a) {
    case 1 -> switch (b) {
        case 2 -> switch (c) {
            case 3 -> "deep";
            default -> "no";
        };
        default -> "no";
    };
    default -> "no";
};

// 推荐：使用 if-else 或提前返回
```

## 性能对比

| 场景 | 传统 switch | Switch 表达式 | 差异 |
|------|------------|--------------|------|
| 枚举查找 | 100% | 100% | 无差异 |
| 整数分支 | 100% | 100% | 无差异 |
| 字符串匹配 | 100% | 100% | 无差异 |

**性能相同**，因为编译后字节码基本一致。
