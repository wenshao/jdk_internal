# JEP 359: Records (Preview)

> **Status**: Preview | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 14 (2020-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 Record 类预览，提供不可变数据载体。

---

## 语法

```java
// 传统写法
public final class Point {
    private final int x;
    private final int y;
    
    public Point(int x, int y) {
        this.x = x;
        this.y = y;
    }
    
    // getters, equals, hashCode, toString...
}

// Record
public record Point(int x, int y) {}
```
