## 1. 概述

允许在模式匹配中使用原始类型 (primitive types)，避免包装类型的拆箱开销。

---

## 2. 核心特性

### instanceof 模式匹配

```java
Object obj = 42;

// 直接使用原始类型模式
if (obj instanceof int i) {
    System.out.println("int: " + i);
}
```

### switch 模式匹配

```java
String result = switch (getValue()) {
    case int i -> "int: " + i;
    case long l -> "long: " + l;
    case double d -> "double: " + d;
    case String s -> "String: " + s;
    default -> "unknown";
};
```
