# JEP 511: Module Import Declarations

> **Status**: Closed / Delivered
> **Release**: JDK 25
> **Owner**: Ron Pressler
> **Issue**: [JDK-8331029](https://bugs.openjdk.org/browse/JDK-8331029)

## 1. 概述

引入模块导入声明，允许导入整个模块的公共 API，简化模块化代码的编写。

## 2. 关键特性

- `import module` 语法
- 导入模块的所有公共包
- 与现有导入语句兼容
- 简化模块化应用开发

## 3. 示例

```java
// 导入整个模块
import module java.sql;

// 现在可以直接使用模块中的所有公共类
Connection conn = DriverManager.getConnection(url);
Statement stmt = conn.createStatement();
```

## 4. 动机

1. **简化代码**: 减少大量单独的 import 语句
2. **模块化开发**: 更好地支持模块化应用
3. **学习曲线**: 降低 Java 初学者的入门门槛
