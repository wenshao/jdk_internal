## 1. 概述

文本块 (Text Blocks) 是 Java 的多行字符串字面量，旨在简化嵌入代码、SQL、JSON 等多行文本的编写。

---

## 2. 语法

### 基本语法

```java
// 传统方式 (JDK 13 之前)
String json = "{\n" +
    "  \"name\": \"John\",\n" +
    "  \"age\": 30\n" +
    "}\n";

// Text Blocks (JDK 15+)
String json = """
    {
      "name": "John",
      "age": 30
    }
    """;
```

### 语法规则

```
"""  [开始定界符]
    <内容>
"""  [结束定界符]
```

**规则**：
1. 以 `"""` 开始
2. 内容可以跨多行
3. 以 `"""` 结束
4. 结束的 `"""` 位置决定缩进

## 3. 编译时处理

### 1. 行终止符规范化

```java
// 输入 (混合 \n 和 \r\n)
String text = """
    line1
    line2
    """;

// 输出 (统一为 \n)
// "line1\nline2\n"
```

### 2. 移除附带的空白

```java
// 附带空白 = 结束 """ 左侧的空白
String html = """
...............
        <html>
            <body>Hello</body>
        </html>
...............
    """.replace(".", " ");
// 结果: "<html>\n    <body>Hello</body>\n</html>\n"
```

### 3. 转义序列处理

```java
String text = """
    文本块中支持 \n, \t, \", 等转义
    """;
```

---

## 4. 新增方法

### stripIndent()

```java
// 移除每行的共同前导空白
String text = """
    Line 1
      Line 2
    Line 3
    """.stripIndent();
// 结果:
// "Line 1\n  Line 2\nLine 3\n"
```

### translateEscapes()

```java
// 处理转义序列
String text = "\\n\\t".translateEscapes();
// 结果: "\n\t"
```

### formatted(Object... args)

```java
// JDK 15 新增
String text = """
    Hello %s!
    You are %d years old.
    """.formatted("Alice", 25);
// 结果:
// "Hello Alice!\nYou are 25 years old.\n"
```

## 5. 常见用例

### JSON

```java
String json = """
    {
        "name": "%s",
        "age": %d,
        "active": %b
    }
    """.formatted(name, age, active);
```

### SQL

```java
String query = """
    SELECT id, name, email
    FROM users
    WHERE status = 'ACTIVE'
      AND created_at > '%s'
    ORDER BY name
    """.formatted(date);
```

### HTML

```java
String html = """
    <html>
        <head><title>%s</title></head>
        <body>
            <h1>Welcome</h1>
        </body>
    </html>
    """.formatted(title);
```

### 正则表达式

```java
String regex = """
    \\b           # 单词边界
    [A-Z]+       # 大写字母
    \\s+         # 空白
    [a-z]+       # 小写字母
    \\b          # 单词边界
    """;
```

## 6. 重要特性

### 1. 保留尾随空格

```java
String text = """
    line1
    line2
    """;
// line1 有 3 个空格, line2 有 2 个空格
```

### 2. 禁止尾随逗号

```java
// ❌ 编译错误
String text = """
    line1,
    line2,
    """;

// ✅ 正确
String text = """
    line1,
    line2
    """;
```

### 3. \s 转义 (JDK 15 新增)

```java
// \s = 单个空格 (避免 stripIndent 移除)
String text = """
    line1\s
    line2\s
    """.stripIndent();
// 每行末尾保留一个空格
```

### 4. \ 行终止 (JDK 15 新增)

```java
// \ = 取消行终止
String text = """
    line1 \
    line2 \
    line3
    """;
// 结果: "line1 line2 line3\n"
```

## 7. 性能考虑

### 编译时

```java
// 编译器将 Text Block 转换为常量
// 等价于:
String text = "line1\nline2\nline3\n";

// 没有运行时开销
```

### 运行时

| 操作 | 性能 |
|------|------|
| 字面量 Text Block | 常量加载 |
| formatted() | 与 String.format() 类似 |
| stripIndent() | O(n) 扫描 |

## 10. 最佳实践

### ✅ 推荐

```java
// 用于多行文本字面量
String json = """
    {"key": "value"}
    """;

// 用于代码模板
String code = """
    public class %s {
        public static void main(String[] args) {
            System.out.println("Hello");
        }
    }
    """.formatted(className);
```

### ❌ 避免

```java
// 单行字符串 (不如用 "")
String single = """
    text
    """;

// 复杂字符串拼接 (用 String Templates... 等等，它被撤回了)
```
