## 1. 概述

JEP 482 是 Flexible Constructor Bodies 的第二轮预览（首次预览为 JEP 447，JDK 22）。该特性允许在构造器中的 `super()` 或 `this()` 调用之前放置语句，打破了 Java 长期以来构造器体第一条语句必须是超类构造器调用的限制。

---

## 2. 变更内容

在传统 Java 中，构造器的第一条语句必须是 `super(...)` 或 `this(...)` 调用。该特性放宽了这一限制，允许在显式构造器调用之前插入不引用正在构造的实例的语句，例如参数验证、参数计算等。

---

## 3. 与前一预览版的差异

相较于 JEP 447（JDK 22 首次预览，当时名为 "Statements before super()"）：

- **重命名**：特性名称从 "Statements before super()" 改为 "Flexible Constructor Bodies"，更准确地描述其范围。
- **扩展能力**：除了在 `super()` 前执行语句外，还允许在调用 `super()` 前初始化当前类声明的字段，以便超类构造器回调时这些字段已有值。

---

## 4. 代码示例

```java
// JDK 23 预览：在 super() 之前验证参数
public class PositiveInteger extends Number {
    private final int value;

    public PositiveInteger(int value) {
        if (value <= 0) throw new IllegalArgumentException("Must be positive");
        this.value = value; // 在 super() 前初始化字段
        super();
    }
}
```

在此特性之前，参数验证需要使用辅助静态方法等变通方案。
