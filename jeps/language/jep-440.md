# JEP 440: Record Patterns

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 21 (2023-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

扩展模式匹配支持 Record 类型，允许在 instanceof 和 switch 中解构 Record，简化数据访问代码。

---

## 目标

- 简化 Record 数据访问
- 支持嵌套模式匹配
- 与现有模式匹配一致
- 减少样板代码

---

## 语法

### JDK 20 及之前

```java
// 传统写法
record Point(int x, int y) {}
record Rectangle(Point upperLeft, Point lowerRight) {}

Object obj = getShape();
if (obj instanceof Rectangle r) {
    Point ul = r.upperLeft();
    Point lr = r.lowerRight();
    if (ul instanceof Point p1 && lr instanceof Point p2) {
        int width = p2.x() - p1.x();
        int height = p2.y() - p1.y();
    }
}
```

### JDK 21+

```java
// Record 模式匹配
if (obj instanceof Rectangle(Point ul, Point lr)) {
    // 直接解构
    int width = lr.x() - ul.x();
    int height = lr.y() - ul.y();
}

// 嵌套模式
if (obj instanceof Rectangle(Point(int x1, int y1), Point(int x2, int y2))) {
    // 完全解构
    int width = x2 - x1;
    int height = y2 - y1;
}
```

## 关键特性

### 1. Switch 中的 Record 模式

```java
String describe(Shape shape) {
    return switch (shape) {
        case Point(int x, int y) -> 
            "Point at (" + x + ", " + y + ")";
        case Rectangle(Point ul, Point lr) -> 
            "Rectangle from " + ul + " to " + lr;
        case Circle(Point center, double radius) -> 
            "Circle at " + center + " with radius " + radius;
        default -> "Unknown shape";
    };
}
```

### 2. 嵌套 Record 模式

```java
record Author(String name, String email) {}
record Book(String title, Author author, int year) {}

void process(Book book) {
    if (book instanceof Book(String title, Author(String name, _), int year)) {
        System.out.println(title + " by " + name + " (" + year + ")");
    }
}
```

### 3. 泛型 Record 模式

```java
record Pair<T>(T first, T second) {}

void process(Pair<Integer> pair) {
    if (pair instanceof Pair<Integer>(Integer first, Integer second)) {
        System.out.println(first + ", " + second);
    }
    
    // 类型推断（JDK 21+）
    if (pair instanceof Pair(var first, var second)) {
        // first 和 second 推断为 Integer
    }
}
```

### 4. 带条件的模式

```java
String classify(Shape shape) {
    return switch (shape) {
        case Circle c when c.radius() < 1 -> "Small circle";
        case Circle c when c.radius() < 10 -> "Medium circle";
        case Circle c -> "Large circle";
        default -> "Not a circle";
    };
}
```

## 使用场景

### 推荐

```java
// AST 处理
sealed interface Expr {}
record Literal(int value) implements Expr {}
record Add(Expr left, Expr right) implements Expr {}

int eval(Expr expr) {
    return switch (expr) {
        case Literal(int v) -> v;
        case Add(Expr l, Expr r) -> eval(l) + eval(r);
    };
}

// API 响应处理
record Response<T>(int status, T data, String error) {}

void handle(Response<User> resp) {
    if (resp instanceof Response(200, User user, _)) {
        System.out.println("User: " + user.name());
    } else if (resp instanceof Response(_, _, String err)) {
        System.err.println("Error: " + err);
    }
}
```

### 不推荐

```java
// ❌ 过度嵌套
if (obj instanceof A(B(C(D(int x))))) {
    // 太深嵌套，难以阅读
}

// 推荐：分层处理
```

## 编译后

```java
// 源代码
if (obj instanceof Rectangle(Point ul, Point lr)) {
    System.out.println(ul.x());
}

// 编译后（简化）
if (obj instanceof Rectangle) {
    Rectangle r = (Rectangle) obj;
    Point ul = r.upperLeft();
    Point lr = r.lowerRight();
    if (ul != null && lr != null) {
        System.out.println(ul.x());
    }
}
```
