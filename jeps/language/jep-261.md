# JEP 261: Module System

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Alan Bateman (Oracle)
>
> **JDK Version**: JDK 9 (2017-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 Java 平台模块系统（Project Jigsaw），为 Java SE 提供模块化结构，增强封装性、可维护性和安全性。

---

## 目标

- 提供可靠的封装
- 支持模块化 JDK
- 提高安全性
- 支持可选配置

---

## 语法

### 模块声明

```java
// module-info.java
module com.example.myapp {
    requires java.base;
    requires java.sql;
    
    exports com.example.api;
    exports com.example.spi to com.example.impl;
    
    uses com.example.spi.ServiceProvider;
    provides com.example.spi.ServiceProvider 
        with com.example.impl.ProviderImpl;
}
```

### 模块指令

| 指令 | 说明 |
|------|------|
| `requires` | 依赖其他模块 |
| `exports` | 导出包 |
| `opens` | 开放包（反射） |
| `uses` | 使用服务 |
| `provides` | 提供服务 |

---

## 关键特性

### 1. 封装

```java
// 模块内部
package com.example.internal;
class InternalClass {}  // 外部不可见

// module-info.java
module com.example {
    exports com.example.api;  // 只导出公共 API
    // com.example.internal 不导出
}
```

### 2. 可读性

```java
// 显式声明依赖
module com.example {
    requires java.sql;
    requires com.example.api;
}
```

### 3. 服务机制

```java
// 服务接口
module com.example.api {
    exports com.example.spi;
    uses com.example.spi.Provider;
}

// 服务实现
module com.example.impl {
    provides com.example.spi.Provider 
        with com.example.impl.ProviderImpl;
}
```

## 模块化 JDK

### JDK 9 模块结构

```
java.base        // 基础模块（隐式依赖）
java.sql         // JDBC
java.logging     // 日志
java.xml         // XML 处理
java.desktop     // AWT/Swing
jdk.compiler     // 编译器
...
```

### 运行时配置

```bash
# 列出模块
java --list-modules

# 仅包含必要模块
jlink --add-modules java.base,java.logging \
      --output custom-jre

# 启动时指定模块
java --module com.example.myapp/com.example.Main
```

## 迁移指南

### 传统应用

```bash
# 无需修改，运行在类路径上
java -jar myapp.jar
```

### 模块化应用

```bash
# 模块路径
java --module-path mods \
     --module com.example.myapp
```
