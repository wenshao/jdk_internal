## 1. 概述

简化 Java 入门，允许隐式类声明和实例 main 方法。

---

## 2. 核心特性

### 隐式类

```java
// 无需类声明
void main() {
    System.out.println("Hello, World!");
}
```

### 实例 main 方法

```java
// 可以是非静态的
void main() {
    System.out.println("Hello!");
}
```
