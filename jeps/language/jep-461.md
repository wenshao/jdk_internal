> **Status**: Preview | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Paul Sandoz (Oracle)
>
> **JDK Version**: JDK 23 (2024-09)
>

## 概览

引入 Stream Gatherers API，允许自定义流操作，扩展 Stream API 的功能。

---

## 目标

- 提供自定义流操作的 API
- 支持复杂的流转换
- 保持与现有 Stream API 兼容
- 通过预览收集反馈

---

## 核心 API

### Gatherer 接口

```java
// 自定义流操作
Gatherer<T, A, R> gatherer = Gatherer.of(
    () -> new ArrayList<A>(),  // 供应器
    (input, downstream) -> {   // 整合器
        // 处理逻辑
        downstream.push(transform(input));
    },
    (combiner) -> {},          // 合并器
    (accumulator, downstream) -> { // 结束器
        // 清理逻辑
    }
);

// 使用自定义 gatherer
stream.gather(gatherer)
      .collect(Collectors.toList());
```

### 内置 Gatherers

```java
// 限制
stream.gather(Gatherers.limit(10))

// 去重
stream.gather(Gatherers.distinct())

// 折叠
stream.gather(Gatherers.fold(() -> "", 
    (acc, elem) -> acc + elem))

// 扫描
stream.gather(Gatherers.scan(() -> 0, 
    (acc, elem) -> acc + elem))
```

## 使用场景

### 自定义转换

```java
// 批处理
Gatherer<List<T>, List<T>, List<T>> batch(int size) {
    return Gatherer.of(
        () -> new ArrayList<>(),
        (list, acc, downstream) -> {
            list.add(elem);
            if (list.size() >= size) {
                downstream.push(new ArrayList<>(list));
                list.clear();
            }
        }
    );
}

// 使用
stream.gather(batch(100))
      .forEach(this::processBatch);
```

### 状态ful 操作

```java
// 滑动窗口
Gatherer<T, Queue<T>, List<T>> window(int size) {
    return Gatherer.of(
        () -> new ArrayDeque<>(),
        (window, elem, downstream) -> {
            window.add(elem);
            if (window.size() > size) {
                window.poll();
            }
            if (window.size() == size) {
                downstream.push(new ArrayList<>(window));
            }
        }
    );
}
```
