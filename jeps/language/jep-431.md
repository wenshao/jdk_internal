# JEP 431: Sequenced Collections

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Paul Sandoz (Oracle)
>
> **JDK Version**: JDK 21 (2023-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

为集合 API 引入有序集合接口，提供统一的访问第一个和最后一个元素的方法。

---

## 目标

- 提供统一的有序集合 API
- 支持访问首尾元素
- 支持反向迭代
- 保持向后兼容

---

## 核心接口

### SequencedCollection

```java
// 新接口
public interface SequencedCollection<E> extends Collection<E> {
    // 访问首尾元素
    E getFirst();
    E getLast();
    
    // 移除首尾元素
    E removeFirst();
    E removeLast();
    
    // 添加首尾元素
    void addFirst(E e);
    void addLast(E e);
    
    // 反向迭代
    SequencedCollection<E> reversed();
}
```

### SequencedSet

```java
public interface SequencedSet<E> extends SequencedCollection<E>, Set<E> {
    @Override
    SequencedSet<E> reversed();
}
```

### SequencedMap

```java
public interface SequencedMap<K,V> extends Map<K,V> {
    // 访问首尾条目
    Entry<K,V> firstEntry();
    Entry<K,V> lastEntry();
    
    // 添加首尾条目
    V putFirst(K key, V value);
    V putLast(K key, V value);
    
    // 移除首尾条目
    Entry<K,V> pollFirstEntry();
    Entry<K,V> pollLastEntry();
    
    // 反向视图
    SequencedMap<K,V> reversed();
}
```

## 使用方式

### List

```java
List<String> list = new ArrayList<>(List.of("A", "B", "C"));

// 访问首尾
String first = list.getFirst();  // "A"
String last = list.getLast();    // "C"

// 添加首尾
list.addFirst("Start");
list.addLast("End");

// 移除首尾
String removed = list.removeFirst();  // "Start"
String removed = list.removeLast();   // "End"

// 反向迭代
for (String s : list.reversed()) {
    System.out.println(s);
}
```

### Set

```java
Set<String> set = new LinkedHashSet<>(List.of("A", "B", "C"));

// 访问首尾
String first = set.getFirst();  // "A"
String last = set.getLast();    // "C"
```

### Map

```java
Map<String, Integer> map = new LinkedHashMap<>();
map.put("A", 1);
map.put("B", 2);
map.put("C", 3);

// 访问首尾条目
Entry<String, Integer> first = map.firstEntry();  // A=1
Entry<String, Integer> last = map.lastEntry();    // C=3

// 反向遍历
for (Entry<String, Integer> e : map.reversed().entrySet()) {
    System.out.println(e);
}
```
