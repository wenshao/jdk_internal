# JEP 305: Pattern Matching for instanceof (Preview)

> **Status**: Preview | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 14 (2020-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 instanceof 模式匹配预览，简化类型检查和转换。

---

## 语法

```java
// 传统写法
if (obj instanceof String) {
    String s = (String) obj;
    System.out.println(s);
}

// 新模式匹配
if (obj instanceof String s) {
    System.out.println(s);
}
```
