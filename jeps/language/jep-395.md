# JEP 395: Records

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 16 (2021-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 Record 类，一种不可变的透明数据载体，自动生成构造函数、getter、equals、hashCode 和 toString 方法。

---

## 目标

- 简化数据类编写
- 提供不可变数据载体
- 减少样板代码
- 与现有类系统兼容

---

## 语法

### 基本用法

```java
// JDK 15 及之前
public final class Point {
    private final int x;
    private final int y;
    
    public Point(int x, int y) {
        this.x = x;
        this.y = y;
    }
    
    public int x() { return x; }
    public int y() { return y; }
    
    @Override
    public boolean equals(Object o) { /* ... */ }
    
    @Override
    public int hashCode() { /* ... */ }
    
    @Override
    public String toString() { /* ... */ }
}

// JDK 16+
public record Point(int x, int y) {}
```

### 自动生成方法

```java
public record Point(int x, int y) {}

// 编译器自动生成：
// - Point(int x, int y)  构造函数
// - int x()              getter 方法
// - int y()              getter 方法
// - boolean equals(Object)
// - int hashCode()
// - String toString()
```

## 关键特性

### 1. 紧凑构造函数

```java
public record Rectangle(Point upperLeft, Point lowerRight) {
    // 紧凑构造函数 - 验证参数
    public Rectangle {
        if (upperLeft.x > lowerRight.x) {
            throw new IllegalArgumentException();
        }
    }
}
```

### 2. 自定义方法

```java
public record Point(int x, int y) {
    // 自定义方法
    public double distance() {
        return Math.sqrt(x * x + y * y);
    }
    
    // 静态方法
    public static Point origin() {
        return new Point(0, 0);
    }
}
```

### 3. 局部 Record

```java
void process() {
    // 局部 Record
    record Pair<T>(T first, T second) {}
    
    Pair<String> pair = new Pair<>("a", "b");
}
```

### 4. 模式匹配

```java
// instanceof 模式匹配
if (obj instanceof Point(int x, int y)) {
    System.out.println("Point at (" + x + ", " + y + ")");
}

// switch 模式匹配
return switch (shape) {
    case Point(int x, int y) -> "Point at " + x + "," + y;
    case Rectangle(Point ul, Point lr) -> "Rectangle";
    default -> "Unknown";
};
```

## 使用场景

### 推荐

```java
// DTO/数据传输对象
public record UserDTO(Long id, String name, String email) {}

// 配置对象
public record Config(String host, int port, boolean ssl) {}

// 计算结果
public record Result<T>(T value, boolean success, String error) {}

// 集合元素
public record Entry<K,V>(K key, V value) {}

// 模式匹配
public record Circle(Point center, double radius) {}
public record Rectangle(Point upperLeft, Point lowerRight) {}

public double area(Shape shape) {
    return switch (shape) {
        case Circle c -> Math.PI * c.radius() * c.radius();
        case Rectangle r -> {
            double width = r.lowerRight().x() - r.upperLeft().x();
            double height = r.lowerRight().y() - r.upperLeft().y();
            yield width * height;
        }
    };
}
```

### 不推荐

```java
// ❌ 需要可变状态
public record MutablePoint(int x, int y) {
    public void setX(int x) { /* 不能修改 final 字段 */ }
}

// ❌ 需要继承
public record SpecialPoint(int x, int y) extends Point { }  // 编译错误

// ❌ 复杂业务逻辑
public record ComplexBusinessLogic(...) {
    // Record 应该简单，复杂逻辑应该放在普通类中
}
```

## 特性对比

| 特性 | Record | 传统类 | Lombok @Data |
|------|--------|--------|--------------|
| **不可变性** | ✅ 自动 | ❌ 手动 | ❌ |
| **构造函数** | ✅ 自动 | ❌ 手动 | ✅ |
| **Getter** | ✅ 自动 | ❌ 手动 | ✅ |
| **equals/hashCode** | ✅ 自动 | ❌ 手动 | ✅ |
| **toString** | ✅ 自动 | ❌ 手动 | ✅ |
| **继承** | ❌ 不能 | ✅ | ✅ |
| **可变字段** | ❌ 不能 | ✅ | ✅ |
| **序列化** | ✅ 特殊 | ✅ | ✅ |

## 编译后

```java
// 源代码
public record Point(int x, int y) {}

// 编译后（简化）
public final class Point extends Record {
    private final int x;
    private final int y;
    
    public Point(int x, int y) {
        this.x = x;
        this.y = y;
    }
    
    public int x() { return x; }
    public int y() { return y; }
    
    @Override
    public boolean equals(Object o) { /* 基于所有字段 */ }
    
    @Override
    public int hashCode() { /* 基于所有字段 */ }
    
    @Override
    public String toString() { /* Point[x=1, y=2] */ }
}
```
