## 1. 概述

引入 `_` 作为未命名变量/模式，用于不使用的值。

---

## 2. 核心特性

### 未命名变量

```java
// 在 lambda 中
BiFunction<Integer, Integer, String> add = (x, _) -> "Sum: " + (x + y);

// 在 try-with-resources 中
try (var _ = ScopedContext.acquire()) {
    // ...
}

// 在 for-each 中
for (var _ : listOfItems) {
    // 只关心循环次数
}
```

### 未命名模式

```java
// 在 record 模式中
if (obj instanceof Point(int x, _)) {
    // 只关心 x，忽略 y
}
```
