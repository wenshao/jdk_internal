## 1. 背景与动机

### 问题

Java 长期以来被批评入门门槛高，一个简单的 "Hello World" 需要大量样板代码：

```java
// 传统 Java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

对于初学者，需要理解：
- 类声明 (`public class`)
- `static` 修饰符
- `void` 返回类型
- `String[] args` 参数
- `System.out.println`

### 解决方案

JEP 512 进一步简化，允许：

1. **隐式类**: 无需显式类声明
2. **实例 main 方法**: 无需 `static`
3. **简化的 IO**: 新的 `java.lang.IO` 类

## 2. 使用示例

### 演进对比

```java
// JDK 1.0 - 20: 传统方式
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

// JDK 21 - 24: 隐式类 + 静态 main
void main(String[] args) {
    System.out.println("Hello, World!");
}

// JDK 25+: 隐式类 + 实例 main + 简化 IO
void main() {
    println("Hello, World!");
}
```

### 完整示例

```java
// Hello.java - 最简形式
void main() {
    println("Hello, World!");
}
```

编译和运行：
```bash
java Hello.java
# 输出: Hello, World!
```

```java
// Calculator.java
import module java.base;

void main() {
```

## 3. 实现分析

### 隐式类原理

编译器自动将顶层语句包装在隐式类中：

```java
// 源码: Hello.java
void main() {
    println("Hello");
}

// 编译后等价于
public class Hello {
    public static void main(String[] args) throws Throwable {
        new Hello().main();
    }
    
    void main() {
        IO.println("Hello");
    }
}
```

### IO 类实现

```java
// src/java.base/share/classes/java/lang/IO.java
public final class IO {
    
    private static BufferedReader reader;
    
    /**
     * 读取一行输入
     */
    public static String readln() {
        return readln("");
    }
    
    /**
     * 带提示的读取
     */
    public static String readln(String prompt) {
        if (reader == null) {
            initReader();
        }
        // ... 读取实现
    }
    
    /**
     * 打印一行
     */
    public static void println(Object obj) {
        System.out.println(obj);
    }
    
    /**
     * 打印（无换行）
     */
    public static void print(Object obj) {
        System.out.print(obj);
    }
}
```

### main 方法查找顺序

编译器按以下顺序查找 main 方法：

1. `static void main(String[] args)`
2. `static void main()`
3. `void main(String[] args)` (实例方法)
4. `void main()` (实例方法)

## 4. 开发者影响

### 迁移指南

| 场景 | 建议 |
|------|------|
| 教学场景 | 强烈推荐，大幅简化入门 |
| 脚本工具 | 适合单文件工具类 |
| 生产代码 | 谨慎使用，考虑团队规范 |
| 库开发 | 不适用 |

### 兼容性

- **完全向后兼容**: 传统写法继续有效
- **文件名要求**: 隐式类文件名必须与生成的类名匹配
- **包限制**: 隐式类不能有包声明

### 最佳实践

```java
// ✅ 适合：简单工具脚本
// convert.java
import module java.base;

void main() {
    var input = readln("Enter Celsius: ");
    var celsius = Double.parseDouble(input);
    var fahrenheit = celsius * 9 / 5 + 32;
    println("Fahrenheit: " + fahrenheit);
}

// ✅ 适合：教学示例
void main() {
    println("Let's learn Java!");
}

// ❌ 不适合：复杂应用
// 建议使用传统类结构
public class Application {
    private final Config config;
    
    public Application(Config config) {
        this.config = config;
    }
    
    public static void main(String[] args) {
        var app = new Application(Config.fromArgs(args));
        app.run();
    }
}
```
