# JEP 443: Unnamed Patterns and Variables (Preview)

> **Status**: Preview | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 19 (2022-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入未命名模式和变量，简化代码中不需要的绑定。

---

## 语法

```java
// 传统写法
if (obj instanceof Point(int x, int y)) {
    // 如果只需要 y
    if (y > 0) { ... }
}

// 使用未命名模式
if (obj instanceof Point(int _, int y)) {
    if (y > 0) { ... }
}

// 未命名变量
for (String _ : list) {
    count++;
}
```
