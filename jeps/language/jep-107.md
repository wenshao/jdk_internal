# JEP 107: Collections

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 8 (2014-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 Stream API 和集合的批量操作，支持函数式风格的集合数据处理。

---

## 目标

- 提供集合的批量操作 API
- 支持函数式编程风格
- 支持并行处理
- 保持与现有集合 API 兼容

---

## 核心 API

### Stream 接口

```java
// 创建 Stream
Stream<String> stream = list.stream();
Stream<String> parallel = list.parallelStream();

// 中间操作
stream.filter(s -> s.length() > 3)
      .map(String::toUpperCase)
      .sorted();

// 终端操作
long count = stream.count();
Optional<String> first = stream.findFirst();
List<String> collected = stream.toList();
```

### 常用操作

```java
// 过滤
list.stream().filter(s -> s.startsWith("A"))

// 映射
list.stream().map(String::length)

// 归约
int sum = numbers.stream().reduce(0, Integer::sum)

// 收集
List<String> result = stream.collect(Collectors.toList())
Set<String> unique = stream.collect(Collectors.toSet())
Map<String, Integer> map = stream.collect(
    Collectors.toMap(k -> k, v -> v.length())
)

// 分组
Map<Integer, List<String>> byLength = 
    stream.collect(Collectors.groupingBy(String::length))
```

---

## 并行处理

```java
// 并行流
list.parallelStream()
    .filter(...)
    .map(...)
    .collect(...);

// 顺序流转并行
list.stream().parallel()

// 并行流转顺序
list.parallelStream().sequential()
```

---

## 使用场景

### 数据处理

```java
// 复杂查询
List<String> result = users.stream()
    .filter(u -> u.getAge() > 18)
    .map(User::getName)
    .sorted()
    .limit(10)
    .collect(Collectors.toList());

// 统计
DoubleSummaryStatistics stats = numbers.stream()
    .mapToDouble(Double::doubleValue)
    .summaryStatistics();
```

### 集合转换

```java
// List 转 Map
Map<String, User> map = users.stream()
    .collect(Collectors.toMap(User::getId, u -> u));

// 去重
List<String> unique = list.stream()
    .distinct()
    .collect(Collectors.toList());
```

## 性能考虑

### 适用场景

- ✅ 大数据集处理
- ✅ 可并行化操作
- ✅ 复杂数据转换

### 不适用场景

- ❌ 小数据集（<100 元素）
- ❌ 简单循环可完成的场景
- ❌ 需要精确控制迭代的场景
