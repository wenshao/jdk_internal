# JEP 325: Switch Expressions

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 14 (2020-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

扩展 `switch` 语句支持表达式形式，允许 `switch` 返回值，并引入箭头语法简化 case 分支。

---

## 目标

- 简化 switch 代码
- 消除 fall-through 错误
- 支持作为表达式返回值
- 保持向后兼容

---

## 语法演进

### JDK 13 预览

```java
// 传统 switch 语句
switch (day) {
    case MONDAY:
    case FRIDAY:
    case SUNDAY:
        System.out.println(6);
        break;
    case TUESDAY:
        System.out.println(7);
        break;
    default:
        System.out.println(8);
}

// 新 switch 表达式（预览）
switch (day) {
    case MONDAY, FRIDAY, SUNDAY -> System.out.println(6);
    case TUESDAY -> System.out.println(7);
    default -> System.out.println(8);
}
```

### JDK 14+ 标准

```java
// 作为表达式返回值
int numLetters = switch (day) {
    case MONDAY, FRIDAY, SUNDAY -> 6;
    case TUESDAY -> 7;
    case WEDNESDAY, THURSDAY -> 8;
};

// 使用 yield 返回值
String result = switch (day) {
    case MONDAY -> {
        System.out.println("Monday");
        yield "Start of week";
    }
    case FRIDAY -> {
        System.out.println("Friday");
        yield "End of week";
    }
    default -> "Midweek";
};
```

---

## 关键特性

### 1. 箭头语法

```java
// 传统：需要 break
case MONDAY:
    System.out.println("Monday");
    break;

// 新：自动终止
case MONDAY -> System.out.println("Monday");
```

### 2. 多 case 合并

```java
// 传统
case MONDAY:
case FRIDAY:
case SUNDAY:
    // ...

// 新
case MONDAY, FRIDAY, SUNDAY -> // ...
```

### 3. 表达式返回值

```java
// 直接返回
int value = switch (x) {
    case 1 -> 100;
    case 2 -> 200;
    default -> 0;
};
```

### 4. yield 关键字

```java
// 复杂逻辑时使用 yield
String description = switch (grade) {
    case A -> "Excellent";
    case B -> "Good";
    case C, D -> {
        System.out.println("Passing");
        yield "Average";
    }
    default -> "Failing";
};
```

## 编译后

```java
// 源代码
int value = switch (day) {
    case MONDAY, FRIDAY -> 6;
    case TUESDAY -> 7;
    default -> 8;
};

// 字节码（简化）
int value;
switch (day.ordinal()) {
    case 0: // MONDAY
    case 4: // FRIDAY
        value = 6;
        break;
    case 1: // TUESDAY
        value = 7;
        break;
    default:
        value = 8;
}
```

## 优势

| 特性 | 传统 switch | 新 switch | 改进 |
|------|------------|-----------|------|
| **Fall-through** | 容易出错 | 自动终止 | ✅ |
| **代码行数** | 多 | 少 | -50% |
| **返回值** | 不支持 | 支持 | ✅ |
| **多 case** | 重复写 | 逗号分隔 | ✅ |

---

## 使用场景

### 推荐

```java
// 枚举映射
String name = switch (day) {
    case MONDAY -> "Monday";
    case TUESDAY -> "Tuesday";
    // ...
};

// 类型转换
int code = switch (status) {
    case SUCCESS -> 200;
    case ERROR -> 500;
    default -> 400;
};
```

### 不推荐

```java
// 复杂逻辑（使用传统 if-else）
var result = switch (complexCondition()) {
    case true -> /* 复杂逻辑 */;
    case false -> /* 复杂逻辑 */;
};
```
