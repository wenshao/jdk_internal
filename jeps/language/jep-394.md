# JEP 394: Pattern Matching for instanceof

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 16 (2021-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

扩展 `instanceof` 运算符支持模式匹配，将类型检查和类型转换合并为一步，减少样板代码。

---

## 目标

- 简化类型检查和转换
- 减少强制转换错误
- 提高代码可读性
- 为 switch 模式匹配铺路

---

## 语法

### JDK 15 及之前

```java
// 传统写法
if (obj instanceof String) {
    String s = (String) obj;
    System.out.println(s.toLowerCase());
}
```

### JDK 16+

```java
// 新模式匹配
if (obj instanceof String s) {
    System.out.println(s.toLowerCase());
}
```

---

## 关键特性

### 1. 模式变量

```java
// 基本用法
if (obj instanceof String s) {
    // s 自动转换为 String 类型
    System.out.println(s.length());
}

// 模式变量作用域
if (!(obj instanceof String s)) {
    return;  // s 不在此作用域
}
// s 在此作用域可用
System.out.println(s);
```

### 2. 逻辑运算

```java
// AND 运算
if (obj instanceof String s && s.length() > 5) {
    System.out.println("Long string: " + s);
}

// OR 运算
if (obj instanceof String s || obj instanceof Integer i) {
    // s 和 i 在各自分支可用
}
```

### 3. 作用域规则

```java
// 模式变量在 if-else 中可用
if (obj instanceof String s) {
    System.out.println("String: " + s);
} else {
    // s 不在此作用域
    System.out.println("Not a String");
}

// 重新赋值会遮蔽模式变量
if (obj instanceof String s) {
    String s = "new";  // 编译错误：变量已存在
}
```

## 使用场景

### 推荐

```java
// equals 方法
@Override
public boolean equals(Object o) {
    if (o instanceof Point p) {
        return x == p.x && y == p.y;
    }
    return false;
}

// 集合处理
for (Object item : items) {
    if (item instanceof String s) {
        processString(s);
    } else if (item instanceof Integer i) {
        processInteger(i);
    }
}

// 事件处理
if (event instanceof MouseEvent me) {
    handleMouseEvent(me);
} else if (event instanceof KeyEvent ke) {
    handleKeyEvent(ke);
}
```

### 不推荐

```java
// 过度嵌套
if (obj instanceof Outer o) {
    if (o.inner instanceof Inner i) {
        if (i.value instanceof String s) {
            // 太深嵌套
        }
    }
}
```

## 编译后

```java
// 源代码
if (obj instanceof String s) {
    System.out.println(s);
}

// 字节码（简化）
if (obj instanceof String) {
    String s = (String) obj;  // 隐式转换
    System.out.println(s);
}
```

**编译后与传统写法相同**，只是语法糖。
