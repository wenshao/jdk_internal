## 1. 概述

Vector API 提供在支持 SIMD (单指令多数据) 指令的 CPU 上进行向量运算的能力，使 Java 开发者能够编写可移植的高性能数值计算代码。

**第11次孵化** (JDK 26 GA 2026-03-17) 的主要变更：
- FFM API 集成 - 数学库支持迁移到 FFM API
- VectorShuffle 与 MemorySegment 交互
- UMIN/UMAX 归约支持
- Float16 重构和优化
- VectorShape 公开访问

## 2. 核心概念

### VectorSpecies (向量种类)

```java
import jdk.incubator.vector.*;

// 预定义种类
VectorSpecies<Float> species = FloatVector.SPECIES_PREFERRED;  // 平台最优
VectorSpecies<Float> s256 = FloatVector.SPECIES_256;           // 256-bit (8 floats)
VectorSpecies<Float> s512 = FloatVector.SPECIES_512;           // 512-bit (16 floats)

// 查询信息
int length = species.length();        // 向量元素数
int bitSize = species.vectorBitSize(); // 向量位宽
```

### 向量操作

```java
static final VectorSpecies<Float> SPECIES = FloatVector.SPECIES_PREFERRED;

void vectorComputation(float[] a, float[] b, float[] c) {
    int i = 0;
    int upperBound = SPECIES.loopBound(a.length);

    for (; i < upperBound; i += SPECIES.length()) {
        FloatVector va = FloatVector.fromArray(SPECIES, a, i);
        FloatVector vb = FloatVector.fromArray(SPECIES, b, i);
        
        // 向量运算: c = a * a + b * b
        FloatVector vc = va.mul(va).add(vb.mul(vb));
        vc.intoArray(c, i);
    }

    // 处理尾部元素
    for (; i < a.length; i++) {
        c[i] = a[i] * a[i] + b[i] * b[i];
    }
}
```

## 3. JDK 26 新特性

### 1. FFM API 集成

```java
// JDK 26+: 数学库使用 FFM API
import java.lang.foreign.*;

// Vector API 数学函数现在通过 FFM API 调用原生库
FloatVector v = FloatVector.fromArray(species, data, 0);
FloatVector sin = v.lanewise(VectorOperators.SIN);  // 使用 FFM API
```

### 2. VectorShuffle 与 MemorySegment

```java
import java.lang.foreign.*;

// JDK 26+: 从 MemorySegment 创建 Shuffle
MemorySegment segment = ...
VectorShuffle<Integer> shuffle = VectorShuffle.fromArray(species, segment, 0);

// 应用 Shuffle
IntVector v = IntVector.fromArray(species, array, 0);
IntVector shuffled = v.rearrange(shuffle);

// 将 Shuffle 存储到 MemorySegment
shuffle.intoMemorySegment(segment, 0, ByteOrder.nativeOrder());
```

### 3. UMIN/UMAX 归约

```java
// JDK 26+: 无符号最小/最大归约
IntVector a = IntVector.fromArray(species, array1, 0);
IntVector b = IntVector.fromArray(species, array2, 0);

// 无符号比较
IntVector min = a.lanewise(VectorOperators.UMIN, b);
IntVector max = a.lanewise(VectorOperators.UMAX, b);

// 归约
int unsignedMin = min.reduceLanes(VectorOperators.UMIN);
int unsignedMax = max.reduceLanes(VectorOperators.UMAX);
```

### 4. VectorShape 公开

```java
// JDK 26+: VectorShape#largestShapeFor 公开访问
VectorShape shape = VectorShape.largestShapeFor(int.class);
System.out.println("最大向量形状: " + shape);
```

## 7. 性能对比

### 向量加法 (1M 元素)

| 实现方式 | 耗时 | 加速比 |
|----------|------|--------|
| 标量循环 | ~4ms | 1x |
| Vector API (SSE) | ~1ms | 4x |
| Vector API (AVX2) | ~0.5ms | 8x |
| Vector API (AVX-512) | ~0.25ms | 16x |

### SAXPY (单精度浮点乘加)

```java
// 标量版本
void saxpyScalar(float[] a, float[] x, float[] y, float alpha) {
    for (int i = 0; i < a.length; i++) {
        a[i] = alpha * x[i] + y[i];
    }
}

// 向量版本 (AVX-512: 16 floats/iteration)
void saxpyVector(float[] a, float[] x, float[] y, float alpha) {
    VectorSpecies<Float> species = FloatVector.SPECIES_PREFERRED;
    int i = 0, bound = species.loopBound(a.length);

    for (; i < bound; i += species.length()) {
        FloatVector vx = FloatVector.fromArray(species, x, i);
        FloatVector vy = FloatVector.fromArray(species, y, i);
        vx.mul(alpha).add(vy).intoArray(a, i);
    }

    for (; i < a.length; i++) {
        a[i] = alpha * x[i] + y[i];
    }
}
```

## 9. 为什么孵化这么久？

Vector API 已孵化 11 个版本 (JDK 16-26)，主要原因：

1. **SVE 可变长度** - ARM SVE 向量长度运行时确定 (128-2048 bits)
2. **跨平台一致性** - x86/ARM/RISC-V 指令差异大
3. **C2 编译器优化** - 需要大量 intrinsic 实现
4. **测试覆盖** - 平台数 × 操作数 × 类型数 = 巨大组合

**预计毕业时间**: JDK 27-28 (2026-2027)
