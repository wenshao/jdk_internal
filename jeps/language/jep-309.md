# JEP 309: Dynamic Class-File Constants

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: John Rose (Oracle)
>
> **JDK Version**: JDK 11 (2018-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入动态类文件常量池，允许在运行时解析常量，提高类文件灵活性和性能。

---

## 目标

- 扩展类文件常量池
- 支持运行时解析
- 提高引导性能
- 支持语言实现

---

## 技术细节

### CONSTANT_Dynamic

```java
// 类文件常量池扩展
// CONSTANT_Dynamic 结构
CONSTANT_Dynamic {
    u1 tag;
    u2 bootstrap_method_attr_index;
    u2 constant_pool_index;
}
```

### 引导方法

```java
// 引导方法解析动态常量
public static Object bootstrap(MethodHandles.Lookup caller,
                                String name,
                                Class<?> type) {
    // 运行时解析常量
    return computeConstantValue();
}
```

### 使用场景

```java
// Lambda 元数据
// 使用 invokedynamic 和动态常量
invokedynamic #bootstrap, #name, #type

// 字符串模板
// 使用动态常量存储模板
```

---

## 优势

| 特性 | 传统方式 | 动态常量 |
|------|----------|----------|
| **解析时机** | 类加载时 | 运行时 |
| **灵活性** | 低 | 高 |
| **性能** | 中等 | 优化后更高 |
| **语言支持** | 有限 | 广泛 |
