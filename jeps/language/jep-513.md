## 1. 概述

灵活的构造函数体允许在构造函数中更自由地组织代码，特别是在调用 super() 之前执行语句。

---

## 2. 核心特性

### 构造函数初始化

```java
// 之前：必须先调用 super()
public class MyClass extends Base {
    private final int value;

    public MyClass(int value) {
        super();  // 必须是第一行
        this.value = validate(value);  // 无法在 super 前执行
    }
}

// 现在：可以在 super() 前执行语句
public class MyClass extends Base {
    private final int value;

    public MyClass(int value) {
        int validated = validate(value);  // 可以在 super 前执行
        super();
        this.value = validated;
    }

    private int validate(int v) {
        if (v < 0) throw new IllegalArgumentException();
        return v;
    }
}
```

### 验证逻辑前置

```java
public class Person extends Entity {
    private final String name;
    private final int age;

    public Person(String name, int age) {
        // 在调用父构造函数前验证参数
        Objects.requireNonNull(name, "name cannot be null");
        if (age < 0 || age > 150) {
            throw new IllegalArgumentException("Invalid age: " + age);
        }

        super();  // 验证完成后调用父构造函数

        this.name = name;
        this.age = age;
    }
}
```

---

## 3. 限制

- 不能在显式构造函数调用之前访问实例字段
- 不能在显式构造函数调用之前调用实例方法
- 可以访问静态字段和方法
- 可以创建和初始化局部变量
