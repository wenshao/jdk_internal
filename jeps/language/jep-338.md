# JEP 338: Vector API (Incubator)

> **Status**: Incubator | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Sandhya Viswanathan (Intel)
>
> **JDK Version**: JDK 16 (2021-03)
>
> **Organization**: [Intel](../../contributors/orgs/intel.md)

---

## 概览

引入向量 API 孵化器，提供表达向量计算的 API，编译时映射到最优的 CPU 向量指令。

---

## 目标

- 提供向量计算 API
- 映射到硬件向量指令
- 提高 SIMD 性能
- 收集社区反馈

---

## 核心 API

```java
// 定义向量类型
VectorSpecies<Float> SPECIES = FloatVector.SPECIES_256;

// 创建向量
FloatVector a = FloatVector.fromArray(SPECIES, array1, 0);
FloatVector b = FloatVector.fromArray(SPECIES, array2, 0);

// 向量运算
FloatVector c = a.add(b);

// 存储结果
c.intoArray(resultArray, 0);
```
