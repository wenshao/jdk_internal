# JEP 402: Unified Generics

> **Status**: Preview | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 24+ (开发中)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

统一泛型系统，支持原始类型作为泛型类型参数，消除类型擦除的限制。

---

## 目标

- 支持原始类型泛型
- 消除自动装箱
- 提高性能
- 保持向后兼容

---

## 语法

```java
// 当前：需要装箱
List<Integer> list = new ArrayList<>();
list.add(42);  // 自动装箱

// 未来：直接支持原始类型
List<int> list = new ArrayList<>();
list.add(42);  // 无装箱
```
