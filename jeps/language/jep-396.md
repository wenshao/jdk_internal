# JEP 396: Strongly Encapsulate JDK Internals by Default

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE
>
> **Lead**: Alan Bateman (Oracle)
>
> **JDK Version**: JDK 16 (2021-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

默认强封装 JDK 内部 API，仅保留关键内部 API 用于深度反射。

---

## 目标

- 默认封装内部 API
- 提高安全性
- 减少技术债务
