# JEP 401: Primitive Classes

> **Status**: Preview | **Type**: Feature | **Component**: Java SE / Language
>
> **Lead**: Brian Goetz (Oracle)
>
> **JDK Version**: JDK 23+ (开发中)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入原始类型类，作为 Project Valhalla 的一部分，提供值类型的支持。

---

## 目标

- 提供值类型语义
- 消除对象头开销
- 支持泛型特化
- 提高性能

---

## 语法

```java
// 值类型定义
value class Point {
    int x;
    int y;
}

// 使用
Point p = new Point(1, 2);
Point p2 = p;  // 按值复制
```
