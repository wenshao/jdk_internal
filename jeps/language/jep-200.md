# JEP 200: The Modular JDK

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE
>
> **Lead**: Alan Bateman (Oracle)
>
> **JDK Version**: JDK 9 (2017-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

将 JDK 本身模块化，使用模块系统来组织和封装 JDK 的内部实现。

---

## 目标

- 将 JDK 划分为约 90 个模块
- 提供可靠的封装
- 支持可选配置
- 提高可维护性

---

## 模块结构

### JDK 9 模块分类

```
java.* 模块（标准 API）
├── java.base          // 基础模块
├── java.sql           // JDBC
├── java.logging       // 日志
├── java.xml           // XML
├── java.desktop       // AWT/Swing
└── ...

jdk.* 模块（JDK 工具和服务）
├── jdk.compiler       // 编译器
├── jdk.javadoc        // Javadoc
├── jdk.jfr            // Flight Recorder
└── ...
```
