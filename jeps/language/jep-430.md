## 1. 概述

String Templates（字符串模板）原计划为 Java 提供安全、易读的字符串插值方式。但在 JDK 21-22 经历三次预览后，于 JDK 23 前被**撤回**。

**撤回原因**:
- 设计需要进一步讨论
- 社区反馈需要更多考虑
- 没有确定何时会重新引入

详见: [JDK-8343684](https://bugs.openjdk.org/browse/JDK-8343684)

---

## 2. 核心特性

### 模板表达式

```java
String name = "World";
String message = STR."Hello, \{name}!";
```

### 内置模板处理器

| 处理器 | 用途 | 示例 |
|--------|------|------|
| STR | 普通字符串 | `STR."Value: \{x}"` |
| FMT | 格式化字符串 | `FMT."Value: \{x}%.2f"` |
| JSON | JSON 格式 | `JSON."\{\"key\": \{x}\}"` |

---

## 3. 使用示例

### 基础用法

```java
// 嵌入变量
int x = 10, y = 20;
String s = STR."\{x} + \{y} = \{x + y}";

// 嵌入表达式
String status = STR."User: \{user.getName()}, Age: \{user.getAge()}";
```

### 多行模板

```java
String json = JSON."""
    {
        "name": "\{name}",
        "age": \{age}
    }
    """;
```

### 安全特性

```java
// JSON 自动转义
String input = "<script>";
String safe = JSON."\{\"content\": \"\{input}\"}";
// {"content": "<script>"}
```
