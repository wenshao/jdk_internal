# JEP 454: Foreign Function & Memory API

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Maurizio Cimadamore (Oracle)
>
> **JDK Version**: JDK 22 (2024-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入外部函数和内存 API，允许 Java 程序调用外部代码和处理外部内存。

---

## 目标

- 提供外部内存访问
- 提供外部函数调用
- 替代 JNI
- 保证内存安全

---

## 核心 API

### MemorySegment

```java
// 分配内存
MemorySegment segment = Arena.ofAuto().allocate(100);

// 读写数据
segment.set(ValueLayout.JAVA_INT, 0, 42);
int value = segment.get(ValueLayout.JAVA_INT, 0);
```

### Linker

```java
// 调用本地函数
Linker linker = Linker.nativeLinker();
SymbolLookup lib = SymbolLookup.libraryLookup("libc", session);
MemorySegment strlen = lib.find("strlen").get();

FunctionDescriptor descriptor = FunctionDescriptor.of(
    ValueLayout.JAVA_LONG,
    ValueLayout.ADDRESS
);
long len = (long) linker.downcallHandle(strlen, descriptor)
    .invokeExact(segment.address());
```
