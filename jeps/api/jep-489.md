> **Status**: Preview | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Joe Darcy (Oracle)
>
> **JDK Version**: JDK 25 (2025-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 JSON 转义 API 预览，提供安全的 JSON 字符串转义功能。

---

## 目标

- 提供 JSON 转义工具
- 防止 XSS 攻击
- 支持 HTML 上下文
- 收集社区反馈

---

## 核心 API

### 基本转义

```java
// 转义字符串
String escaped = JsonEscaper.escapeJson("Hello \"World\"");
// 输出：Hello \"World\"

// 转义包含特殊字符的字符串
String input = "<script>alert('XSS')</script>";
String escaped = JsonEscaper.escapeJson(input);
```

### HTML 上下文

```java
// 在 HTML 中安全使用 JSON
String jsonInHtml = JsonEscaper.escapeForHtmlContext(jsonString);

// 在 JavaScript 中安全使用
String jsonInJs = JsonEscaper.escapeForJavaScriptContext(jsonString);
```
