# JEP 264: Platform Logging API

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Daniel Fuchs (Oracle)
>
> **JDK Version**: JDK 9 (2017-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

为 Platform Logger 定义 API，允许 JVM 和 JDK 组件使用统一的日志记录机制。

---

## 目标

- 提供平台日志 API
- 支持 JDK 内部日志
- 可配置日志后端
- 保持向后兼容

---

## 核心 API

### System.Logger

```java
// 获取日志记录器
private static final System.Logger log = 
    System.getLogger(MyClass.class.getName());

// 记录日志
log.log(Level.INFO, "Message");
log.info("Info message");
log.error("Error message", exception);
```

### 日志级别

```java
enum Level {
    ALL,
    TRACE,
    DEBUG,
    INFO,
    WARNING,
    ERROR,
    OFF
}
```

### 配置

```properties
# logging.properties
handlers = java.util.logging.ConsoleHandler
.level = INFO
MyClass.level = DEBUG
```

---

## 使用场景

### JDK 组件日志

```java
// JDK 内部组件使用
System.getLogger("jdk.jfr").info("JFR started");
System.getLogger("java.net.http").debug("HTTP request");
```

### 应用日志

```java
public class MyApp {
    private static final System.Logger log = 
        System.getLogger(MyApp.class.getName());
    
    public void start() {
        log.info("Application starting");
        // ...
        log.debug("Initialization complete");
    }
}
```
