## 1. 背景与动机

### 问题

PEM (Privacy-Enhanced Mail) 是常用的加密对象文本编码格式：

```
-----BEGIN PRIVATE KEY-----
MIIEvgIBADANBgkqhkiG9w0BAQEFAASCBKgwggSkAgEAAoIBAQC...
-----END PRIVATE KEY-----
```

之前 Java 没有标准 API 处理 PEM：
- 需要第三方库 (Bouncy Castle)
- 手动 Base64 解码
- 不同格式处理不一致

### 解决方案

JEP 470 提供标准 PEM 编解码 API：
- `PEMEncoder` - 编码为 PEM 格式
- `PEMDecoder` - 从 PEM 解码
- `PEMRecord` - PEM 记录结构

## 2. 使用示例

### 编码为 PEM

```java
import java.security.PEMEncoder;
import java.security.KeyPair;
import java.security.KeyPairGenerator;

// 生成密钥对
KeyPairGenerator kpg = KeyPairGenerator.getInstance("RSA");
kpg.initialize(2048);
KeyPair keyPair = kpg.generateKeyPair();

// 编码私钥为 PEM
PEMEncoder encoder = new PEMEncoder();
String pem = encoder.encode(keyPair.getPrivate());

System.out.println(pem);
// 输出:
// -----BEGIN PRIVATE KEY-----
// MIIEvgIBADANBgkqhkiG9w0BAQEFAASCBKgwggSkAgEAAoIBAQC...
// -----END PRIVATE KEY-----
```

### 从 PEM 解码

```java
import java.security.PEMDecoder;
import java.security.PrivateKey;

String pem = """
    -----BEGIN PRIVATE KEY-----
    MIIEvgIBADANBgkqhkiG9w0BAQEFAASCBKgwggSkAgEAAoIBAQC...
    -----END PRIVATE KEY-----
    """;

PEMDecoder decoder = new PEMDecoder();
PrivateKey key = decoder.decode(pem, PrivateKey.class);
```

### 编码证书

```java
import java.security.cert.X509Certificate;

// 编码证书
PEMEncoder encoder = new PEMEncoder();
String certPem = encoder.encode(certificate);

// 输出:
// -----BEGIN CERTIFICATE-----
// MIIDXTCCAkWgAwIBAgIJAJC1HiIAZAiUMA0GCSqGSIb3Qq87...
// -----END CERTIFICATE-----
```

### 加密私钥

```java
import javax.crypto.EncryptedPrivateKeyInfo;
import javax.crypto.spec.PBEParameterSpec;

// 加密私钥
char[] password = "secret".toCharArray();
EncryptedPrivateKeyInfo encrypted = EncryptedPrivateKeyInfo.create(
    keyPair.getPrivate(),
    password,
    "PBEWithSHA1AndDESede"
);

// 编码加密私钥
String encryptedPem = encoder.encode(encrypted);
// -----BEGIN ENCRYPTED PRIVATE KEY-----
// ...
// -----END ENCRYPTED PRIVATE KEY-----
```

## 3. 实现分析

### API 结构

```java
// java.security.PEMEncoder
public final class PEMEncoder {
    public PEMEncoder();
    public String encode(Key key);
    public String encode(X509Certificate cert);
    public String encode(X509CRL crl);
    public String encode(EncryptedPrivateKeyInfo info);
}

// java.security.PEMDecoder
public final class PEMDecoder {
    public PEMDecoder();
    public <T> T decode(String pem, Class<T> type);
    public PEMRecord decodeToRecord(String pem);
}

// java.security.PEMRecord
public final class PEMRecord {
    public String getType();      // "PRIVATE KEY", "CERTIFICATE", etc.
    public byte[] getEncoded();   // DER 编码的数据
    public Map<String, String> getHeaders();
}

// java.security.DEREncodable
public interface DEREncodable {
    byte[] getEncoded();  // DER 编码
}
```

### 支持的类型

| PEM 类型 | Java 类型 |
|----------|-----------|
| PRIVATE KEY | PrivateKey |
| PUBLIC KEY | PublicKey |
| CERTIFICATE | X509Certificate |
| ENCRYPTED PRIVATE KEY | EncryptedPrivateKeyInfo |
| PKCS7 | PKCS7 |
| X509 CRL | X509CRL |

## 4. 开发者影响

### 与 Bouncy Castle 对比

```java
// Bouncy Castle
import org.bouncycastle.openssl.PEMParser;
import org.bouncycastle.openssl.PEMWriter;

// 读取
PEMParser parser = new PEMParser(new StringReader(pem));
Object obj = parser.readObject();

// 写入
StringWriter writer = new StringWriter();
PEMWriter pemWriter = new PEMWriter(writer);
pemWriter.writeObject(key);
String pem = writer.toString();

// JDK 26 标准 API
PEMDecoder decoder = new PEMDecoder();
PrivateKey key = decoder.decode(pem, PrivateKey.class);

PEMEncoder encoder = new PEMEncoder();
String pem = encoder.encode(key);
```

### 迁移指南

```java
// 旧代码：手动处理
String pem = "-----BEGIN PRIVATE KEY-----\n" +
    Base64.getEncoder().encodeToString(key.getEncoded()) +
    "\n-----END PRIVATE KEY-----";

// 新代码：使用标准 API
PEMEncoder encoder = new PEMEncoder();
String pem = encoder.encode(key);
```

### 最佳实践

```java
// ✅ 推荐：使用标准 API
PEMEncoder encoder = new PEMEncoder();
String pem = encoder.encode(privateKey);

// ✅ 推荐：类型安全解码
PrivateKey key = decoder.decode(pem, PrivateKey.class);

// ✅ 推荐：处理加密私钥
EncryptedPrivateKeyInfo encrypted = EncryptedPrivateKeyInfo.create(
    privateKey, password, algorithm
);
String encryptedPem = encoder.encode(encrypted);

// ⚠️ 注意：密码保护
char[] password = getPassword();  // 从安全来源获取
// 使用后清除
Arrays.fill(password, '\0');
```
