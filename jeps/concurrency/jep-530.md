## 1. 背景与动机

### 问题

Java 模式匹配之前不支持原始类型：

```java
// 之前：需要包装类
Object obj = 42;
if (obj instanceof Integer i) {
    int value = i.intValue();  // 拆箱
}

// switch 不支持原始类型
switch (obj) {
    case Integer i -> process(i.intValue());
    case Double d -> process(d.doubleValue());
    // ...
}
```

### 解决方案

JEP 530 允许在模式匹配中直接使用原始类型：

```java
Object obj = 42;
if (obj instanceof int i) {
    process(i);  // 直接使用原始类型
}
```

## 2. 使用示例

### instanceof 模式匹配

```java
Object obj = getValue();

// 原始类型模式匹配
if (obj instanceof int i) {
    System.out.println("int: " + i);
} else if (obj instanceof long l) {
    System.out.println("long: " + l);
} else if (obj instanceof double d) {
    System.out.println("double: " + d);
}

// 自动类型转换
if (obj instanceof int i) {
    long l = i;  // int 自动提升为 long
}
```

### switch 模式匹配

```java
Object value = getValue();

String result = switch (value) {
    case int i -> "int: " + i;
    case long l -> "long: " + l;
    case float f -> "float: " + f;
    case double d -> "double: " + d;
    case boolean b -> "boolean: " + b;
    case char c -> "char: " + c;
    case String s -> "String: " + s;
    default -> "unknown";
};
```

### 精确匹配

```java
// 精确类型匹配
switch (value) {
    case int i when i > 0 -> "positive int";
    case int i when i < 0 -> "negative int";
    case int i -> "zero";
    case long l -> "long";
    // ...
}
```

### 类型转换规则

```java
// 允许的转换
Object obj = 42;

if (obj instanceof int i) {
    // int 可以匹配 Integer 包装的对象
}

if (obj instanceof long l) {
    // long 可以匹配 int 值（自动拓宽）
}

// 禁止的转换
if (obj instanceof byte b) {
    // byte 不能匹配 int 值（需要显式转换）
}
```

## 4. 开发者影响

### 简化代码

```java
// 旧代码
static String format(Object value) {
    if (value instanceof Integer i) {
        return "int: " + i.intValue();
    } else if (value instanceof Long l) {
        return "long: " + l.longValue();
    } else if (value instanceof Double d) {
        return "double: " + d.doubleValue();
    }
    return "unknown";
}

// 新代码
static String format(Object value) {
    return switch (value) {
        case int i -> "int: " + i;
        case long l -> "long: " + l;
        case double d -> "double: " + d;
        default -> "unknown";
    };
}
```

### 性能优势

```java
// 避免拆箱开销
if (obj instanceof int i) {
    sum += i;  // 直接使用原始类型，无拆箱
}

// vs
if (obj instanceof Integer i) {
    sum += i.intValue();  // 需要拆箱
}
```

### 最佳实践

```java
// ✅ 推荐：使用原始类型模式
switch (value) {
    case int i -> processInt(i);
    case long l -> processLong(l);
    case double d -> processDouble(d);
}

// ✅ 推荐：配合守卫条件
switch (value) {
    case int i when i >= 0 -> "non-negative";
    case int i -> "negative";
}

// ⚠️ 注意：类型匹配顺序
switch (value) {
    case long l -> ...   // long 匹配 int 和 long
    case int i -> ...    // 如果 long 在前，这行永不执行
}
```
