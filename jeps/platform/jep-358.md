# JEP 358: Apple Silicon Support

> **Status**: Permanent | **Type**: Feature | **Component**: JDK / Port
>
> **Lead**: Volker Simonis (SAP, currently AWS)
>
> **JDK Version**: JDK 17 (2021-09)
>
> **Organization**: [SAP (currently AWS)](../../contributors/orgs/sap.md)

---

## 概览

为 Apple Silicon M1 芯片（AArch64 架构）提供 macOS 移植支持。

---

## 目标

- 支持 macOS/AArch64
- 提供完整的 JDK 功能
- 优化 M1 芯片性能
- 保持与 x86_64 兼容

---

## 技术细节

### 架构特性

- **处理器**: Apple M1/M2/M3 系列
- **架构**: ARM64 (AArch64)
- **内存模型**: TSO (Total Store Order)
- **指令集**: ARMv8.5-A

### 优化特性

- NEON SIMD 指令优化
- 硬件乘加指令
- 分支预测优化
- 缓存优化

---

## 使用方式

### 下载

```bash
# 从 Adoptium 下载
wget https://github.com/adoptium/temurin17-binaries/releases/download/jdk-17.0.1%2B12/OpenJDK17U-jdk_aarch64_mac_hotspot_17.0.1_12.tar.gz

# 从 Azul 下载
wget https://cdn.azul.com/zulu/bin/zulu17.30.15-ca-jdk17.0.1-macosx_aarch64.tar.gz
```

### 验证

```bash
# 检查架构
java -version
# 输出应包含：aarch64

# 检查是否是原生运行
arch -arm64 java -version
```
