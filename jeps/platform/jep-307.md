# JEP 307: Container Awareness

> **Status**: Permanent | **Type**: Feature | **Component**: JDK / Platform
>
> **Lead**: Bob Vandette (Oracle)
>
> **JDK Version**: JDK 10 (2018-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

使 JDK 能够感知容器环境（如 Docker），正确限制资源使用。

---

## 目标

- 检测容器环境
- 限制 CPU 使用
- 限制内存使用
- 自动调整 JVM 参数

---

## 核心特性

### CPU 限制

```bash
# Docker 容器限制 CPU
docker run --cpus="1.5" myapp

# JVM 自动检测
java -XX:+UseContainerSupport -jar myapp.jar
# JVM 会使用 1.5 个 CPU 计算线程数
```

### 内存限制

```bash
# Docker 容器限制内存
docker run --memory="2g" myapp

# JVM 自动计算堆大小
java -XX:+UseContainerSupport \
     -XX:MaxRAMPercentage=75.0 \
     -jar myapp.jar
# JVM 会使用 2GB * 75% = 1.5GB 作为最大堆
```

---

## JVM 参数

### 容器支持参数

```bash
# 启用容器支持（JDK 10+ 默认启用）
-XX:+UseContainerSupport

# 最大 RAM 百分比
-XX:MaxRAMPercentage=75.0

# 初始 RAM 百分比
-XX:InitialRAMPercentage=50.0

# 最小 RAM 百分比
-XX:MinRAMPercentage=50.0
```

### 调试参数

```bash
# 打印容器信息
-XX:+PrintContainerInfo

# 显示 JVM 参数
-XX:+PrintFlagsFinal
```
