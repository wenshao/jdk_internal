# JEP 290: Filter Incoming Serialization Data

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Serialization
>
> **Lead**: Lance Andersen (Oracle)
>
> **JDK Version**: JDK 9 (2017-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

增强序列化数据过滤机制，防止反序列化攻击。

---

## 目标

- 提供序列化数据过滤
- 防止反序列化攻击
- 配置化过滤规则
- 保持向后兼容

---

## 核心 API

```java
// 创建序列化过滤器
ObjectInputFilter filter = info -> {
    String className = info.serialClass().getName();
    if (className.startsWith("com.trusted.")) {
        return ObjectInputFilter.Status.ALLOWED;
    }
    if (className.startsWith("java.lang.Runtime")) {
        return ObjectInputFilter.Status.REJECTED;
    }
    return ObjectInputFilter.Status.UNDECIDED;
};

// 应用过滤器
ObjectInputStream ois = new ObjectInputStream(inputStream);
ois.setObjectInputFilter(filter);
```
