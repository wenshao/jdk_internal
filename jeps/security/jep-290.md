## 概览

增强序列化数据过滤机制，防止反序列化攻击，提高 Java 序列化安全性。

---

## 目标

- 提供序列化数据过滤 API
- 防止反序列化攻击
- 配置化过滤规则
- 保持向后兼容

---

## 核心 API

### ObjectInputFilter

```java
// 创建序列化过滤器
ObjectInputFilter filter = info -> {
    String className = info.serialClass().getName();
    
    // 允许白名单类
    if (className.startsWith("com.trusted.")) {
        return ObjectInputFilter.Status.ALLOWED;
    }
    
    // 拒绝危险类
    if (className.startsWith("java.lang.Runtime")) {
        return ObjectInputFilter.Status.REJECTED;
    }
    
    return ObjectInputFilter.Status.UNDECIDED;
};

// 应用过滤器
ObjectInputStream ois = new ObjectInputStream(inputStream);
ois.setObjectInputFilter(filter);
```

### 配置方式

```properties
# 通过系统属性配置
jdk.serialFilter=java.base/java.lang.*;java.base/java.util.*;!java.lang.Runtime

# 通过安全属性配置
jdk.serialFilter=!*;java.lang.String;java.util.*
```

---

## 安全场景

### 反序列化攻击防护

```java
// 攻击示例：反序列化恶意对象
// 攻击者构造包含 Runtime.exec() 的序列化数据

// 防护：使用过滤器
ObjectInputFilter.Config.setSerialFilterFactory(
    (config) -> new MySecurityFilter()
);
```

### 深度限制

```java
// 限制反序列化深度
ObjectInputFilter filter = ObjectInputFilter.Config.createFilter(
    "maxdepth=10;maxrefs=50"
);
```
