# JEP 332: TLS 1.3

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / Security
>
> **Lead**: Xue-Lei Fan (Oracle)
>
> **JDK Version**: JDK 11 (2018-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 TLS 1.3 协议支持，提供更安全和高效的传输层安全协议。

---

## 目标

- 实现 TLS 1.3 (RFC 8446)
- 提高安全性
- 改进握手性能
- 保持向后兼容

---

## 关键特性

### 安全性改进

- 移除弱加密算法（RC4, 3DES, MD5）
- 强制使用前向保密（Forward Secrecy）
- 加密更多握手消息
- 防止降级攻击

### 性能改进

- 1-RTT 握手（减少延迟）
- 0-RTT 模式（恢复会话）
- 简化的密码套件

---

## 使用方式

### 启用 TLS 1.3

```java
// 通过系统属性
System.setProperty("https.protocols", "TLSv1.3");

// 通过代码
SSLContext context = SSLContext.getInstance("TLSv1.3");
context.init(null, null, null);
```

### 配置

```properties
# java.security
jdk.tls.client.protocols=TLSv1.3
jdk.tls.server.protocols=TLSv1.3
```
