## 1. 概述

本 JEP 旨在限制代理的动态加载，提升 Java 应用安全性。

---

## 2. 变更内容

### 废弃的 API

| API | 替代方案 |
|-----|----------|
| `String.appendToClassLoaderSearches` | - |
| `ClassLoader.appendToClassPathForToolAgents` | - |

### 影响范围

- 使用 `javaagent` 的应用
- 动态类加载框架
- APM/监控工具

---

## 3. 迁移建议

```java
// 之前：动态加载代理（将被禁止）
java -javaagent:agent.jar MyApp

// 现在：启动时指定
java -javaagent:agent.jar -jar MyApp.jar
```
