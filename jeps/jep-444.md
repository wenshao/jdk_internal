# JEP 444: Virtual Threads

> **状态**: ✅ 正式版 (JDK 21) | **JEP 链接**: https://openjdk.org/jeps/444

---

## 概述

Virtual Threads（虚拟线程）是 JDK 25 正式引入的特性，提供轻量级线程实现，支持百万级并发。

---

## 核心特性

### 创建虚拟线程

```java
// 方式 1: 直接创建
Thread vthread = Thread.ofVirtual().start(() -> {
    System.out.println("Hello from virtual thread");
});

// 方式 2: 使用 ExecutorService
try (var executor = Executors.newVirtualThreadPerTaskExecutor()) {
    executor.submit(() -> {
        // 任务
    });
}
```

### 与平台线程对比

| 特性 | 平台线程 | 虚拟线程 |
|------|----------|----------|
| 创建成本 | 高 | 极低 |
| 内存占用 | ~2MB | ~KB 级别 |
| 最大数量 | 数千 | 数百万 |
| 适用场景 | CPU 密集 | I/O 密集 |

---

## 使用示例

### Web 服务器示例

```java
try (var server = HttpServer.create()) {
    server.setExecutor(Executors.newVirtualThreadPerTaskExecutor());
    server.start();
}
```

### 批量处理

```java
List<Future<Result>> futures = new ArrayList<>();
try (var executor = Executors.newVirtualThreadPerTaskExecutor()) {
    for (Task task : tasks) {
        futures.add(executor.submit(() -> process(task)));
    }
}
```
