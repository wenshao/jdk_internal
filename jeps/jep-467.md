# JEP 467: Scoped Values

> **状态**: 🔍 预览版 (JDK 25, 第3次预览) | **JEP 链接**: https://openjdk.org/jeps/467

---

## 概述

Scoped Values 提供线程安全的隐式参数传递，替代 ThreadLocal。

---

## 核心特性

### 定义作用域值

```java
public static final ScopedValue<String> USER = ScopedValue.newInstance();
```

### 设置和访问

```java
ScopedValue.where(USER, "alice")
    .run(() -> {
        System.out.println(USER.get()); // "alice"
    });
```

---

## 使用示例

### 替代 ThreadLocal

```java
// 之前：ThreadLocal
private static final ThreadLocal<String> CONTEXT = new ThreadLocal<>();

// 现在：Scoped Value
private static final ScopedValue<String> CONTEXT = ScopedValue.newInstance();
```

### 嵌套作用域

```java
ScopedValue.where(KEY, "outer")
    .run(() -> {
        System.out.println(KEY.get()); // "outer"

        ScopedValue.where(KEY, "inner")
            .run(() -> {
                System.out.println(KEY.get()); // "inner"
            });

        System.out.println(KEY.get()); // "outer"
    });
```
