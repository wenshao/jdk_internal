# JEP 490: ZGC: Remove the Non-Generational Mode

> **状态**: ✅ 正式版 (JDK 24) | **JEP 链接**: https://openjdk.org/jeps/490

---

## 概述

移除 ZGC 的非分代模式，分代模式成为唯一选项。

---

## 背景

```
JDK 21: JEP 439 - 分代 ZGC (可选)
JDK 23: JEP 474 - 分代 ZGC 设为默认
JDK 24: JEP 490 - 移除非分代模式
```

---

## 影响

```bash
# 之前：可以显式禁用分代模式
java -XX:+UseZGC -XX:-ZGenerational MyApp

# JDK 24+：分代模式总是启用
java -XX:+UseZGC MyApp  # 自动使用分代模式
```
