## 1. 概述

JDK 23 将分代 ZGC 设为默认模式，进一步提升性能。

---

## 2. 核心变化

- **默认启用**: 使用 `-XX:+UseZGC` 即启用分代模式
- **非分代模式废弃**: `-XX:-ZGenerational` 标记为废弃
- **性能提升**: 开箱即用更好的吞吐量和延迟

---

## 3. 使用方法

```bash
# JDK 23+ 默认分代 ZGC
java -XX:+UseZGC MyApp

# 显式启用（默认已是分代）
java -XX:+UseZGC -XX:+ZGenerational MyApp
```
