## 1. 概述

分代 ZGC 将 Java 堆分为年轻代和老年代，大幅降低 GC 开销。

---

## 2. 核心改进

| 指标 | 非分代 ZGC | 分代 ZGC |
|------|-----------|----------|
| GC 开销 | ~8% | ~4% |
| 吞吐量 | 基准 | +10-20% |
| 堆占用 | 基准 | -30% |

---

## 3. 使用方法

```bash
# 启用分代 ZGC (JDK 21)
java -XX:+UseZGC -XX:+ZGenerational MyApp

# 查看分代统计
java -Xlog:gc*:gc.log -XX:+UseZGC -XX:+ZGenerational MyApp
```
