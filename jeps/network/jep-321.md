# JEP 321: HTTP Client

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Daniel Fuchs (Oracle)
>
> **JDK Version**: JDK 11 (2018-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入新的 HTTP Client API，支持 HTTP/2 和 WebSocket，替代老旧的 HttpURLConnection。

---

## 目标

- 支持 HTTP/2 协议
- 支持 WebSocket
- 提供异步编程模型
- 支持响应式流（Reactive Streams）

---

## 核心 API

### 基本使用

```java
// 创建客户端
HttpClient client = HttpClient.newHttpClient();

// 创建请求
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/data"))
    .GET()
    .build();

// 发送请求（同步）
HttpResponse<String> response = client.send(
    request, 
    HttpResponse.BodyHandlers.ofString()
);

// 发送请求（异步）
CompletableFuture<HttpResponse<String>> future = 
    client.sendAsync(request, HttpResponse.BodyHandlers.ofString());
```

### HTTP/2 特性

```java
// 多路复用（单个连接）
HttpClient client = HttpClient.newBuilder()
    .version(HttpClient.Version.HTTP_2)
    .build();

// 自动降级
HttpClient client = HttpClient.newBuilder()
    .version(HttpClient.Version.HTTP_2)
    .build();
// 如果服务器不支持 HTTP/2，自动降级到 HTTP/1.1
```

### 请求体

```java
// POST JSON
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/data"))
    .POST(HttpRequest.BodyPublishers.ofString(json))
    .header("Content-Type", "application/json")
    .build();

// POST 文件
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/upload"))
    .POST(HttpRequest.BodyPublishers.ofFile(path))
    .build();

// POST 字节数组
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/data"))
    .POST(HttpRequest.BodyPublishers.ofByteArray(data))
    .build();
```

### 响应处理

```java
// 字符串响应
HttpResponse<String> response = client.send(
    request, 
    HttpResponse.BodyHandlers.ofString()
);

// 文件响应
HttpResponse<Path> response = client.send(
    request, 
    HttpResponse.BodyHandlers.ofFile(path)
);

// 流式响应
HttpResponse<InputStream> response = client.send(
    request, 
    HttpResponse.BodyHandlers.ofInputStream()
);

// 多部分响应
HttpResponse<MultiPart> response = client.send(
    request, 
    HttpResponse.BodyHandlers.ofMultiPart(path)
);
```

## 异步编程

### CompletableFuture

```java
// 基本异步
CompletableFuture<HttpResponse<String>> future = 
    client.sendAsync(request, HttpResponse.BodyHandlers.ofString());

future.thenApply(HttpResponse::body)
      .thenAccept(System.out::println)
      .join();

// 链式调用
CompletableFuture<Void> all = CompletableFuture.allOf(
    client.sendAsync(req1, handler).thenAccept(this::process),
    client.sendAsync(req2, handler).thenAccept(this::process),
    client.sendAsync(req3, handler).thenAccept(this::process)
);
```

### 超时处理

```java
// 请求超时
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/data"))
    .timeout(Duration.ofSeconds(30))
    .GET()
    .build();

// 异步超时
CompletableFuture<HttpResponse<String>> future = 
    client.sendAsync(request, handler)
          .orTimeout(30, TimeUnit.SECONDS);
```

## WebSocket 支持

```java
// 创建 WebSocket
WebSocket ws = client.newWebSocketBuilder()
    .buildAsync(URI.create("wss://echo.example.com"), listener);

// WebSocket 监听器
WebSocket.Listener listener = new WebSocket.Listener() {
    @Override
    public void onOpen(WebSocket webSocket) {
        webSocket.request(1);
    }
    
    @Override
    public void onText(WebSocket webSocket, CharSequence data, boolean last) {
        System.out.println("Received: " + data);
        webSocket.request(1);
    }
    
    @Override
    public void onClose(WebSocket webSocket, int statusCode, String reason) {
        System.out.println("Closed: " + statusCode + " " + reason);
    }
};
```
