> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Alan Bateman (Oracle)
>
> **JDK Version**: JDK 16 (2021-03)
>

---

## 概览

引入 Unix Domain Sockets API，支持高效的进程间通信（IPC）。

---

## 目标

- 提供 Unix Domain Socket 支持
- 实现高效 IPC
- 跨平台 API
- 保持向后兼容

---

## 核心 API

```java
// 创建服务器
ServerSocketChannel server = ServerSocketChannel.open(StandardProtocolFamily.UNIX);
server.bind(UnixDomainSocketAddress.of("/tmp/mysocket"));

// 创建客户端
SocketChannel client = SocketChannel.open(StandardProtocolFamily.UNIX);
client.connect(UnixDomainSocketAddress.of("/tmp/mysocket"));

// 接受连接
SocketChannel accepted = server.accept();
```
