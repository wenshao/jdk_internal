# JEP 356: Enhanced Random Number Generator

> **Status**: Permanent | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Guy Steele (Oracle)
>
> **JDK Version**: JDK 17 (2021-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

增强随机数生成器 API，提供多种随机数生成算法和更好的可组合性。

---

## 目标

- 提供多种 RNG 算法
- 支持可组合的随机数生成
- 改进 API 设计
- 保持向后兼容

---

## 核心 API

### RandomGenerator 接口

```java
// 创建随机数生成器
RandomGenerator rng = RandomGenerator.getDefault();

// 使用特定算法
RandomGenerator rng = SplittableGenerator.of("L32X64MixRandom");

// 生成随机数
int i = rng.nextInt(100);
double d = rng.nextDouble();
boolean b = rng.nextBoolean();
```

### 算法类型

```java
// 可分割生成器
SplittableGenerator splittable = 
    SplittableGenerator.of("L32X64MixRandom");

// 跳跃生成器
JumpableGenerator jumpable = 
    JumpableGenerator.of("Xoshiro256PlusPlus");

// 算术生成器
ArithmeticGenerator arithmetic = 
    ArithmeticGenerator.of("L64X128MixRandom");
```

### 流式操作

```java
// 生成随机数流
IntStream stream = rng.ints(10, 0, 100);
stream.forEach(System.out::println);

// 生成对象流
Stream<Double> doubles = rng.doubles()
    .filter(d -> d > 0.5)
    .limit(10);
```
