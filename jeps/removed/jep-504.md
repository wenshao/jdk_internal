# JEP 504: Remove the Applet API

> **Status**: Closed / Delivered
> **Release**: JDK 26
> **Owner**: Phil Race
> **Issue**: [JDK-8359053](https://bugs.openjdk.org/browse/JDK-8359053)

## 1. 概述

移除已弃用的 Applet API，这是 Java 历史上重要的遗留代码清理。

## 2. 移除的 API

| 包 | 类/接口 |
|-----|---------|
| `java.applet` | `Applet` |
| `java.applet` | `AppletStub` |
| `java.applet` | `AudioClip` |
| `java.applet` | `AppletContext` |
| `java.applet` | `AppletInitializer` |

## 3. 迁移指南

### 替代方案

| 用途 | 替代技术 |
|------|----------|
| Web 应用 | Java Web Start (已弃用) → Web 应用框架 |
| 富客户端 | JavaFX |
| 嵌入式应用 | JLink + 自定义运行时 |

### 迁移步骤

1. 识别所有使用 Applet API 的代码
2. 评估替代方案
3. 重构为现代架构
4. 测试迁移后的功能

## 4. 历史背景

| 版本 | 状态 |
|------|------|
| JDK 9 | 标记为废弃 |
| JDK 17 | 计划移除 |
| JDK 26 | 正式移除 |
