# JEP 472: Prepare to Restrict the Use of JNI

> **状态**: ⚠️ 废弃警告 (JDK 24) | **JEP 链接**: https://openjdk.org/jeps/472

---

## 概述

警告开发者限制 JNI (Java Native Interface) 的使用，提升安全性。

---

## 变更内容

### 废弃的 API

| API | 替代方案 |
|-----|----------|
| 部分 JNI 函数 | Foreign Function & Memory API |
