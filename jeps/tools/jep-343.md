# JEP 343: Packaging Tool (Incubator)

> **Status**: Incubator | **Type**: Feature | **Component**: JDK Tools
>
> **Lead**: Alan Bateman (Oracle)
>
> **JDK Version**: JDK 13 (2019-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 jpackage 工具孵化器，用于打包 Java 应用为本地安装包。

---

## 目标

- 创建本地安装包
- 支持多平台
- 简化部署
- 收集社区反馈

---

## 使用方式

```bash
# Windows
jpackage --input target/ \
         --name MyApp \
         --main-jar myapp.jar \
         --type exe

# macOS
jpackage --input target/ \
         --name MyApp \
         --main-jar myapp.jar \
         --type dmg

# Linux
jpackage --input target/ \
         --name MyApp \
         --main-jar myapp.jar \
         --type deb
```
