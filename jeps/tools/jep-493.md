## 1. 概述

允许直接从模块路径链接运行时镜像，无需先创建 JMOD 文件。

---

## 2. 核心特性

### 之前的方式

```bash
# 1. 先创建 JMOD 文件
jmod create --class-path mods/com.example lib/com.example.jmod

# 2. 使用 JMOD 创建运行时镜像
jlink --module-path lib --add-modules com.example --output runtime
```

### 现在的方式

```bash
# 直接从模块路径链接
jlink --module-path mods --add-modules com.example --output runtime
```

---

## 3. 优势

- 简化构建流程
- 减少中间文件
- 加快构建速度
