## 概览

引入 jlink 工具，用于组装和优化模块及其依赖，创建自定义的运行时镜像。

---

## 目标

- 创建自定义运行时镜像
- 支持模块优化
- 减小部署体积

---

## 使用方式

```bash
# 创建自定义 JRE
jlink --add-modules java.base,java.logging \
      --output custom-jre

# 压缩
jlink --add-modules java.base \
      --compress=2 \
      --output custom-jre
```
