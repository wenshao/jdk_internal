# JEP 416: Code Snippets in Java API Documentation

> **Status**: Permanent | **Type**: Feature | **Component**: JDK Tools / javadoc
>
> **Lead**: Jonathan Gibbons (Oracle)
>
> **JDK Version**: JDK 18 (2022-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

扩展 javadoc 的 `@snippet` 标签，支持在 API 文档中嵌入格式化的代码示例，提高代码可读性和可维护性。

---

## 目标

- 改进 API 文档中的代码示例
- 支持语法高亮
- 支持外部代码文件引用
- 便于代码维护和测试

---

## 语法

### 基本用法

```java
/**
 * 计算两个数的和。
 *
 * {@snippet :
 * int result = add(2, 3);
 * System.out.println(result);  // 输出：5
 * }
 */
public int add(int a, int b) { ... }
```

### 外部文件引用

```java
/**
 * 使用示例：
 *
 * {@snippet file="Example.java" region="example"}
 */
public void method() { ... }
```

### 语法高亮

```java
/**
 * {@snippet lang="java" :
 * List<String> list = new ArrayList<>();
 * list.add("Hello");
 * }
 */
```

---

## 特性

### 1. 语言支持

```java
{@snippet lang="java" : ... }
{@snippet lang="python" : ... }
{@snippet lang="json" : ... }
```

### 2. 区域引用

```java
// 外部文件 Example.java
// @region example
public void example() {
    // 示例代码
}
// @endregion

// Javadoc 引用
{@snippet file="Example.java" region="example"}
```

### 3. 链接到 API

```java
/**
 * {@snippet :
 * List<String> list = new ArrayList<>();
 * // List 和 ArrayList 会自动链接到 API 文档
 * }
 */
```
