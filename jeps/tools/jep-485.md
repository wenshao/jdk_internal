# JEP 485: Stream Gatherers

> **状态**: ✅ 正式版 (JDK 24) | **JEP 链接**: https://openjdk.org/jeps/485

---

## 概述

Stream Gatherers 增强了 Stream API，允许自定义中间操作。

---

## 核心特性

### Gatherer 接口

```java
import java.util.stream.Gatherer;
import java.util.stream.Gatherers;

// 使用内置 Gatherer
List<Integer> result = Stream.of(1, 2, 3, 4, 5)
    .gather(Gatherers.windowFixed(3))
    .toList();
// [[1, 2, 3], [2, 3, 4], [3, 4, 5]]

// 自定义 Gatherer
Gatherer<Integer, ?, Integer> custom = Gatherer.of(
    initializer, integrator, combiner, finisher
);
```

---

## 使用示例

### 窗口操作

```java
// 固定大小窗口
List<List<Integer>> windows = Stream.of(1, 2, 3, 4, 5)
    .gather(Gatherers.windowFixed(3))
    .toList();
// [[1, 2, 3], [2, 3, 4], [3, 4, 5]]

// 滑动窗口
List<List<Integer>> sliding = Stream.of(1, 2, 3, 4, 5)
    .gather(Gatherers.windowSliding(3))
    .toList();
// [[1, 2, 3], [2, 3, 4], [3, 4, 5]]
```

### 扫描操作

```java
// 前缀和
List<Integer> sum = Stream.of(1, 2, 3, 4, 5)
    .gather(Gatherers.scan(() -> 0, (a, b) -> a + b))
    .toList();
// [1, 3, 6, 10, 15]
```
