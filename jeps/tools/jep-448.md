# JEP 448: JVM Code Heap Segmentation

> **状态**: ✅ 正式版 (JDK 25) | **JEP 链接**: https://openjdk.org/jeps/448

---

## 概述

JVM 代码堆分段优化启动时间和内存占用。

---

## 核心改进

| 改进 | 效果 |
|------|------|
| 更快的方法查找 | ~5% 启动提升 |
| 更低的代码缓存占用 | ~10% 内存减少 |
| 更好的 JIT 编译器性能 | ~3% 吞吐量提升 |
