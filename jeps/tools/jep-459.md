> **Status**: Preview | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: Adam Sotona (Oracle)
>
> **JDK Version**: JDK 22 (2024-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 Class-File API 预览，提供标准化的 API 用于解析、生成和转换 Java 类文件。

---

## 目标

- 提供标准类文件操作 API
- 简化字节码操作
- 替代 ASM 等第三方库
- 收集社区反馈

---

## 语法

```java
// 解析类文件
ClassModel cm = ClassFile.of().parse(classBytes);

// 生成类文件
byte[] bytes = ClassFile.of().build("HelloWorld", cb -> cb
    .withMethodSymbol("main", "([Ljava/lang/String;)V", mb -> mb
        .withFlags(AccFlag.STATIC)
        .withCode(cb2 -> cb2
            .loadConstant("Hello")
            .invokeVirtual(System.class, "out", "println")
            .return_()
        )
    )
);
```
