## 1. 概述

Derived Record Creation 简化代码，允许从现有记录派生新记录，仅指定不同的组件。

---

## 2. 核心特性

### with 派生

```java
record Point(int x, int y) {}

Point p = new Point(1, 2);
Point p2 = p.with { x = 10; };  // 仅修改 x，y 保持不变
// p2 = Point(10, 2)
```

### 嵌套记录

```java
record Address(String city, String street) {}
record Person(String name, Address address) {}

Person p = new Person("Alice", new Address("Beijing", "Main St"));
Person p2 = p.with { address.city = "Shanghai"; };
// 嵌套修改
```
