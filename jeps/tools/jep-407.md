## 1. 概述

JEP 407 在 JDK 17 中移除了 RMI Activation 机制，包括 `java.rmi.activation`
包及相关的 `rmid` 守护进程工具。这是 JDK 15 中
[JEP 385](https://openjdk.org/jeps/385) 废弃操作的后续步骤。

需要强调的是，**核心 RMI（`java.rmi`）保持不变**。JEP 407 仅移除了
Activation 这一子系统，即按需激活远程对象的机制。常规 RMI 远程方法调用、
`UnicastRemoteObject` 导出、RMI Registry 等功能均不受影响。

### 1.1 移除的组件

- **`java.rmi.activation`** 包——Activation 相关 API
- **`rmid`** 工具——RMI Activation 守护进程
- **`ActivationGroup`**、**`Activatable`**、**`ActivationDesc`** 等类

## 2. 背景：RMI Activation 的历史

### 2.1 分布式计算时代的产物（JDK 1.2, 1998）

RMI Activation 在 JDK 1.2 中引入，旨在解决分布式对象的生命周期管理问题。
传统 RMI 要求远程对象在被调用前必须已经创建并注册。Activation 机制允许
远程对象按需激活——当客户端发起调用时，`rmid` 守护进程会自动启动包含
目标对象的 JVM 并创建该对象实例。

### 2.2 未能流行

尽管让远程对象按需"休眠"和"唤醒"的理念有一定前瞻性，RMI Activation 从未
获得广泛采用。CORBA、EJB、后来的 Web Services 和 REST API 逐步取代了
RMI 在分布式计算中的地位。Activation 的复杂配置和 `rmid` 守护进程的
运维成本使其在实际生产环境中几乎无人使用。

---

## 3. JDK 17 中的变更

### 3.1 API 移除

`java.rmi.activation` 包被完整移除，包括 `Activatable`、`ActivationDesc`、
`ActivationGroup`、`ActivationID`、`ActivationSystem`、`Activator` 等核心类。

### 3.2 工具移除

`rmid` 守护进程工具从 JDK 发行版中移除。

### 3.3 保留的 RMI 功能

核心 RMI 不受影响：`java.rmi.Remote`、`UnicastRemoteObject`、
`Registry`、`rmiregistry` 工具以及 RMI 序列化和远程调用协议均保留。

---

## 4. 为何移除

### 4.1 极低的使用率

Oracle 的调研和社区反馈一致表明，RMI Activation 的实际使用率极低。
在 JEP 385 废弃公告发布后，社区几乎没有收到反对意见，进一步证实了
该机制已不再被依赖。

### 4.2 过时的分布式计算模型

现代分布式系统采用微服务架构、容器编排（Kubernetes）、服务网格等技术。
这些方案在服务发现、负载均衡、弹性扩展等方面远优于 RMI Activation
提供的简单按需激活。

### 4.3 安全风险与维护成本

`rmid` 守护进程具有按需启动 JVM 进程的能力，在现代安全环境中构成潜在
攻击面。移除该组件有助于减少 JDK 的安全攻击表面和整体维护负担。

---

## 5. 影响范围与迁移

使用标准 RMI（`UnicastRemoteObject`、`Registry`）的应用不受影响。
只有依赖 `java.rmi.activation` 包的应用需要迁移。

### 5.1 迁移建议

| 当前用法 | 迁移方案 |
|---------|---------|
| 按需激活远程对象 | 使用容器编排实现按需启动 |
| `Activatable` 扩展 | 改为继承 `UnicastRemoteObject` |
| `rmid` 管理进程 | 使用 systemd、Kubernetes 等进程管理 |
| 分布式对象调用 | 考虑迁移至 gRPC 或 REST API |

### 5.2 废弃与移除时间线

| 时间 | 版本 | 事件 |
|------|------|------|
| 1998 | JDK 1.2 | RMI Activation 引入 |
| **2020** | **JDK 15** | **JEP 385：标记为 Deprecated for Removal** |
| **2021** | **JDK 17** | **JEP 407：完成移除** |
