>
> **Lead**: Gavin Bierman (Oracle)

## 1. 概述

JEP 495 是 Simple Source Files and Instance Main Methods 的第四轮预览（此前名为 Unnamed Classes and Instance Main Methods），在 JDK 24 中发布。该特性旨在降低 Java 初学者的入门门槛，允许编写不需要类声明和 `public static void main(String[] args)` 签名的简单程序。

---

## 2. 背景与变更

传统的 Java "Hello World" 要求理解类声明、访问修饰符、静态方法和数组参数等概念。JEP 495 相对于 JEP 477（JDK 23）的变更：

| 变更领域 | 说明 |
|----------|------|
| **术语更新** | "Unnamed Classes" 更名为 "Simple Source Files" |
| **隐式导入** | 简单源文件自动导入 `java.base` 模块的常用类型 |
| **规范改进** | 细化了简单源文件与传统类文件的交互语义 |

---

## 3. 核心特性与示例

```java
// 最简 Hello World
void main() {
    println("Hello, World!");
}
```

```java
// 带有辅助方法和字段的简单源文件
String greeting = "Hello";

void main() {
    println(greeting + ", " + getName());
}

String getName() {
    return "Java Developer";
}
```

Java 启动器按以下优先级选择 main 方法：

| 优先级 | 签名 | 说明 |
|--------|------|------|
| 1 | `static void main(String[])` | 传统签名，最高优先级 |
| 2 | `static void main()` | 静态无参 main |
| 3 | `void main(String[])` | 实例方法带参数 |
| 4 | `void main()` | 实例方法无参数 |
