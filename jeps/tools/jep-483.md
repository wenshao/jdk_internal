# JEP 483: Ahead-of-Time Class Loading & Linking

> **状态**: ✅ 正式版 (JDK 24) | **JEP 链接**: https://openjdk.org/jeps/483

---

## 概述

支持类的 AOT (Ahead-of-Time) 加载和链接，改善启动时间。

---

## 核心特性

- 类预加载
- 符号预解析
- 链接预执行

---

## 使用方法

```bash
# 创建 AOT 缓存
java -XX:AOTCacheConfiguration=cache.aot MyApp

# 使用 AOT 缓存
java -XX:AOTCache=cache.aot MyApp
```
