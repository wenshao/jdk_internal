## 1. 概述

允许 JavaDoc 文档注释使用 Markdown 编写，简化文档编写和阅读。

---

## 2. 核心特性

### Markdown 语法

```java
/**
 * ## 示例方法
 *
 * 这个方法演示了 **Markdown** 文档注释：
 *
 * - 支持*斜体*和**粗体**
 * - 支持`代码`片段
 * - 支持列表
 *
 * @param x 输入参数
 * @return 计算结果
 */
int example(int x) { return x * 2; }
```

### HTML 兼容

```java
/**
 * <h2>HTML 仍然支持</h2>
 * 可以混合使用 Markdown 和 HTML
 */
void mixed() {}
```
