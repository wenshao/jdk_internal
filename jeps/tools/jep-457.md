# JEP 457: Class-File API (Preview)

> **Status**: Preview | **Type**: Feature | **Component**: Java SE / API
>
> **Lead**: [Chen Liang](../../by-contributor/profiles/chen-liang.md) (Oracle)
>
> **JDK Version**: JDK 22 (2024-03)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

引入 Class-File API 作为预览功能，提供标准化的 API 用于解析、生成和转换 Java 类文件，替代 ASM、ByteBuddy 等第三方库。

---

## 目标

- 提供标准类文件操作 API
- 简化字节码操作
- 支持方法体转换
- 收集社区反馈

---

## 动机

### 第三方库的问题

```java
// ASM 示例（复杂）
ClassWriter cw = new ClassWriter(COMPUTE_MAXS);
cw.visit(V1_8, ACC_PUBLIC, "MyClass", null, "java/lang/Object", null);

MethodVisitor mv = cw.visitMethod(ACC_PUBLIC, "hello", "()V", null, null);
mv.visitCode();
mv.visitFieldInsn(GETSTATIC, "java/lang/System", "out", "Ljava/io/PrintStream;");
mv.visitLdcInsn("Hello");
mv.visitMethodInsn(INVOKEVIRTUAL, "java/io/PrintStream", "println", "(Ljava/lang/String;)V", false);
mv.visitInsn(RETURN);
mv.visitMaxs(0, 0);
mv.visitEnd();

byte[] bytes = cw.toByteArray();
```

### Class-File API

```java
// Class-File API（简洁）
ClassFile cf = ClassFile.of();
byte[] bytes = cf.build("MyClass", cb -> cb
    .withMethodSymbol("hello", "()V", mb -> mb
        .withCode(cb2 -> cb2
            .stackOp(Opcodes.GETSTATIC, System.class, "out")
            .loadConstant("Hello")
            .invokeVirtual(PrintStream.class, "println")
            .return_()
        )
    )
);
```

## 使用方式

### 解析类文件

```java
// 从文件读取
byte[] classBytes = Files.readAllBytes(Paths.get("MyClass.class"));
ClassModel cm = ClassFile.of().parse(classBytes);

// 从 Class 对象读取
ClassModel cm = ClassFile.of().parse(MyClass.class);

// 遍历方法
for (MethodModel method : cm.methods()) {
    System.out.println(method.methodName());
}
```

### 生成类文件

```java
// 简单类
byte[] bytes = ClassFile.of().build("HelloWorld", cb -> cb
    .withMethodSymbol("main", "([Ljava/lang/String;)V", mb -> mb
        .withFlags(AccFlag.STATIC)
        .withCode(cb2 -> cb2
            .loadConstant("Hello, World!")
            .invokeVirtual(System.class, "out")
            .return_()
        )
    )
);

// 定义类
Class<?> helloClass = MethodHandles.lookup()
    .defineHiddenClass(bytes)
    .lookupClass();
```

### 转换类文件

```java
// 添加日志
ClassFile cf = ClassFile.of();
byte[] transformed = cf.transform(originalBytes, cm -> cm
    .withMethodTransform("targetMethod", "()V", mb -> mb
        .withCode(cb -> cb
            .loadConstant("Entering targetMethod")
            .invokeVirtual(System.class, "out", "println")
            .delegateTo(mb)  // 调用原方法
            .return_()
        )
    )
);
```

## 关键特性

### 1. 链式 API

```java
ClassFile.of()
    .build("MyClass", cb -> cb
        .withFieldSymbol("count", "I", fb -> fb
            .withFlags(AccFlag.PRIVATE)
        )
        .withMethodSymbol("increment", "()V", mb -> mb
            .withCode(cb2 -> cb2
                .loadThis()
                .dup()
                .getField(MyClass.class, "count")
                .loadConstant(1)
                .arithOp(Opcode.IADD)
                .putField(MyClass.class, "count")
                .return_()
            )
        )
    );
```

### 2. 类型安全

```java
// 类型安全的字段访问
cb.getField(MyClass.class, "count");  // 编译时检查

// 类型安全的方法调用
cb.invokeVirtual(PrintStream.class, "println", String.class);
```

### 3. 代码块构建

```java
// if 语句
cb.ifThen(Opcodes.IFEQ, label, cb2 -> {
    cb2.loadConstant("Zero");
    cb2.return_();
});

// 循环
Label loop = cb.newLabel();
Label end = cb.newLabel();
cb.branchTo(loop);
cb.labelBinding(loop);
// 循环体
cb.branchTo(loop);
cb.labelBinding(end);
```

## 使用场景

### 推荐

```java
// 1. AOP 代理
byte[] proxy = cf.transform(targetBytes, cm -> cm
    .withMethodTransform("*", mb -> addLogging(mb))
);

// 2. 运行时生成
Class<?> dynamicClass = ClassFile.of()
    .build("DynamicClass", cb -> /* ... */)
    .defineHiddenClass()
    .lookupClass();

// 3. 字节码分析
ClassFile.of().parse(bytes).methods().forEach(method -> {
    analyzeMethod(method);
});
```

### 不推荐

```java
// ❌ 复杂字节码操作（仍需要 ASM）
// ❌ 需要精细控制的场景
// ❌ 性能关键路径
```
