## 1. 概述

Class-File API 提供解析和生成类文件的标准 API。这是该 API 的最终版本。

---

## 2. 核心组件

```java
import java.lang.classfile.*;

// 读取类文件
ClassFile cf = ClassFile.of();
ClassModel cm = cf.parse(bytes);

// 遍历方法
for (MethodModel mm : cm.methods()) {
    System.out.println(mm.methodName());
}

// 生成类文件
byte[] newClass = cf.build(cm.thisClass().asSymbol(), cb -> {
    cb.withMethod("compute", MethodTypeDesc.of("(II)I"),
                 ClassFile.ACC_PUBLIC,
                 mb -> { /* ... */ });
});
```
