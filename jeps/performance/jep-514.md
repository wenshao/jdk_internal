# JEP 514: AOT Command-Line Ergonomics

> **Status**: Permanent | **Type**: Feature | **Component**: HotSpot / AOT
>
> **Lead**: John Rose (Oracle)
>
> **JDK Version**: JDK 25 (2025-09)
>
> **Organization**: [Oracle](../../contributors/orgs/oracle.md)

---

## 概览

改进 AOT（Ahead-of-Time）编译的命令行体验，使 AOT 编译更易于使用。

---

## 目标

- 简化 AOT 编译配置
- 提供智能默认值
- 改善错误消息
- 提高 AOT 采用率

---

## 核心特性

### 自动 AOT 编译

```bash
# 传统方式：需要复杂配置
java -XX:AOTMode=on -XX:AOTLibrary=libapp.so MyApp

# 新方式：自动检测和编译
java -XX:+UseAOT MyApp
```

### 智能缓存

```bash
# AOT 编译结果自动缓存
# 下次启动时自动使用
java -XX:+UseAOT MyApp
```

### 渐进式编译

```bash
# 首次启动：解释执行 + 后台 AOT 编译
# 后续启动：使用 AOT 编译结果
java -XX:+UseAOT MyApp
```
