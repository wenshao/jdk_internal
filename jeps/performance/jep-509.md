# JEP 509: JFR CPU-Time Profiling (Experimental)

> **Status**: Closed / Delivered
> **Release**: JDK 25
> **Owner**: Johannes Bechberger
> **Issue**: [JDK-8337789](https://bugs.openjdk.org/browse/JDK-8337789)

## 1. 概述

增强 JFR (Java Flight Recorder) 以支持 CPU 时间采样，提供更准确的性能分析数据。

## 2. 关键特性

- CPU 时间事件
- 异步采样机制
- 与现有 JFR 集成
- 低开销实现

## 3. 使用方式

```bash
# 启用 CPU 时间分析
jcmd <pid> JFR.start settings=cpu-profiling

# 或使用 JVM 参数
-XX:StartFlightRecording=settings=cpu-profiling
```

## 4. 事件类型

| 事件 | 描述 |
|------|------|
| `jdk.CPUTime` | CPU 时间采样 |
| `jdk.CPUTimeSample` | CPU 时间样本 |
| `jdk.ThreadCPULoad` | 线程 CPU 负载 |

## 5. 性能影响

| 指标 | 开销 |
|------|------|
| CPU 开销 | < 2% |
| 内存开销 | < 10MB |
| 启动延迟 | < 100ms |
