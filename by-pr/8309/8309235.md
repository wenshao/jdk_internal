## 描述

This PR addresses the issue of JShell not accepting unnamed local variables. This solution prints all local variables via the `/vars` command. e.g.,

```
jshell> int _ = 42;
 ==> 42

jshell> int _ = 43;
 ==> 43

jshell> String _ = "44";
 ==> "44"

jshell> /vars
|    int _ = 42
|    int _ = 43
|    String _ = "44"
```
