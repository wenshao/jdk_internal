## 描述

Change: 
```C++
struct CodeBlobType { 
  enum { [...] } 
}
```
To: 
```C++
enum class CodeBlobType { 
  [...] 
}; 
```
Using C++11 scoped enums provides a more clear view of intent, as enums can be enforced by the type system instead of being passed around as ints.
