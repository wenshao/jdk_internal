## 描述

For a `String` “s”, `s.indexOf(int)` can never return a value `>= s.length()` so change the check
```
        int pos = syntaxAndInput.indexOf(':');
        if (pos <= 0 || pos == syntaxAndInput.length())
```
to
```
        if (pos <= 0)
```
