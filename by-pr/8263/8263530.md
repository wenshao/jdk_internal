## 描述

SonarCloud reports:
  Remove or correct this "removeAll" call.

```
    void removeAll() {
        items.removeAll(items); // <--- here
        updateScrollbars();
    }
```

Calling `removeAll()` with the same collection risks concurrent modification exceptions. `clear()` would be correct and more efficient.
