## 描述

Ever since [JDK-8257679](https://bugs.openjdk.java.net/browse/JDK-8257679) was merged in 16, MSYS2 became capable of building a functional JDK on Windows. This PR updates some outdated documentation in building.html and building.md to reflect that change.

I appreciate any reviews, and would like to know how I might be able to improve the wording of both updated docs, if possible.
