## 背景

`StringConcatHelper` 是字符串拼接的核心工具类，包含多个重载的 `prepend` 方法，用于在字节数组中前置数据。

**问题发现**
在代码审查或阅读源码时，发现某些 `prepend` 方法的 JavaDoc 注释与实际参数类型不匹配：
- 注释描述为 `boolean value`
- 实际参数是 `char value`、`int value` 或 `long value`

这是一个复制粘贴错误：从 `prepend(boolean)` 复制注释后，忘记更新参数类型。

### 为什么会出错？

```java
// 开发顺序：
1. prepend(..., boolean value, String prefix)  // 首先实现，注释正确
2. prepend(..., char value, String prefix)     // 复制代码和注释，忘记更新
3. prepend(..., int value, String prefix)      // 同上
4. prepend(..., long value, String prefix)     // 同上

// 这是典型的 copy-paste 编程导致的错误
```

## 解决方案

### 核心改动

```java
// 修正后的注释

/**
 * Prepends constant and the stringly representation of value into buffer,
 * given the coder and final index. Index is measured in chars, not in bytes!
 *
 * @param index     final char index in the buffer
 * @param coder     coder of the buffer
 * @param buf       buffer to append to
 * @param value     char value to encode         // ✅ 修正
 * @param prefix    a constant to prepend before value
 * @return          updated index (coder value retained)
 */
static int prepend(int index, byte coder, byte[] buf, char value, String prefix) {
    // ...
}

/**
 * Prepends constant and the stringly representation of value into buffer,
 * ...
 * @param value     int value to encode          // ✅ 修正
 * @param prefix    a constant to prepend before value
 * @return          updated index (coder value retained)
 */
static int prepend(int index, byte coder, byte[] buf, int value, String prefix) {
    // ...
}

/**
 * Prepends constant and the stringly representation of value into buffer,
 * ...
 * @param value     long value to encode         // ✅ 修正
 * @param prefix    a constant to prepend before value
 * @return          updated index (coder value retained)
 */
static int prepend(int index, byte coder, byte[] buf, long value, String prefix) {
    // ...
}
```

### 变更详情

| 方法 | 原注释 | 修正后 |
|------|--------|--------|
| `prepend(..., char, ...)` | `boolean value to encode` | `char value to encode` |
| `prepend(..., int, ...)` | `boolean value to encode` | `int value to encode` |
| `prepend(..., long, ...)` | `boolean value to encode` | `long value to encode` |
