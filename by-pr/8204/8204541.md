## 描述

This change is just for AIX platform with IBM XL C/C++ 16.1.
By this change, lib's entry points are reduced by compilation and linkage option changes.

Without changes:
```
$ dump -X32_64 -Tv lib/libawt.so | grep EXP | wc
914
```
With changes
```
$ dump -X32_64 -Tv lib/libawt.so | grep EXP | wc
142 
```
