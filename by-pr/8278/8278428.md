## 描述

ObjectInputStream.readFully(byte[], off, len) throws IndexOutOfBoundsException when the length passed is less than the offset.
It is treating the length as the toOffset.
The replacement of the range check should have used checkFromIndexSize instead of checkFromToIndex. 

Added a test and minimal test cleanup.
