## 描述

Test 
scimark has a bug, described in the https://bugs.openjdk.org/browse/JDK-8315797
that causes test failure.

The Scimark is not maintained. The main goal of test was to provide example of Artifact-based test with 3rd party binary. There are a couple of other tests using Artifactory. So this test is completely useless now. 
I am removing it just to avoid spending time for anyone who can run test and observe this failure.
