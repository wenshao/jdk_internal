## 描述

The Template Framework uses "#" and "$" to allow binding names in templates, however currently there is no way to escape these characters. This leads to problems when we need them in the generated program, e.g. when templating an IR test that needs a separator, which by default is a "#".

This PR enables escaping "##" to "#" and "$$" to "$" in templates.
