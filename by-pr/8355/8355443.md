## 描述

For tests of the `java.io` package, instead of doing this
```
    public static void main(String[] args) throws Exception {
        if (File.separatorChar != '\\') {
            /* This test is only valid on win32 systems */
            return;
        }
```
do this
```
   @requires (os.family == "windows")
```
