## 描述

I have renamed member and method for `_pagesizes` and `_pre_allocated_pagesizes` to `_os_supported` and `_pre_allocated`.

The name `os_supported` is chosen to to describe that the set is those of the (huge) page sizes the operating supports, in contrast to those that the hardware supports (possibly more) or those that are pre-allocated (possibly fewer).

The name `pre_allocated` is shortened in the hope that it is understood from the type (`PageSizes`) and the context that it is page sizes we are talking about.

I also took the liberty of changing initialization to use curly bracket syntax. 

I also changed changed name of some local variables to make the code cleaner and also to remove trailing underscore (which in other c++ code bases mean member variable).
