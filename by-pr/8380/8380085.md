## 描述

This PR fixes a segfault in the jpackage Linux launcher caused by partial reads/writes when transferring JVM launcher data through a pipe.

It introduced `readFully` and `writeFully` helper functions in `LinuxLauncher.c` to ensure complete transfer by:
- Using a `while` loop to handle partial I/O.
- Retrying on `EINTR`.
