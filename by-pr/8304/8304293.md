## 描述

The following intrinsics in C1 are controlled by supports_atomic_xxx, but they are not set properly on RISC-V:
- _getAndAddInt
- _getAndAddLong
- _getAndSetInt
- _getAndSetLong
- _getAndSetReference

RISC-V provides a set of atomic instructions [1], these intrinsics could be enabled by default.
