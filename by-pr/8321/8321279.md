## 描述

Currently `Heap*Buffer.hashCode()` is inherited from `*Buffer` and implemented as 
```java
public int hashCode() {
    int h = 1;
    int p = position();
    for (int i = limit() - 1; i >= p; i--)
        h = 31 * h + (int)get(i);
    return h;
}
```
This can be improved using `ArraysSupport.vectorizedHashCode()`
