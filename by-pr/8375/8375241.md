## 描述

\-\-with\-native\-debug\-symbols\-level was added some time ago to allow selecting the level of debugging information created for compiled binaries. The UTIL\_ARG\_WITH utility that the option implementation uses has several features that can be used to simplify handling of this option, so that is done in this change.
