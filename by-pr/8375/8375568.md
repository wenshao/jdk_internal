## 描述

Thread names under debug mode has length limitation. Currently Controller threads are showing up as `Shenand..Thread`, which is the same as what regulator thread would show up as. Shenandoah worker threads are showing up as `Shenand..eads#0`. Hard to differentiate them all. Abbreviate the thread names here.

Tests: GHA passed.
