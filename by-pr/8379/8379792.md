## 描述

Patch fixes signed overflow in right_n_bits_typed. Unlike signed integer overflow, unsigned integer overflow is not undefined behavior. It is also an undefined behavior if the shift count is negative or greater than or equal to the width of the (promoted) operand.

Summary of changes:
- Compute the mask in the unsigned type to avoid signed overflow for n==bits-1.
- Guard nth_bit_typed: return 0 when the shift count is negative or >= width.
