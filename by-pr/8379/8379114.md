## 描述

[JDK-8272758] (#29264) incorrectly added the `if (contextPath == '/') { return true; }` fast path to the `HttpServer` context matcher. As a result, `GET foo` started matching context `/`, where it should not. Fix this.

[JDK-8272758]: https://bugs.openjdk.org/browse/JDK-8272758
