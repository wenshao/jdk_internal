## 描述

Please review this change to make tests in `test/langtools/jdk/internal/shellsupport/doc/` use JUnit.

The refactoring mostly converts `@Test` annotations from the class declaration to the method level, replaces `@run testng/...` tags with `@run junit/...`, and also flips the arguments of a single `assertEqual()` to have the expected value in first position.

A local run of `$ make test TEST=test/langtools/jdk/internal/shellsupport/doc/` now shows:
```
Running test 'jtreg:test/langtools/jdk/internal/shellsupport/doc'
Passed: jdk/internal/shellsupport/doc/JavadocFormatterTest.java
Passed: jdk/internal/shellsupport/doc/JavadocHelperTest.java
Passed: jdk/internal/shellsupport/doc/FullJavadocHelperTest.java
Test results: passed: 3
Framework-based tests: 21 = 0 TestNG + 21 JUnit
```
