## 描述

Please review this change to utilities/debug to use Atomic<T> instead of
AtomicAccess. The Events::_logs static data member is changed to a file scoped
static (so Events no longer needs to befriend EventLog), renamed accordingly,
and changed to Atomic<T>.

Testing: mach5 tier1
