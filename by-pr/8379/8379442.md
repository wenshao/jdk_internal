## 描述

One error meant the assertion was not being enforced.

The other error meant we were unlikely to recognize momentary triggers.  In the default configuration, with ShenandoahMomentaryAllocationRateSampleSize equal to 2, our numerator would be lacking one term, so we falsely believed the momentary allocation rate is smaller than actual.

This PR addresses both errors.
