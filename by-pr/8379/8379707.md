## 描述

The HTTP/3 request should have been configured with HTTP_3_URI_ONLY.
This will prevent falling back to TCP, and since only HTTP/3 is to be used the client will be able to use the full HTTP/3 timeout instead of falling back to TCP if the server does not reply fast enough.
This is particularly sensitive when running with fastdebug builds which have a tendency of slowing everything down.
