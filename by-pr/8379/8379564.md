## PR 描述

Hi, please consider this small change fixing a build issue with gcc-14.

Type of `_claimed_index` has been changed from `volatile jint` to `Atomic<jint>` in https://bugs.openjdk.org/browse/JDK-8378804. But seems that `clear_claimed()` missed updating after the type change, which is triggering the build error with gcc-14.

Verified by doing a successfull build on linux-x64 with gcc-14.
