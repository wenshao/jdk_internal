## 描述

After marking, if Shenandoah is able to recycle a sufficient amount of regions containing no live objects it can skip evacuation entirely. These "abbreviated cycles" provide significant performance improvements. The generational mode, however, ignores this shortcut when there are regions above the tenuring age. This change gives precedence to immediate garbage for the generational mode, so that neither evacuations, nor promotions occur when there is sufficient immediate garbage.

This change shows an 11% improvement on the Dacapo `sunflow` benchmark and an 8% improvement on the Renaissance `page-rank` benchmark. No statistically significant regressions were detected:
