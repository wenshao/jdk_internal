## 描述

Fix the issue that when creating a runtime bundle, jpackage doesn't read the version from the plist file, resulting in an inconsistency between the model and the output bundle. It manifests as an incorrect version suffix in the output bundle name.

Additionally, rename `TempKeychain` in `ActiveKeychainList` and add unit tests.
