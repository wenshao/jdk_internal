## 描述

Hi all,

JDK fails to build when c1 or c2 is disabled after JDK-8282420.
The reason is that c1/c2 code is used without `#ifdef` guards.
The fix just adds some `#ifdef`.

Testing:
 - build   on Linux/x64
 - jdk/jfr on Linux/x64

Thanks.
Best regards,
Jie
