## 描述

This adds a `TraceSplitIf` diagnostic flag to get more visibility into the Split-If optimization. When enabled, it logs the if-node being split, nodes being cloned up and down, and spinup activity. This is useful for debugging and understanding the optimization, and follows up on JDK-8330469 which removed the previous `PrintOpto && VerifyLoopOptimizations` guard.
