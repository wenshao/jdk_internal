## 描述

Hi all,

could you please review this small and trivial patch that unifies the names of hotspot gtests? in some cases, "_test" is added to gtest names, in others it isn't. given some tests specify "test" in their names, so we get "test_test" like in `UninitializedDoubleElementWorkerDataArrayTest.sum_test_test_vm`. the patch removes `_test` from the suffixes added by `TEST*` macros.

testing:
✅ `make test TEST=gtest` on macosx-x64
✅ `test/hotspot/jtreg/gtest/` on {linux,windows,macosx}-x64
