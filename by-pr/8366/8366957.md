## 描述

The `Arena::grow` is called for Arena allocations (i.e., `Amalloc`). There is a MallocLimit check in the `grow()` that should consider the AllocFailMode of the call. If the caller wants OOM, then it should do so.
A unit test is added specifically for this case.
