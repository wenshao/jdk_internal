## 描述

Without this method, `PSSParameterSpec::toString` shows something like:
```
MD: SHA-256
MGF: java.security.spec.MGF1ParameterSpec@77b52d12
SaltLength: 32
TrailerField: 1
```
This is ugly.

Noreg-trivial.
