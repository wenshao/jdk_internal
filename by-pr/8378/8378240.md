## 描述

Hi,

`MemNode::can_see_stored_value` assumes that `this` can never be a `StoreVector`. This is true, because for `StoreNode`s, `can_see_stored_value` is only called in `StoreNode::Identity`, and there it is guarded with `phase->type(val)->is_zero_type()`, which is never true for vectors. However, it is better if we take `StoreVectorNode` into consideration there. This PR also makes it explicit that `can_see_stored_value` only works for vanilla vector accesses and not masked ones or gathers/scatters.
