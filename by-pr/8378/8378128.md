## 描述

## Changes                                                                                                       
- `PLABStats` has atomic data members (`_allocated`, `_wasted`, `_undo_wasted`, `_unused`) with protected access. After JDK-8375971, the derived G1EvacStats accesses these via accessor functions rather than directly, so they can now be made private.
- `_description` remains protected as it is still directly accessed by G1EvacStats.

## Test
- Verified that `make images` completes without errors.
- Confirmed that `Tier1` tests pass successfully.
