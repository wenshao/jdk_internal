## 描述

Please find here a change that refactors `test/jdk/java/net/httpclient/http3/H3ErrorHandlingTest.java` to use JUnit.

This fix also renames `QuicStandaloneServer::addHandler` to `QuicStandaloneServer::setHandler` - since the method doesn't add anything but simply sets a variable. 

Finally, it also sets up `test/jdk/java/net/httpclient/TEST.properties` to reject `testng` actions in jtreg tests under `test/jdk/java/net/httpclient`, since that was the last test that needed to be converted in this test hierarchy.
