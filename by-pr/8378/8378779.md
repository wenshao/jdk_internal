## 描述

The test `compiler/gallery/TestParticleLife.java#ir`introduced in #29787 times out in C1 stress testing because the scalar fallback of the VectorAPI. This PR requires `TieredStopAtLevel > 3` to prevent the test running with C1 only, but allow other stress testing to take place.
