## 描述

Fixed an issue in CaseFolding cache, where the cache could incorrectly hit with the code point U+0000. The issue has been found while upgrading the Unicode data to 18-alpha,  and it cannot be reproduced (luckily) with the existing Unicode 17 data, so regression tests aren't provided.
