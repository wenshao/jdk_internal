## 描述

As per https://github.com/openjdk/jdk/pull/28002#issuecomment-3789507594 on JDK-8370691 pull request, splitting out portion of PR#28002 into a separate patch.


Patch adds new JTREG scalar operation wrapper routines for math operators and APIs, these wrapper are used for computing expected values of vector operations. These operations to compute expected values are passed though test generation templates and for float16 we perform argument upcasting to float and back to float after operation in these wrappers. 

All existing Vector API jtreg test are passing with the patch.

Kindly review and share your feedback.
