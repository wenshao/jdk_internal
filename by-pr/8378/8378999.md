## 描述

java.beans.beancontext.BeanContextSupport has add() and remove() methods which synchronize on their arguments.

1) This won't work with value types such as Integer, Byte, Short, etc ..
2) It seems unnecessary. The sychronization is already wrapped in       
 synchronized(BeanContext.globalHierarchyLock) {
..
}

The diff looks big but that's because it has to ex-dent a bunch of lines. It is really just a 4 line change.
