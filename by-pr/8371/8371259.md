# ML-DSA Intrinsics (后量子密码优化)

> JDK-8371259 | Volodymyr Paprotski | JDK 26

---

## 概述

ML-DSA (Module-Lattice-Based Digital Signature Algorithm) 是 NIST 标准化的后量子密码签名算法 (FIPS 204)。本 PR 为 JDK 26 引入了 AVX2 和 AVX-512 intrinsics 优化，显著提升 ML-DSA 的签名和验证性能。

## 背景

### 后量子密码威胁

```
传统签名算法:
- RSA, ECDSA, DSA
- 基于大数分解 / 离散对数
- 量子计算机可在多项式时间内破解

ML-DSA:
- 基于格 (Lattice) 问题
- 抗量子计算攻击
- NIST FIPS 204 标准
```

### 算法特点

| 特性 | ML-DSA-44 | ML-DSA-65 | ML-DSA-87 |
|------|-----------|-----------|-----------|
| 安全级别 | 2 | 3 | 5 |
| 公钥大小 | 1,312 bytes | 1,952 bytes | 2,592 bytes |
| 签名大小 | 2,420 bytes | 3,309 bytes | 4,627 bytes |
| 性能 | 最快 | 平衡 | 最安全 |

### 核心优化

#### 1. 多项式乘法 (AVX-512)

```cpp
// 文件: src/hotspot/cpu/x86/stubGenerator_x86_64_dilithium.cpp

// ML-DSA 核心运算的 AVX-512 优化
void generate_ml_dsa_avx512_stubs() {
    // 多项式乘法
    StubRoutines::_ml_dsa_poly_mul = generate_poly_mul_avx512();

    // NTT (Number Theoretic Transform) 变换
    StubRoutines::_ml_dsa_ntt = generate_ntt_avx512();

    // 逆 NTT 变换
    StubRoutines::_ml_dsa_inv_ntt = generate_inv_ntt_avx512();

    // 挑战采样
    StubRoutines::_ml_dsa_sample = generate_sample_avx512();

    // 拒绝采样
    StubRoutines::_ml_dsa_rej_sample = generate_rej_sample_avx512();
}

// 多项式乘法核心循环
// 使用 AVX-512 的 512-bit 向量寄存器
// 一次处理 16 个 32-bit 系数
void poly_mul_avx512(int32_t* a, int32_t* b, int32_t* c, int n) {
    __m512i va, vb, vc;
    for (int i = 0; i < n; i += 16) {
        va = _mm512_load_si512(&a[i]);
        vb = _mm512_load_si512(&b[i]);
        vc = _mm512_mullo_epi32(va, vb);
        _mm512_store_si512(&c[i], vc);
    }
}
```

#### 2. NTT 变换优化

```cpp
// NTT 变换: O(n log n) 多项式乘法
// 使用 Montgomery 约减避免除法

void ntt_forward_avx512(int32_t* poly, int n, int32_t q) {
    // 原地 NTT 变换
    for (int len = n/2; len > 0; len >>= 1) {
        int32_t w = 1;
        int32_t wlen = pow_mod(ROOT, n/(2*len), q);
        
        for (int i = 0; i < n; i += 2*len) {
            for (int j = 0; j < len; j++) {
                // AVX-512 向量化处理
                __m512i u = _mm512_load_si512(&poly[i+j]);
                __m512i v = _mm512_mullo_epi32(
                    _mm512_load_si512(&poly[i+j+len]),
                    _mm512_set1_epi32(w)
                );
                
                _mm512_store_si512(&poly[i+j], 
                    _mm512_add_epi32(u, v));
                _mm512_store_si512(&poly[i+j+len], 
                    _mm512_sub_epi32(u, v));
            }
            w = mul_mod(w, wlen, q);
        }
    }
}
```

#### 3. AArch64 NEON 实现

```cpp
// 文件: src/hotspot/cpu/aarch64/ml_dsa_aarch64.S

// NEON 向量化的多项式运算
// 使用 128-bit NEON 寄存器

function ml_dsa_poly_mul_neon
    // x0: a, x1: b, x2: c, x3: n
    mov     x4, #0
    mov     x5, x3
    
.loop:
    ldr     q0, [x0, x4]        // 加载 4 个 32-bit 系数
    ldr     q1, [x1, x4]
    mul     v0.4s, v0.4s, v1.4s  // 向量乘法
    str     q0, [x2, x4]
    
    add     x4, x4, #16
    cmp     x4, x5
    b.lt    .loop
    
    ret
end_function
```

## API 使用

### 基本用法

```java
import java.security.*;
import java.security.spec.*;

public class MLDSAExample {
    
    public static void main(String[] args) throws Exception {
        // 1. 生成密钥对
        KeyPairGenerator kpg = KeyPairGenerator.getInstance("ML-DSA-65");
        KeyPair keyPair = kpg.generateKeyPair();
        
        // 2. 签名
        byte[] data = "Hello, Post-Quantum World!".getBytes();
        
        Signature signer = Signature.getInstance("ML-DSA-65");
        signer.initSign(keyPair.getPrivate());
        signer.update(data);
        byte[] signature = signer.sign();
        
        // 3. 验证
        Signature verifier = Signature.getInstance("ML-DSA-65");
        verifier.initVerify(keyPair.getPublic());
        verifier.update(data);
        boolean valid = verifier.verify(signature);
        
        System.out.println("Signature valid: " + valid);
        System.out.println("Signature size: " + signature.length + " bytes");
    }
}
```

## 安全考虑

### 侧信道防护

```cpp
// 常数时间实现，防止时序攻击
// 所有分支都不依赖秘密数据

int32_t constant_time_compare(int32_t a, int32_t b) {
    // 使用位操作而非条件分支
    int32_t diff = a ^ b;
    return (diff == 0) ? 1 : 0;
}

// 避免内存访问模式泄露
void constant_time_memset(void* ptr, int value, size_t len) {
    volatile unsigned char* p = ptr;
    for (size_t i = 0; i < len; i++) {
        p[i] = value;
    }
}
```

## 迁移指南

### 从 RSA/ECDSA 迁移

```java
// 旧代码 (RSA)
KeyPairGenerator kpg = KeyPairGenerator.getInstance("RSA");
kpg.initialize(2048);

// 新代码 (ML-DSA)
KeyPairGenerator kpg = KeyPairGenerator.getInstance("ML-DSA-65");
// 无需指定密钥大小

// 旧代码 (ECDSA)
Signature sig = Signature.getInstance("SHA256withECDSA");

// 新代码 (ML-DSA)
Signature sig = Signature.getInstance("ML-DSA-65");
// ML-DSA 内置哈希，无需指定
```

### 混合签名方案

```java
// 过渡期：同时使用传统和后量子签名
public class HybridSigner {
    
    public byte[] hybridSign(PrivateKey rsaKey, PrivateKey mldsaKey, 
                             byte[] data) throws Exception {
        // 1. RSA 签名
        Signature rsa = Signature.getInstance("SHA256withRSA");
        rsa.initSign(rsaKey);
        rsa.update(data);
        byte[] rsaSig = rsa.sign();
        
        // 2. ML-DSA 签名
        Signature mldsa = Signature.getInstance("ML-DSA-65");
        mldsa.initSign(mldsaKey);
        mldsa.update(data);
        byte[] mldsaSig = mldsa.sign();
        
        // 3. 组合签名
        ByteArrayOutputStream out = new ByteArrayOutputStream();
        out.write(rsaSig.length);
        out.write(rsaSig);
        out.write(mldsaSig);
        
        return out.toByteArray();
    }
}
```

### 单元测试

```java
// 文件: test/jdk/java/security/MLDSA.java

@Test
public void testKeyGeneration() {
    for (String alg : List.of("ML-DSA-44", "ML-DSA-65", "ML-DSA-87")) {
        KeyPairGenerator kpg = KeyPairGenerator.getInstance(alg);
        KeyPair kp = kpg.generateKeyPair();
        
        assertNotNull(kp.getPublic());
        assertNotNull(kp.getPrivate());
        assertEquals(kp.getPublic().getAlgorithm(), "ML-DSA");
    }
}

@Test
public void testSignatureRoundTrip() {
    KeyPairGenerator kpg = KeyPairGenerator.getInstance("ML-DSA-65");
    KeyPair kp = kpg.generateKeyPair();
    
    byte[] data = "test data".getBytes();
    
    Signature sig = Signature.getInstance("ML-DSA-65");
    sig.initSign(kp.getPrivate());
    sig.update(data);
    byte[] signature = sig.sign();
    
    sig.initVerify(kp.getPublic());
    sig.update(data);
    assertTrue(sig.verify(signature));
}

@Test
public void testInvalidSignature() {
    KeyPairGenerator kpg = KeyPairGenerator.getInstance("ML-DSA-65");
    KeyPair kp = kpg.generateKeyPair();
    
    byte[] data = "original data".getBytes();
    byte[] tampered = "tampered data".getBytes();
    
    Signature sig = Signature.getInstance("ML-DSA-65");
    sig.initSign(kp.getPrivate());
    sig.update(data);
    byte[] signature = sig.sign();
    
    sig.initVerify(kp.getPublic());
    sig.update(tampered);
    assertFalse(sig.verify(signature));
}
```
