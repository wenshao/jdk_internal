## 描述

The SA src/jdk.hotspot.agent/doc/index.html file contains a table of SA scripts and their descriptions. All of these scripts are now gone. They were used to make it easier to launch what today we would call SA tools. Now all these tools are launched as sub-commands of the jhsdb command, which has its own man page. 

I also did some other minor fixes and cleanups.

The rendered html can be found here:
https://htmlpreview.github.io/?https://raw.githubusercontent.com/openjdk/jdk/dc0311f0a60ae550b01a3885d74fa23288df37d1/src/jdk.hotspot.agent/doc/index.html
