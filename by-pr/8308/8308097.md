## 描述

ZGC's current constructor syntax works well with some editors, but not all. There is a wish to move over from the current syntax:

```
ZClass:ZClass() :
    ZSuper(),
    _member0,
    _member1 {
  // Code
  doit();
}
```

to the following syntax:
```
ZClass:ZClass()
  : ZSuper(),
    _member0,
    _member1 {
  // Code
  doit();
}
```

I propose that make this change.
