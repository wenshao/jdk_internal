Please review the build guide update for Alpine Linux. 

building.html was generated by "make update-build-docs". However, this command produced a slightly different building.html, with some other sections modified as well. This is probably due to another version of the tooling installed. I have manually removed the modifications to other, not relevant sections of the building.html change introduced by the tool, and kept only the changes relevant to the md file change.
