## 描述

This test draws the same tray icon, has similar instructions to, and is subsumed by the related test   DisposeInActionEventTest.java
And because this test has incomplete instructions not covering macOS it is problem-listed.
The other test explicitly covers macOS. 
So the right answer is to remove this manual applet test which adds no valuable coverage and just costs time + effort to run.
