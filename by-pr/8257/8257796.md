Hi all,

TestUseSHA512IntrinsicsOptionOnSupportedCPU.java failed on our x86_32 platforms.

For x86, UseSHA512Intrinsics is only supported on x86_64 [1].
So It shouldn't be run on x86_32.

Thanks.
Best regards,
