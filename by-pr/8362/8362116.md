## 描述

When in JShell, doing:
```
jshell> IO.readln();
//Ctrl-D
$1 ==> null

jshell> IO.readln()
$2 ==> null //no input allowed
```

Where relying input from the main process to the remote/agent process, need to propagate `-1` as `-1` without close, which is what this PR is attempting to do.
