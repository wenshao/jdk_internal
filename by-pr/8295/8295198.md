## 描述

In [JDK-8294618](https://bugs.openjdk.org/browse/JDK-8294618), many of the old references to openjdk.java.net was updated.

The test code was intentionally left out of that change, but some other instances were missed, though.

This patch will fix those misses (but will still leave test code changes to a separate fix).
