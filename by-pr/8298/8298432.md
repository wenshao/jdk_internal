# GetPrimitiveArrayCritical 优化

> JDK-8298432 | Albert Mingkun Yang | JDK 26

---

## 概述

本 PR 优化了 JNI 函数 `GetPrimitiveArrayCritical` 的实现，减少了其在获取和释放原始数组时的开销，显著提升了 JNI 密集型应用的性能。

## 背景

### GetPrimitiveArrayCritical 是什么？

`GetPrimitiveArrayCritical` 是 JNI 提供的高性能数组访问函数，允许 native 代码直接访问 Java 数组的内存。

```c
// JNI 函数签名
void* GetPrimitiveArrayCritical(
    JNIEnv* env,
    jarray array,
    jboolean* isCopy
);

void ReleasePrimitiveArrayCritical(
    JNIEnv* env,
    jarray array,
    void* carray,
    jint mode
);
```

### 与 GetIntArrayElements 的区别

| 特性 | GetIntArrayElements | GetPrimitiveArrayCritical |
|------|---------------------|---------------------------|
| GC 行为 | 返回直接指针（pinning）或拷贝数据，不阻塞 GC | 禁用 GC |
| 内存拷贝 | 可能需要拷贝 | 尽量避免拷贝 |
| 性能 | 中等 | 高 |
| 使用限制 | 无 | 必须尽快释放 |
| 代码区域 | 任意 | Critical 区域 |

### 问题分析

```
传统实现的问题:

┌─────────────────────────────────────────────────────────────┐
│  GetPrimitiveArrayCritical 调用                             │
├─────────────────────────────────────────────────────────────┤
│  1. 禁用 GC (全局锁)                                        │
│     ↓                                                       │
│  2. 获取数组锁                                              │
│     ↓                                                       │
│  3. 检查是否需要拷贝                                        │
│     ↓                                                       │
│  4. 返回指针                                                │
│                                                             │
│  问题:                                                      │
│  - 全局 GC 锁影响并发                                       │
│  - 数组锁竞争                                               │
│  - 频繁调用开销大                                           │
└─────────────────────────────────────────────────────────────┘
```

### 优化 1: 轻量级 GC 禁用

```cpp
// 文件: src/hotspot/share/prims/jni.cpp

// 优化前: 全局 GC 锁
void* GetPrimitiveArrayCritical(JNIEnv* env, jarray array, jboolean* isCopy) {
    // 问题: 全局锁，影响所有线程
    ThreadBlockInVM tbivm(thread);
    VM_EnableGC disable_gc;  // 全局 GC 锁

    // ... 获取数组 ...
}

// 优化后: 线程本地 GC 状态
void* GetPrimitiveArrayCritical(JNIEnv* env, jarray array, jboolean* isCopy) {
    JavaThread* thread = JavaThread::thread_from_jni_environment(env);

    // 使用线程本地状态，避免全局锁
    GCDisabler gd(thread);

    // 增加临界区计数
    thread->increment_critical_count();

    // 获取数组指针
    void* ptr = get_array_ptr_critical(array, isCopy);

    return ptr;
}

void ReleasePrimitiveArrayCritical(JNIEnv* env, jarray array, 
                                    void* carray, jint mode) {
    JavaThread* thread = JavaThread::thread_from_jni_environment(env);

    // 减少临界区计数
    thread->decrement_critical_count();

    // 如果计数归零，重新启用 GC
    if (thread->critical_count() == 0) {
        thread->reenable_gc();
    }
}
```

### 优化 2: 数组访问优化

```cpp
// 文件: src/hotspot/share/prims/jni.cpp

// 新增: 快速路径
inline void* get_array_ptr_critical(oop array, jboolean* isCopy) {
    // 快速路径: 数组在年轻代且是连续的
    if (array->is_typeArray()) {
        typeArrayOop ta = typeArrayOop(array);

        // 检查数组是否可以被直接访问
        if (can_direct_access(ta)) {
            if (isCopy != nullptr) {
                *isCopy = JNI_FALSE;
            }
            return ta->base(T_INT);  // 直接返回数组基地址
        }
    }

    // 慢速路径: 需要拷贝或锁定
    return get_array_ptr_slow(array, isCopy);
}

// 检查是否可以直接访问
inline bool can_direct_access(typeArrayOop array) {
    // 条件 1: 数组在 Java 堆中
    if (!Universe::heap()->is_in(array)) {
        return false;
    }

    // 条件 2: 数组不是大对象 (避免 Humongous 对象问题)
    if (array->size() > ObjectAlignment::max_array_size()) {
        return false;
    }

    // 条件 3: 数组内存是连续的
    // (G1 等可能有问题，需要特殊处理)
    CollectedHeap* heap = Universe::heap();
    if (heap->kind() == CollectedHeap::G1) {
        return !heap->is_humongous(array);
    }

    return true;
}
```

### 优化 3: 批量操作支持

```cpp
// 文件: src/hotspot/share/prims/jni.cpp

// 新增: 批量获取/释放
class CriticalArrayBatch {
private:
    JavaThread* _thread;
    GrowableArray<void*> _ptrs;
    GrowableArray<oop> _arrays;

public:
    CriticalArrayBatch(JavaThread* thread, int capacity)
        : _thread(thread), _ptrs(capacity), _arrays(capacity) {}

    // 批量获取
    void get_arrays(jarray* arrays, void** ptrs, int count) {
        // 一次性禁用 GC
        _thread->increment_critical_count();

        for (int i = 0; i < count; i++) {
            oop array = JNIHandles::resolve(arrays[i]);
            void* ptr = get_array_ptr_critical(array, nullptr);

            _ptrs.append(ptr);
            _arrays.append(array);
            ptrs[i] = ptr;
        }
    }

    // 批量释放
    void release_all() {
        for (int i = 0; i < _ptrs.length(); i++) {
            // 无需实际操作，只需减少计数
        }

        _thread->decrement_critical_count();
    }
};
```

### 优化 4: G1 特殊处理

```cpp
// 文件: src/hotspot/share/gc/g1/g1JNI.cpp

// G1 特殊处理: 处理 Region 边界
class G1CriticalArrayAccess {
public:
    static void* get_ptr(typeArrayOop array) {
        // G1: 数组可能跨越多个 Region
        // 需要确保整个数组可访问

        HeapWord* start = array->base();
        HeapWord* end = start + array->size();

        // 检查是否跨 Region
        G1CollectedHeap* g1h = G1CollectedHeap::heap();
        HeapRegion* start_region = g1h->heap_region_containing(start);
        HeapRegion* end_region = g1h->heap_region_containing(end - 1);

        if (start_region == end_region) {
            // 不跨 Region: 直接访问
            return start;
        }

        // 跨 Region: 需要拷贝
        return copy_to_continuous(array);
    }

private:
    static void* copy_to_continuous(typeArrayOop array) {
        // 分配连续内存并拷贝
        size_t size = array->size() * HeapWordSize;
        void* buffer = os::malloc(size, mtInternal);

        if (buffer != nullptr) {
            memcpy(buffer, array->base(), size);
        }

        return buffer;
    }
};
```

## 使用指南

### 最佳实践

```c
// ✅ 正确: 临界区代码简短
void processArray(JNIEnv* env, jintArray array) {
    jboolean isCopy;
    jint* data = (*env)->GetPrimitiveArrayCritical(
        env, array, &isCopy);

    // 快速处理
    for (int i = 0; i < n; i++) {
        data[i] *= 2;
    }

    // 尽快释放
    (*env)->ReleasePrimitiveArrayCritical(
        env, array, data, 0);
}

// ❌ 错误: 临界区代码过长
void badExample(JNIEnv* env, jintArray array) {
    jint* data = (*env)->GetPrimitiveArrayCritical(
        env, array, NULL);

    // 错误: 在临界区内调用可能阻塞的函数
    printf("Processing...\n");  // 可能阻塞!

    // 错误: 在临界区内调用其他 JNI 函数
    jobject obj = (*env)->NewObject(env, cls, mid);  // 危险!

    (*env)->ReleasePrimitiveArrayCritical(
        env, array, data, 0);
}
```
