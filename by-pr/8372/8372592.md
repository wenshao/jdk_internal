## 描述

We had a previous mysterious failure that looks like it was due to a logger being gc'd.
To prevent that update all the marlin tests to use a logger stored in a static.
The 'final' modifier probably isn't necessary for this but is harmless, I'm doing the minimum here
so I don't want to go change log -> LOG everywhere just because its now final.
