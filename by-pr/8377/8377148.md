## 描述

This PR removes obsolete functions `Matcher::interpreter_frame_pointer_reg() ` and `Matcher::is_spillable_arg()`. When` is_spillable_arg()` is removed, the only use of function `Matcher::can_be_java_arg()` is inside an assert. This PR adds all (platform specific) definition of `Matcher::can_be_java_arg()` inside an assert block. As suggested in comments below,  this PR performs some ADL interpreter frame pointer declarations and corresponding ADLC parsing infrastructure cleanup by removing `ADLParser::interpreter_frame_pointer_parse`, `interpreter_frame_pointer`, and `_interpreter_frame_pointer_reg`


The call to `Matcher::interpreter_frame_pointer_reg()` was removed as a part of  [JDK-8315488](https://bugs.openjdk.org/browse/JDK-8315488) and call to `Matcher::is_spillable_arg()` was removed as a part of [JDK-8266561](https://bugs.openjdk.org/browse/JDK-8266561). 

I have checked the Valhalla repository and I do not see these functions used.
