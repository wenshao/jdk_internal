## 描述

IntrinsicPredicates.isSHA3IntrinsicAvailable() - tells if intrinsics is enabled in JDK by default through WhiteBox API
IntrinsicPredicates.SHA3_INSTRUCTION_AVAILABLE - tells if CPU supports the intrinsics

Generally, both these must match. In SHA3 case, where CPU is of aarch64 architecture with sha3 support but not on Apple or Qualcomm silicon, SHA3 is not enabled on default. The default setting can be manually overridden by sending -XX:+UseSHA3Intrinsics flag. Hence, the flag related tests don't work well when JDK support is taken as the criterion.

Fix: Switch from isSHA3IntrinsicAvailable() to SHA3_INSTRUCTION_AVAILABLE

Testing: tier1,tier2
