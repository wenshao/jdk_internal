## 描述

Dial down the number of iterations in the GetStackTraceALotWhenPinned.java and GetStackTraceALotWhenBlocking.java stress tests.

We have several stress tests that bash on Thread::getStackTrace while virtual threads are in transitions (mount/unmount when parking, blocking, waiting, ..). Some of these tests are running slower since JDK-8376568. Further work is required to reduce/avoid the wait in MountUnmountDisabler::disable_transition_for_one, tracked as JDK-8377312. In the mean-time, we dial down the tests.
