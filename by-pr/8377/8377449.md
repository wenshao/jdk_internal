## 描述

[JDK-8373026](http://java-service.client.nvidia.com/?q=8373026) added vector algorithm tests in `TestVectorAlgorithms.java` with C2 IR checks for x86 and AArch64. Some tests still had no AArch64 IR assertions. This change adds the missing AArch64 IR checks so coverage matches the x86 side.
