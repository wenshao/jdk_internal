## 描述

## Summary

This PR removes the temporary exclusion logic for Min/Max nodes in `PhaseIterGVN::verify_Identity_for`.

With the fixes for JDK-8373134 and JDK-8374896, the iterative GVN can now correctly verify the identity of these nodes without triggering verification failures

## Testing

Confirmed that the following tests pass on `macosx-aarch64-server-fastdebug` and `linux-x86_64-server-fastdebug`:

- `compiler/codegen/TestBooleanVect.java` with `-XX:VerifyIterativeGVN=1110`
- `compiler/igvn/TestMinMaxIdentity.java` with `-XX:VerifyIterativeGVN=1110`
- `tier1`
