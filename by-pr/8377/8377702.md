## 描述

The SpinPause gtest currently fails on Windows AArch64 because the SpinPause function returns 0 on that platform. This test should be disabled until [JDK-8343249 [Windows] Implement SpinPause](https://bugs.openjdk.org/browse/JDK-8343249) is fixed.
