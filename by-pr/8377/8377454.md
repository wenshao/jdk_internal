## 描述

Hi all,

This PR adds +/- compact object headers to the test matrix of `runtime/cds/appcds/TestZGCWithCDS.java`. If COH is explicitly set via the JTREG `vmoptions` or `javaoptions`, that flag is respected and only the corresponding variant is tested to save resources. While implementing this, I realized that the `UseCompressedOops` and `UseCompressedClassPointers` flags were also overrideable. This test is now skipped when `-XX:+/-UseCompressedOops` and/or `-XX:+/-UseCompressedClassPointers` are given via the command line.

Testing: tiers 1-4 on Linux (x64, AArch64), macOS (x64, AArch64), Windows (x64).
