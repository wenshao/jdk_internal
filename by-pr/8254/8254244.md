## 描述

On x86, arm, aarch64 and s390, TemplateTable::branch emits code to allocate a MethodData which is never called if running TieredCompilation. Skipping it slightly reduces interpreter code size and results in a minor startup improvement (~100k instructions less).

The PPC implementation differs significantly, and is left untouched.
