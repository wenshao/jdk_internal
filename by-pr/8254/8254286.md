## 描述

Basically, for code like:

        var t1 = switch (s) {
            case 1 -> i1;
            case 2 -> null;
            default -> i2;
        };

Attr.condType is currently passing <type-of-i1>, BOT, <type-of-i2> to Types.lub. But Types.lub needs the list without BOT, so filtering BOT(s) out of the list.
