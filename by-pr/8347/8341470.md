## 背景

### stripTrailingZeros() 方法

`stripTrailingZeros()` 移除 BigDecimal 末尾的零：

```java
BigDecimal bd = new BigDecimal("12.500");
bd = bd.stripTrailingZeros();  // 结果: 12.5 (或 125E-1)

BigDecimal bd2 = new BigDecimal("1000");
bd2 = bd2.stripTrailingZeros();  // 结果: 1E+3
```

### 原始实现问题

**JDK-8341470 之前**：

```java
// 朴素算法
public BigDecimal stripTrailingZeros() {
    // BigDecimal 内部表示: intVal * scale
    // 例如: 12.500 = 12500 * 10^-3

    // 问题：需要逐次尝试除以 2 和 5
    while (intVal.mod(BigInteger.TEN).equals(BigInteger.ZERO)) {
        intVal = intVal.divide(BigInteger.TEN);
        scale++;
    }

    return new BigDecimal(intVal, scale);
}
```

**问题**：
1. **逐次除法** - 对于大数，需要多次除法操作
2. **线性复杂度** - O(n) 其中 n 是末尾零的数量
3. **大整数运算慢** - BigInteger.mod() 和 divide() 是昂贵的操作

## 核心变更

### 重复平方法 (Repeated Squares Trick)

**核心思想**：不逐次除以 10，而是一次性检测 2^k 和 5^k 的因子。

#### 算法原理

```
10^n = 2^n * 5^n

要判断一个数能否被 10^n 整除：
等价于判断它能否同时被 2^n 和 5^n 整除

例如：
12000 能被 10^3 整除吗？
→ 能被 2^3 = 8 整除吗？ 12000 / 8 = 1500 ✓
→ 能被 5^3 = 125 整除吗？ 12000 / 125 = 96 ✓
→ 12000 能被 10^3 整除 ✓
```

### 代码实现

**变更前 (朴素算法)**:
```java
public BigDecimal stripTrailingZeros() {
    BigInteger intVal = this.intVal;
    int scale = this.scale;

    // 逐次检查
    while (intVal.mod(BigInteger.TEN).equals(BigInteger.ZERO)) {
        intVal = intVal.divide(BigInteger.TEN);
        scale++;
    }

    return new BigDecimal(intVal, scale);
}
```

**变更后 (重复平方法)**:
```java
public BigDecimal stripTrailingZeros() {
    BigInteger intVal = this.intVal;
    int scale = this.scale;

    // 使用重复平方法
    int n = findTrailingZeros(intVal);
    if (n > 0) {
        // 一次性除以 10^n
        BigInteger divisor = BigInteger.TEN.pow(n);
        intVal = intVal.divide(divisor);
        scale += n;
    }

    return new BigDecimal(intVal, scale);
}

// 使用二分查找找到最大 n
private static int findTrailingZeros(BigInteger value) {
    // 预计算的 2^k 和 5^k
    BigInteger[] powersOf2 = {BigInteger.valueOf(2),
                               BigInteger.valueOf(4),
                               BigInteger.valueOf(8),
                               ...};

    BigInteger[] powersOf5 = {BigInteger.valueOf(5),
                               BigInteger.valueOf(25),
                               BigInteger.valueOf(125),
                               ...};

    int n = 0;
    int k = 32;  // 从 2^32 开始检查

    while (k > 0) {
        if (value.mod(powersOf2[k]).equals(BigInteger.ZERO) &&
            value.mod(powersOf5[k]).equals(BigInteger.ZERO)) {
            n += k;
            value = value.divide(BigInteger.TEN.pow(k));
            k = Math.min(k, value.bitLength());
        } else {
            k /= 2;
        }
    }

    return n;
}
```

## 技术分析

### 算法对比

| 方面 | 朴素算法 | 重复平方法 |
|------|----------|-----------|
| **时间复杂度** | O(n) | O(log n) |
| **除法次数** | n 次 | log₂(n) 次 |
| **适合场景** | 小数 | 大数 |

### 性能对比

| 末尾零数量 | 朴素算法 | 重复平方法 | 提升 |
|-----------|----------|-----------|------|
| **3** | 3 次除法 | 2 次除法 | +33% |
| **10** | 10 次除法 | 4 次除法 | +60% |
| **100** | 100 次除法 | 7 次除法 | +93% |
| **1000** | 1000 次除法 | 10 次除法 | +99% |

### 示例

```
输入: new BigDecimal("1000000000000") (12 个零)

朴素算法:
- 第 1 次迭代: 检查 mod 10, 除以 10
- 第 2 次迭代: 检查 mod 10, 除以 10
- ...
- 第 12 次迭代: 检查 mod 10, 除以 10
总计: 12 次除法

重复平方法:
- 检查 10^8: 可整除, 除以 10^8, n=8
- 检查 10^4: 可整除, 除以 10^4, n=12
- 检查 10^2: 不可整除
总计: 2 次除法
```
