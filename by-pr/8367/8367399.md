## 描述

Added MemorySegment test cases to validate that the optimizations enabled by [JDK-8329077](https://bugs.openjdk.org/browse/JDK-8329077) apply as expected.

I considered adding these tests to `compiler.loopopts.superword.TestMemorySegment` but I ended adding them to `compiler.loopopts.superword.TestCompatibleUseDefTypeSize` as it seemed a better fit.

I have tested these changes on darwin/aarch64 (M4 Pro).
