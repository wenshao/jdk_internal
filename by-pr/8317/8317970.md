## 描述

macOS 10.x is no longer receiving updates - the most recent/last release was 10.15.7 back in July of 2022. It's time to bump the target macOS version (min/max) for macosx-x64. macOS 11.x is still receiving updates.

This change updates the target version for macosx-x64 to 11.00.00, which is the same version used for macosx-aarch64.
