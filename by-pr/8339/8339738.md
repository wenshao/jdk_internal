Hi,
Can you help to review this patch?
Thanks.

This patch improves the existing scalar version of CRC32 (table lookup algorithm) by vectorizing the scalar algorithm. This improvement is based on java.base/share/native/libzip/zlib/zcrc32.c, I invented the new vectorized version by making some modification to N (to 16) related code, then re-generate the tables needed, finally vectorize the code (original implementation in zcrc32.c is just scalar code).
