## 描述

The J2DBench.ColorConvertOp tests are used to track the performance of the littlecms library in the JDK. The new version of the littlecms will add support for the premultiplied alpha. But right now the J2DBench does not test the pre-alpha formats, so I have added all standard types we have.

FYI: I have tested all combinations of these flags w/o issues.
