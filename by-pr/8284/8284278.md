## 描述

Diagnostic Fix added for the Bug - [JDK-8283070](https://bugs.openjdk.java.net/browse/JDK-8283070)
javax/swing/JTable/8236907/LastVisibleRow.java failed with "RuntimeException: Test Case Failed!!". 
The bug was not reproducible in local machine and with runs on mach5. It was however observed in mach5 testing intermittently. The LastVisibleRow.java captures two Images before and After Mouse click of last Row in a Table. The Test Fails if the two images does not match. In order to find the root cause, the two images are written to a file for diagnostic purpose.
