## 描述

The log for Finished message looks like the below,
```
"Finished": {
  "verify data": {
    0000: ... ...
  }'}  // looks weird
```
because a line feed is missing in the format pattern.
```
"\"Finished\": '{'\n" +
"  \"verify data\": '{'\n" +
"{0}\n" +
"  '}'" +  // a line feed is needed
"'}'",
```
