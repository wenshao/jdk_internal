## 1. 核心特性

JDK 9 是 Java 历史上最重要的版本之一，引入了模块系统（Project Jigsaw）。

| 特性 | 影响 | 详情 |
|------|------|------|
| **模块系统 (JPMS)** | ⭐⭐⭐⭐⭐ | JEP 261，彻底改变 Java 架构 |
| **接口私有方法** | ⭐⭐⭐ | JEP 213 |
| **HTTP/2 Client（孵化）** | ⭐⭐⭐⭐ | JEP 110 |
| **G1 作为默认 GC** | ⭐⭐⭐ | JEP 248 |
| **集合工厂方法** | ⭐⭐⭐ | JEP 269 |
| **Process API** | ⭐⭐⭐ | JEP 102 |
| **jshell** | ⭐⭐⭐⭐ | JEP 222，Java Shell |
| **多版本 JAR** | ⭐⭐⭐ | JEP 238 |
| **Stack-Walking API** | ⭐⭐ | JEP 259 |
| **紧凑字符串** | ⭐⭐⭐ | JEP 254 |
| **变量句柄** | ⭐⭐⭐ | JEP 193 |
| **代码缓存分段** | ⭐⭐ | JEP 197 |
| **平台日志 API** | ⭐⭐ | JEP 264 |
| **UTF-8 属性文件** | ⭐⭐ | JEP 226 |

## 3. 代码示例

### 模块系统

```java
// module-info.java
module com.example.myapp {
    requires java.base;
    requires java.logging;
    exports com.example.api;
}
```

### 接口私有方法

```java
public interface MyInterface {
    default void doSomething() {
        helperMethod();
    }

    private void helperMethod() {
        System.out.println("Helper");
    }
}
```

### 集合工厂方法

```java
// 不可变集合
List<String> list = List.of("a", "b", "c");
Set<Integer> set = Set.of(1, 2, 3);
Map<String, Integer> map = Map.of("a", 1, "b", 2);
```

### jshell

```bash
$ jshell
|  Welcome to JShell -- Version 9
|  For an introduction type: /help intro

jshell> int x = 10
x ==> 10

jshell> System.out.println(x)
10
```

## 4. 破坏性变更

| 变更 | 影响 |
|------|------|
| 类加载器变更 | 需要适配模块系统 |
| `--illegal-access` | 默认警告 |
| GC 日志格式改变 | 需要更新 |
| 移除工具 | hprof, jhat, demos/samples |
| 默认密钥库格式 | 改为 PKCS12 |
