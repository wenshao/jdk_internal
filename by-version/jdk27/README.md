## 2. 版本概览

JDK 27 是继 JDK 26 (2026-03-17 GA) 之后的下一个 Feature Release。目前处于早期开发阶段，接受 bug 修复、小型增强和通过 JEP 流程提出的新特性。

### 关键特性

| JEP | 特性 | 状态 | 类别 |
|-----|------|------|------|
| [JEP 527](https://openjdk.org/jeps/527) | Post-Quantum Hybrid Key Exchange for TLS 1.3 | Targeted | 安全 |

> 注: JDK 27 仍处于早期开发阶段，更多 JEP 将在后续几个月中确定。
