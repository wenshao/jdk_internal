## 概述

SAPMachine 是 SAP 提供的 OpenJDK 发行版，针对 SAP 应用和 SAP 环境优化。

| 属性 | 值 |
|------|-----|
| **组织** | SAP |
| **官网** | https://sap.github.io/SapMachine/ |
| **下载** | https://github.com/SAP/SapMachine/releases |
| **许可证** | GPLv2 + CPEx |
| **商业支持** | ✅ (SAP 支持) |

---

## 特点

- ✅ **SAP 优化**: 针对 SAP 应用优化
- ✅ **长期支持**: 与 SAP 产品版本同步
- ✅ **SAP 集成**: 与 SAP 技术栈深度集成

```bash

```
